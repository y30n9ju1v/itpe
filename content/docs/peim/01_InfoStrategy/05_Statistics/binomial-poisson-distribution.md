## 개요

이항분포(Binomial Distribution)와 포아송분포(Poisson Distribution)는 모두 이산확률분포로, 사건 발생 횟수를 모델링한다. 이항분포는 고정된 시행 횟수에서 성공 횟수를, 포아송분포는 단위 시간/공간에서 희귀 사건 발생 횟수를 모델링한다.

## 핵심 내용

### 이항분포 (Binomial Distribution)

- **조건**: n번의 독립 시행, 각 시행에서 성공 확률 p 일정
- **확률변수**: X = n번 시행 중 성공 횟수
- **표기**: X ~ B(n, p)

**확률 질량 함수**:
```
P(X=k) = C(n,k) × p^k × (1-p)^(n-k)
```

**기댓값 및 분산**:
- E(X) = np
- Var(X) = np(1-p)

**예시**: 동전 10번 던져 앞면이 3번 나올 확률

### 포아송분포 (Poisson Distribution)

- **조건**: 단위 시간/공간에서 평균 λ회 발생하는 희귀 사건
- **확률변수**: X = 단위 시간/공간에서 사건 발생 횟수
- **표기**: X ~ Poisson(λ)

**확률 질량 함수**:
```
P(X=k) = (e^(-λ) × λ^k) / k!
```

**기댓값 및 분산**:
- E(X) = λ
- Var(X) = λ  ← 기댓값 = 분산 (포아송의 특징)

**예시**: 1시간 동안 콜센터에 걸려오는 전화 수

### 포아송분포 ≈ 이항분포 (근사 조건)

n → ∞, p → 0, np = λ 일정 → 이항분포가 포아송분포에 근사

### 비교 요약

| 구분 | 이항분포 | 포아송분포 |
|------|---------|---------|
| 시행 횟수 | 고정 (n) | 없음 (연속) |
| 성공 확률 | 고정 (p) | 없음 (λ로 대체) |
| 파라미터 | n, p | λ |
| 기댓값 | np | λ |
| 분산 | np(1-p) | λ |
| 적합 상황 | 유한 시행 반복 | 희귀 사건, 무한 시행 |
| 예시 | 불량품 수, 합격률 | 방사선 붕괴, 트래픽 패킷 |

## 출제 포인트

- 두 분포의 파라미터·기댓값·분산 공식 암기
- 포아송 특징: 기댓값 = 분산 = λ
- 이항→포아송 근사 조건: n→∞, p→0, np=λ 일정
