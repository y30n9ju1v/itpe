## 개요

베이즈 정리는 사전 확률(Prior)과 새로운 증거(Likelihood)를 결합하여 사후 확률(Posterior)을 갱신하는 조건부 확률 이론이다. 머신러닝의 나이브 베이즈 분류기, 스팸 필터, 의료 진단 등에 널리 활용된다.

## 핵심 내용

### 수식

```
P(A|B) = P(B|A) × P(A) / P(B)
```

- **P(A|B)**: 사후 확률 — B가 관측된 후 A의 확률
- **P(B|A)**: 우도(Likelihood) — A일 때 B가 관측될 확률
- **P(A)**: 사전 확률 — 관측 전 A의 확률
- **P(B)**: 주변 확률 — 정규화 상수

### 베이즈 갱신 (Bayesian Update)

새로운 데이터가 추가될수록 사후 확률이 점진적으로 갱신되는 특성. 이전 사후 확률이 다음 단계의 사전 확률이 됨.

### 나이브 베이즈 분류기

- 각 특징이 독립이라고 가정 (Naive 가정)
- 텍스트 분류, 스팸 필터에 효과적
- 학습 데이터가 적어도 동작 가능

### 빈도주의 vs 베이지안 통계

| 구분 | 빈도주의 | 베이지안 |
|------|----------|---------|
| 확률 해석 | 장기 빈도 | 믿음의 정도 |
| 사전 지식 | 미사용 | 사전 확률로 반영 |
| 추론 방법 | p-value, 신뢰구간 | 사후 분포 |

## 실무 적용 예시

의료 진단: 특정 질병의 유병률(사전 확률)과 검사 민감도(우도)를 결합하여 검사 양성 시 실제 환자일 확률(사후 확률) 계산. 위양성률을 고려한 정확한 판단 가능.
