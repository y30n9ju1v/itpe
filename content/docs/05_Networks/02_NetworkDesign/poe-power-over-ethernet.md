## 개요

PoE(Power over Ethernet)는 이더넷 케이블(UTP)을 통해 데이터와 전원을 동시에 전송하는 기술이다. IP 카메라, 무선 AP, IP 전화기 등 별도 전원 공급 없이 네트워크 케이블만으로 동작할 수 있어 배선 비용과 설치 복잡성을 크게 줄인다.

## 핵심 내용

### 가. 개념

- **정의**: IEEE 802.3 표준 기반, Cat5e 이상 이더넷 케이블로 데이터 + 직류 전력 동시 전송
- **구성**: PSE(Power Sourcing Equipment, 전력 공급 장치) → PD(Powered Device, 전력 수신 장치)
- **PSE 유형**: PoE 스위치, PoE 인젝터(미드스팬)

### 나. IEEE 표준

| 표준 | 연도 | 최대 전력(PSE) | PD 수신 전력 | 특징 |
|------|------|------------|------------|------|
| **IEEE 802.3af** | 2003 | 15.4W | 12.95W | 초기 표준, IP 전화·카메라 |
| **IEEE 802.3at (PoE+)** | 2009 | 30W | 25.5W | 무선 AP, PTZ 카메라 |
| **IEEE 802.3bt Type 3 (PoE++)** | 2018 | 60W | 51W | 고성능 AP, 스마트 조명 |
| **IEEE 802.3bt Type 4 (PoE++)** | 2018 | 100W | 71.3W | 노트북, 디스플레이 |

### 다. 국내 기술기준

- **전기통신설비의 기술기준에 관한 규정**: PoE 설비는 과전류 보호, 접지 요건 준수
- **건축물 통신 설비 기준**: 공공건물 PoE 설계 시 Cat6 이상 케이블 권고
- **소방 관련**: PoE 기반 스프링클러 제어기 등 안전 요건 적용

### PoE 동작 과정

```
1. PSE: 탐지 신호 전송 (저전압)
2. PD: 저항값으로 응답 (PoE 지원 여부)
3. PSE: 전력 클래스 협상 (0~8 클래스)
4. 전력 공급 시작 (Direct current over pairs)
5. 연결 해제 시 자동 차단
```

### 활용 분야

| 분야 | 활용 |
|------|------|
| 보안 | IP 카메라, 출입 통제 리더기 |
| 통신 | IP 전화기, 무선 AP |
| IoT | 스마트 조명, 환경 센서 |
| 의료 | 침대 콘솔, 의료기기 모니터 |

## 출제 포인트

- IEEE 802.3af/at/bt 3단계 표준과 전력 용량 비교
- PSE(공급)·PD(수신) 역할 구분
- 탐지(Detection) → 협상(Classification) → 전력 공급 순서
