## 개요

자기회귀모형(AR: Autoregressive Model)과 이동평균모형(MA: Moving Average Model)은 시계열 데이터 분석의 핵심 모형으로, 과거 값 또는 과거 오차를 이용해 현재 값을 예측한다. 두 모형을 결합한 ARIMA가 실무에서 널리 사용된다.

## 핵심 내용

### 자기회귀모형 AR(p)

```
X_t = c + φ₁X_{t-1} + φ₂X_{t-2} + ... + φ_pX_{t-p} + ε_t
```

- 현재 값 = 과거 p개 시점의 값의 선형 결합 + 백색잡음(ε)
- **p**: 차수(사용하는 과거 시점의 수)
- 특성: 자기상관함수(ACF)가 서서히 감소, 편자기상관함수(PACF)가 p 이후 절단

### 이동평균모형 MA(q)

```
X_t = μ + ε_t + θ₁ε_{t-1} + θ₂ε_{t-2} + ... + θ_qε_{t-q}
```

- 현재 값 = 과거 q개 시점의 오차항의 선형 결합
- **q**: 차수(사용하는 과거 오차의 수)
- 특성: ACF가 q 이후 절단, PACF가 서서히 감소

### AR vs MA 비교

| 구분 | AR(p) | MA(q) |
|------|-------|-------|
| 구성 요소 | 과거 관측값 | 과거 오차항 |
| ACF | 서서히 감소 | q 이후 절단 |
| PACF | p 이후 절단 | 서서히 감소 |
| 적합 데이터 | 관성이 큰 시계열 | 외부 충격 반응 |

### ARMA, ARIMA, SARIMA

- **ARMA(p,q)**: AR + MA 결합
- **ARIMA(p,d,q)**: d차 차분으로 비정상 시계열을 정상화 후 ARMA 적용
- **SARIMA**: 계절성(Seasonality) 추가

### 모형 선택 절차

1. 시각화 → 정상성 확인
2. ADF 검정 → 비정상이면 차분
3. ACF/PACF 분석 → p, q 결정
4. 모형 적합 → AIC/BIC로 최적 모형 선택
5. 잔차 검정 → 백색잡음 확인
