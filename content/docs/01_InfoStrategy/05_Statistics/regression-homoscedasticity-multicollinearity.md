## 개요

회귀모형의 신뢰성을 확보하기 위해 오차의 등분산성(Homoscedasticity)과 다중공선성(Multicollinearity) 가정을 검증해야 한다. 등분산성은 오차 분산의 일정성, 다중공선성은 독립변수 간 강한 상관관계 문제다.

## 핵심 내용

### 등분산성 (Homoscedasticity)

**개념**: 모든 독립변수 값에 대해 오차(잔차)의 분산이 일정한 성질

- **등분산성 만족**: Var(εᵢ) = σ² (상수)
- **이분산성(Heteroscedasticity)**: Var(εᵢ) = f(Xᵢ) → 회귀계수 비효율, 검정 신뢰도 저하

**검정 방법**:

| 방법 | 설명 |
|------|------|
| 잔차 산점도 | X축=예측값, Y축=잔차. 무작위 패턴이면 등분산 |
| Breusch-Pagan 검정 | 잔차 제곱에 대한 회귀 검정 |
| White 검정 | 비모수적 이분산 검정 |

**해결 방법**:
- 종속변수 변환 (로그·제곱근 변환)
- 가중최소제곱법(WLS, Weighted Least Squares) 적용

### 다중공선성 (Multicollinearity)

**개념**: 회귀모형의 두 개 이상의 독립변수 간 강한 선형 관계 존재

- 회귀계수 추정 불안정, 표준오차 과대 → 유의성 검정 신뢰도 저하

**탐지 방법**:

| 방법 | 기준 |
|------|------|
| **VIF(분산팽창지수)** | VIF > 10이면 심각한 다중공선성 |
| 상관계수 행렬 | |r| > 0.9이면 의심 |
| 조건 지수(CI) | CI > 30이면 심각 |

**VIF 공식**:
```
VIFⱼ = 1 / (1 - Rⱼ²)
```
- Rⱼ²: j번째 독립변수를 나머지 독립변수로 회귀했을 때 결정계수

**해결 방법**:
1. 상관성 높은 변수 제거
2. 주성분 분석(PCA)으로 변수 합성
3. 릿지 회귀(Ridge Regression): L2 정규화로 계수 안정화
4. 표준화(Standardization) 적용

### 비교 요약

| 구분 | 등분산성 | 다중공선성 |
|------|---------|----------|
| 관련 대상 | 오차(잔차) | 독립변수 간 관계 |
| 주요 검정 | Breusch-Pagan, 잔차 산점도 | VIF, 상관행렬 |
| 해결 | WLS, 변수 변환 | 변수 제거, PCA, 릿지 회귀 |

## 출제 포인트

- 등분산성: 잔차 분산의 일정성, 이분산 시 WLS 적용
- VIF > 10: 다중공선성 기준
- 릿지 회귀: 다중공선성 해결의 실무 대안
