## 개요

불편추정량(Unbiased Estimator)은 모수(Parameter)를 추정할 때 추정량의 기댓값이 실제 모수와 일치하는 추정량이다. 즉, 반복 표본추출 시 추정 오차가 체계적으로 한 방향으로 치우치지 않는 추정량이다.

## 핵심 내용

### 정의

추정량 T̂가 모수 θ의 불편추정량이 되려면:

```
E(T̂) = θ
```

- **편향(Bias)**: B(T̂) = E(T̂) - θ
- 불편추정량: Bias = 0

### 대표적인 불편추정량

| 추정 대상 | 불편추정량 | 편향추정량 |
|-----------|-----------|-----------|
| 모평균 μ | 표본평균 X̄ | - |
| 모분산 σ² | 표본분산 S² = Σ(Xi-X̄)²/(n-1) | MLE 분산 = Σ(Xi-X̄)²/n |
| 모비율 p | 표본비율 p̂ | - |

### 표본분산의 불편성: 왜 (n-1)인가?

- 표본평균 X̄를 사용하면 자유도(Degree of Freedom)가 1 감소
- n으로 나누면 E(S²) < σ² → 편향(과소 추정)
- **(n-1)로 나누면** E(S²) = σ² → 불편 추정량 확보

### 불편성 vs 효율성 트레이드오프

- 불편성: 편향 없음, 그러나 분산이 클 수 있음
- 효율성: MSE(Mean Squared Error) = Bias² + Variance 최소화
- 실무에서는 편향-분산 트레이드오프(Bias-Variance Tradeoff) 고려

### 일치추정량(Consistent Estimator)과의 비교

| 구분 | 불편추정량 | 일치추정량 |
|------|-----------|-----------|
| 조건 | E(T̂) = θ | n→∞ 시 T̂→θ(확률수렴) |
| 관계 | 불편이라도 비일치 가능 | 일치라도 편향 가능 |

## 출제 포인트

- 불편추정량의 정의: E(T̂) = θ
- 표본분산에서 (n-1) 사용 이유: 자유도 보정
- 불편성·효율성·일치성의 좋은 추정량 요건
