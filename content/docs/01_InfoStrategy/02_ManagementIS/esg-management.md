## 개요

ESG(Environment·Social·Governance) 경영은 재무적 성과와 함께 환경·사회·지배구조 측면의 지속가능성을 통합 추구하는 경영 패러다임이다. IT는 ESG 목표 달성을 위한 데이터 수집·분석·자동화 수단이자, 그 자체로 환경 영향을 미치는 주체이기도 하다.

## 핵심 내용

### 가. ESG 경영의 정의 및 목표

**정의**: 기업 경영에서 환경(E)·사회(S)·지배구조(G) 요소를 전략적으로 통합 관리하여 장기적 지속가능성과 이해관계자 가치를 창출하는 경영 체계

**3대 축**

| 축 | 핵심 과제 |
|----|-----------|
| **E (환경)** | 탄소중립, 에너지 효율, 기후 위험 관리, 순환경제 |
| **S (사회)** | 다양성·포용성, 공급망 윤리, 지역사회 기여, 정보보안 |
| **G (지배구조)** | 이사회 독립성, 투명한 공시, 반부패, IT 거버넌스 |

### 나. ESG 경영의 주요 지표

**환경(E) 지표**

| 지표 | 설명 |
|------|------|
| Scope 1·2·3 배출량 | 직접·간접·공급망 온실가스 배출량 (GHG Protocol) |
| 에너지 집약도 | 매출 1억원당 에너지 소비량 (MWh) |
| 수자원 사용량 | 데이터센터 WUE(Water Usage Effectiveness) |
| 폐기물 재활용율 | IT 장비 폐기 시 재활용 비율 |

**사회(S) 지표**

| 지표 | 설명 |
|------|------|
| 정보보안 사고 건수 | 데이터 침해·유출 발생율 |
| 디지털 포용 지수 | 취약계층 IT 접근성 제공 현황 |
| 개인정보 처리 준수율 | GDPR·개인정보보호법 위반 건수 |

**지배구조(G) 지표**

| 지표 | 설명 |
|------|------|
| IT 거버넌스 성숙도 | COBIT 기준 프로세스 성숙도 |
| 사이버보안 투자 비율 | IT 예산 중 보안 투자 비율 |
| 공시 정확도 | GRI·TCFD 기준 ESG 보고서 품질 |

### 다. ESG 목표 달성을 지원하는 정보기술(IT)

**환경(E) 지원 IT**

| IT | 역할 |
|----|------|
| **에너지 관리 시스템(EMS)** | IoT 센서로 실시간 에너지 소비 모니터링·최적화 |
| **탄소 회계 플랫폼** | Scope 1·2·3 배출량 자동 수집·산출·보고 |
| **그린 클라우드** | 재생에너지 기반 데이터센터, PUE 최적화 |
| **디지털 트윈** | 공장·건물 에너지 시뮬레이션으로 탄소 절감 |

**사회(S) 지원 IT**

| IT | 역할 |
|----|------|
| **개인정보보호 기술(PET)** | 동형암호·차분 프라이버시로 데이터 안전 처리 |
| **AI 공정성 도구** | 알고리즘 편향 탐지·수정 |
| **디지털 접근성** | 웹 접근성(WCAG) 준수, AI 기반 자막·번역 |

**지배구조(G) 지원 IT**

| IT | 역할 |
|----|------|
| **ESG 데이터 플랫폼** | GRI·SASB·TCFD 기준 통합 공시 자동화 |
| **블록체인** | 공급망 추적, 탄소크레딧 투명성 보장 |
| **IT 감사 자동화** | SIEM, SOAR로 지속적 컴플라이언스 모니터링 |

## 출제 포인트

- ESG 3축 정의와 각 축의 핵심 과제
- Scope 1·2·3 배출량 구분 (GHG Protocol)
- IT가 E·S·G 각각에 기여하는 방식 구분
- 데이터센터 자체의 환경 영향(PUE, WUE)
