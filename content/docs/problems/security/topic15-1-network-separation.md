## 해설

### 출제 배경 및 의도

망분리는 공공기관 및 금융기관의 법적 의무 사항이다. 물리적·논리적 망분리의 구현 방식과 망연계 자료전송 기술(망간자료전송, 보안USB 등)을 법률 근거와 함께 서술해야 한다.

### 1. 망분리 법률적 개념 및 특징

- **법적 근거**: 정보통신망법, 전자금융감독규정, 개인정보보호법
- **의무 대상**: 공공기관(국가기관, 지자체), 금융기관, 개인정보 대량 처리 사업자

| 구분 | 인터넷망 | 업무망 |
|------|---------|--------|
| 목적 | 외부 인터넷 접속 | 내부 업무 시스템 운영 |
| 분리 이유 | 인터넷 경유 악성코드 침입 차단 | 중요 데이터 보호 |
| 접근 제한 | 업무망에서 인터넷 직접 접속 불가 | 허가된 경로로만 자료 이동 |

### 2. 물리적 망분리 vs 논리적 망분리

1) 물리적 망분리

- **방법**: 인터넷용 PC와 업무용 PC를 완전히 별도로 분리·설치
- **특징**: 완전한 격리, 높은 보안 수준, 높은 구축 비용

```
인터넷 PC (인터넷망) ──┐
                        ├── 망연계 장비 ──[자료 이동] ── 업무 PC (업무망)
업무 PC (업무망)   ───┘
```

2) 논리적 망분리

- **방법**: 한 PC에서 가상화(VDI)나 OS 이중화로 인터넷·업무 환경 분리

| 기술 | 설명 | 특징 |
|------|------|------|
| **VDI(Virtual Desktop Infrastructure)** | 서버의 가상 데스크톱에서 업무 수행 | 중앙 관리 용이, 네트워크 의존 |
| **OS 이중화(듀얼부팅)** | 인터넷용·업무용 OS를 부팅 시 선택 | 완전 분리, 전환 불편 |
| **가상화 소프트웨어** | VM에서 인터넷·업무 환경 분리 실행 | 유연, 비용 낮음 |
| **SBC(Server-Based Computing)** | 서버에서 실행, 화면만 전송 | 씬클라이언트 환경 |

### 3. 2023년 이후 망분리 규제 완화 동향

| 구분 | 변화 내용 |
|------|-----------|
| **공공기관 클라우드** | 행정·공공기관의 민간 클라우드(SaaS) 업무망 활용 확대 (행안부 디지털 서비스 이용 지원에 관한 규정 개정) |
| **금융권 망분리** | 금융위·금감원 2023년 규제 완화: 생산성 업무(이메일·문서협업)에 한해 인터넷망 접근 일부 허용 |
| **제로 트러스트 대체** | "물리적 망분리 → ZTNA(Zero Trust Network Access) 기반 논리적 분리" 전환 정책 추진 |
| **클라우드 보안 인증** | CC인증·CSAP(클라우드 보안인증)를 받은 서비스에 한해 업무망 대체 허용 확대 |

- **배경**: 원격근무 확산 및 클라우드 전환으로 물리적 망분리의 현실적 한계가 부각되어 보안 수준을 유지하면서도 업무 생산성을 높이는 방향으로 규제가 점진적으로 완화되는 추세임.

### 4. 망연계 자료전송 기술

| 기술 | 설명 | 보안 특성 |
|------|------|-----------|
| **망간자료전송 시스템** | 승인된 파일만 일방향으로 전송 | 콘텐츠 검사, 승인 워크플로우 |
| **단방향 데이터 다이오드** | 물리적으로 단방향만 전송 가능한 장비 | 역방향 통신 물리적 차단 |
| **보안 USB** | 암호화된 승인된 USB만 허용 | 매체 제어 정책 연동 |
| **가상화 기반 파일 전송** | 샌드박스에서 파일 검사 후 이동 | 악성코드 필터링 |
| **CDR(Content Disarm & Reconstruction)** | 파일의 위험 요소 제거 후 재구성 | 악성 매크로·링크 제거 |


- 망분리는 내부망 보호의 핵심 통제이나, 2023년 이후 클라우드 활용 확대와 원격근무 정착으로 제로 트러스트(ZTNA) 기반의 논리적 분리로 패러다임이 전환되고 있으며, 보안 수준을 유지하면서도 업무 연속성을 확보하는 방향으로 정책이 진화 중임.  "끝"

### 실무 제언

**원격근무 망분리 이슈 해결**
- **챌린지**: COVID-19 이후 재택근무가 확산되면서 물리적 망분리가 현실적으로 어려워졌다.
- **제언**: 클라우드 기반 VDI(Citrix Cloud, AWS WorkSpaces)를 도입하여 원격에서도 업무망에 안전하게 접속하고, 제로 트러스트 기반의 조건부 접근(디바이스 상태 검증, MFA)을 결합하여 논리적 망분리 수준의 보안을 원격 환경에서 구현해야 한다.
