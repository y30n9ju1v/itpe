## 문제

최근 한국, 영국, 미국 등 32개국 수사기관이 다크웹(dark web)에 개설된 아동음란물 사이트를 공조 수사해 사이트 이용자 300여 명을 검거했다. 이 중 223명이 한국인으로 범죄의 온상이 되고 있어 국가 차원의 대응이 필요한 상황이다. 다크웹 범죄현상에 대하여 다음에 답하시오.
가. 다크웹 범죄현상과 문제점
나. 다크웹 구축기술 및 추적기술
다. 다크웹에 대한 국내외 규제와 한계 및 기술적, 정책적 대응방안

## 해설

### 1. 다크웹 개요 및 범죄현상

**인터넷 계층 구분:**

| 계층 | 설명 | 접근 방법 |
|------|------|-----------|
| **서피스 웹(Surface Web)** | 일반 검색엔진이 인덱싱하는 공개 웹 | 일반 브라우저 |
| **딥 웹(Deep Web)** | 검색엔진 미인덱싱(로그인 필요, 내부망 등) | 일반 브라우저 + 인증 |
| **다크 웹(Dark Web)** | 특수 소프트웨어(Tor) 없이 접근 불가한 익명 네트워크 | Tor 브라우저 |

**주요 범죄현상:**
- 불법 마약·무기 거래(Silk Road 등 다크마켓)
- 아동성착취물(CSAM) 유통
- 랜섬웨어 RaaS 판매
- 탈취 신용카드·개인정보 판매
- 사이버 용병 서비스(해킹 의뢰)

### 2. 다크웹 구축기술 — Tor(The Onion Router)

```
[Tor 양파 라우팅 원리]
사용자 ──암호화3──▶ 입구 노드(가드)
         ──암호화2──▶ 중간 노드(릴레이)
         ──암호화1──▶ 출구 노드 ──▶ 목적지

각 노드는 이전/다음 노드만 알고, 전체 경로 불인지
→ 3겹 암호화로 출처 은닉
```

| 기술 | 설명 |
|------|------|
| **Tor(The Onion Router)** | 다층 암호화·릴레이 노드 경유로 익명성 보장 |
| **.onion 도메인** | Tor 내부의 숨겨진 서비스(Hidden Service) |
| **I2P(Invisible Internet Project)** | Tor와 유사한 익명 네트워크 |
| **Freenet** | 분산 P2P 기반 익명 파일 공유 |

### 3. 다크웹 추적기술

| 기술 | 설명 |
|------|------|
| **출구 노드 모니터링** | 출구 노드에서 트래픽 분석 |
| **트래픽 분석** | 타이밍 상관관계 분석으로 입구·출구 연결 |
| **비트코인 추적** | 블록체인 분석(CipherTrace, Chainalysis) |
| **법적 협조** | Tor 서버 압수, MLAT(국제사법공조) |
| **운영 보안 취약점** | 사용자 실수(로그인, 사진 메타데이터) 분석 |

### 4. 국내외 규제와 대응방안

| 구분 | 내용 |
|------|------|
| 국내 | 불법정보 유통 접속 차단(방심위), 사이버범죄수사대 운영 |
| 해외 | Europol, FBI 공조 작전(Operation Onymous 등) |
| 한계 | 국제 관할권 문제, Tor 기술 자체의 합법성, 암호화폐 추적 어려움 |

**기술적 대응:** AI 기반 다크웹 모니터링, 비트코인 추적 분석, 출구 노드 설치 및 감시

**정책적 대응:** MLAT 강화, 사이버범죄 국제협약(부다페스트 협약) 가입, 취약계층 교육.  "끝"

### 실무 제언

**다크웹 위협 인텔리전스 활용**
- **챌린지**: 기업 데이터가 다크웹에서 유통되고 있는지 인지하지 못하면 사전 대응이 불가능하다.
- **제언**: 다크웹 모니터링 서비스(Recorded Future, Digital Shadows 등)를 구독하여 자사 도메인, 임직원 이메일, 고객 데이터 유출 여부를 지속 모니터링하고, 유출 탐지 즉시 비밀번호 초기화와 이해관계자 통보 절차를 포함한 IRP를 가동해야 한다.
