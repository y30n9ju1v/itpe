## 해설

### 1. AIoT 보안 개요

- **정의**: AI(인공지능) 기능이 IoT 디바이스에 내장되거나 엣지 컴퓨팅으로 연계되어 지능형 서비스를 제공하는 환경(AIoT)에서의 보안
- **특성**:
  - **분산성**: 수백만 대의 디바이스가 분산 배치
  - **이기종성**: 다양한 제조사·OS·프로토콜
  - **자원 제약**: MCU 기반 저전력 디바이스는 암호화·인증 기능 제한
  - **물리적 접근 취약**: 무인 환경에 배치, 물리적 변조 위험

### 2. AIoT 보안 위협 및 취약점

| 계층 | 위협 | 예시 |
|------|------|------|
| **디바이스** | 하드웨어 백도어, 디폴트 자격증명 | Mirai 봇넷(라우터 기본 PW) |
| | 펌웨어 추출·분석 | JTAG 디버그 포트 노출 |
| | 물리적 변조 | 공급망 공격, 칩 교체 |
| **통신** | 평문 전송 | HTTP, MQTT 미암호화 |
| | 프로토콜 취약점 | Zigbee·Z-Wave 취약한 페어링 |
| | 중간자 공격 | TLS 미검증 인증서 |
| **AI 모델** | 적대적 예제(Adversarial Examples) | 카메라 AI를 속이는 스티커 |
| | 모델 탈취(Model Stealing) | API 반복 쿼리로 모델 복제 |
| | 데이터 포이즈닝 | 학습 데이터 오염으로 판단 오류 |
| **클라우드** | API 보안 취약점 | BOLA, 인증 취약점 |
| | 데이터 유출 | IoT 수집 개인정보 노출 |

### 3. AIoT 디바이스 보안 기술

#### 가. 하드웨어 보안

| 기술 | 내용 |
|------|------|
| **TPM(Trusted Platform Module)** | 보안 키 저장, 원격 증명(Remote Attestation) |
| **Secure Boot** | 부팅 시 펌웨어 서명 검증, 무결성 확인 |
| **HSM(Hardware Security Module)** | 암호화 키 물리적 보호 |
| **물리적 복제 방지(PUF)** | 칩 고유 물리 특성으로 디바이스 인증 |
| **디버그 포트 비활성화** | JTAG·UART 포트 생산 시 잠금 |

#### 나. 통신 보안

| 기술 | 내용 |
|------|------|
| **경량 TLS(TLS 1.3)** | IoT 자원 제약 환경용 최적화 TLS |
| **DTLS** | UDP 기반 IoT 통신 암호화 |
| **CoAP+DTLS** | 저전력 IoT 프로토콜 보안 |
| **mTLS** | 디바이스-서버 양방향 인증 |
| **MQTT over TLS** | IoT 메시징 프로토콜 암호화 |

#### 다. AI 모델 보안

| 기술 | 내용 |
|------|------|
| **모델 암호화** | 엣지 배포 모델 암호화 저장 |
| **Federated Learning** | 원본 데이터 공유 없이 분산 학습 |
| **적대적 훈련** | 적대적 예제 포함 학습으로 내성 강화 |
| **TEE(Trusted Execution Environment)** | ARM TrustZone에서 AI 추론 격리 실행 |

#### 라. 디바이스 수명주기 보안

```
제조 → 공급망 보안, 펌웨어 서명, 기본 자격증명 제거
설치 → 디바이스 등록, 인증서 자동 프로비저닝(SCEP/EST)
운영 → OTA(Over-The-Air) 보안 업데이트, 이상 탐지
폐기 → 암호화 키 안전 삭제, 개인정보 초기화
```

### 실무 제언

**스마트홈 AIoT 환경 보안 설계**
- **챌린지**: 스마트홈 기기는 비전문가 사용자가 디폴트 비밀번호를 변경하지 않아 봇넷에 감염되거나, 업데이트 기능이 없어 취약점이 수년간 방치된다.
- **제언**: IoT 기기 제조 단계에서 ① 고유 자격증명(공장 초기화 시 랜덤 생성), ② Secure Boot, ③ OTA 자동 업데이트 기능을 의무화하고, 출시 후 최소 5년간 보안 패치를 보장하는 ETSI EN 303 645(IoT 보안 표준)를 준수하며, 네트워크 레벨에서 홈 게이트웨이의 IoT 기기 전용 VLAN을 분리하여 감염 시 측면 이동을 차단해야 한다.
