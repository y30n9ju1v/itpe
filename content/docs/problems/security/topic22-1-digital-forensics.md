## 문제

사이버 범죄와 지능범죄가 증가함에 따라 정보기기에 대한 디지털포렌식 기술의 중요성이 높아지고 있다. 범죄수사에 활용되는 디지털 포렌식에 대해서 다음 물음에 답하시오.
가. 디지털포렌식의 절차를 설명하시오.
나. 디지털포렌식 기술을 설명하시오.
다. 포렌식 대응 기술인 안티포렌식에 대해서 설명하시오.

## 해설

### 1. 디지털 포렌식 개요

정 의  • 디지털 기기에 저장된 데이터를 법적으로 유효한 증거로 수집·분석·보존하는 과학적 방법론
- **원칙**:
  - **재현성**: 동일한 방법으로 동일한 결과 도출
  - **무결성**: 원본 증거 변조 없음 증명(해시값)
  - **연계 보관성(Chain of Custody)**: 증거 수집·이동·보관 전 과정 기록

### 2. 디지털 포렌식 절차

```
① 사전 준비
   영장 확보, 포렌식 도구 준비, 팀 구성

② 현장 대응 (First Response)
   시스템 현황 사진 촬영, 휘발성 데이터 우선 수집
   (메모리 > 임시 파일 > 네트워크 상태 > 레지스트리)

③ 수집 (Collection)
   원본 디스크 이미징(bit-by-bit copy), 해시값 기록
   도구: FTK Imager, dd, Autopsy

④ 보존 (Preservation)
   증거 이중 보관(원본 봉인), 변조 방지
   해시(MD5/SHA-256)로 무결성 검증

⑤ 분석 (Examination & Analysis)
   파일 시스템 분석, 삭제 파일 복구, 타임라인 분석
   메타데이터, 레지스트리, 이메일, 브라우저 이력 분석

⑥ 보고 (Reporting)
   기술적·법률적 보고서 작성, 법정 증언 자료 준비
```

### 3. 디지털 포렌식 기술

| 기술 | 설명 |
|------|------|
| **디스크 포렌식** | 파일 시스템 분석, 삭제 파일 복구, 파티션 분석 |
| **메모리 포렌식** | RAM 덤프에서 실행 중 프로세스·키·패스워드 추출 |
| **네트워크 포렌식** | 패킷 캡처 분석, 네트워크 흐름 재구성 |
| **모바일 포렌식** | 스마트폰 메시지·위치·앱 데이터 추출 |
| **클라우드 포렌식** | 클라우드 로그·메타데이터 분석 |
| **이메일 포렌식** | 헤더 분석, 발신자 추적 |
| **스테가노그래피 탐지** | 이미지·음성에 숨겨진 데이터 탐지 |

### 4. 안티포렌식(Anti-Forensics)

정 의  • 디지털 포렌식 수사를 방해하기 위해 증거를 파기·위변조하거나 탐지를 어렵게 하는 기술

| 기법 | 설명 |
|------|------|
| **데이터 삭제** | 안전 삭제(DoD 7회 덮어쓰기), SSD TRIM |
| **암호화** | 전체 디스크 암호화(BitLocker, VeraCrypt) |
| **스테가노그래피** | 이미지·음성에 데이터 은닉 |
| **메타데이터 위조** | 타임스탬프 변조, MAC 주소 스푸핑 |
| **루트킷** | OS 수준에서 프로세스·파일 은닉 |
| **포렌식 도구 무력화** | 포렌식 소프트웨어 탐지 후 종료 |


- 디지털 포렌식은 수집→보존→분석→보고의 절차에서 무결성(해시값 검증)·연속성(Chain of Custody)·재현성을 확보해야 법적 증거 능력이 인정되며, 클라우드·IoT·암호화 환경의 확산으로 휘발성 데이터 수집과 암호화된 저장매체 분석 기술이 핵심 과제임.  "끝"

### 실무 제언

**클라우드 환경 포렌식 대응**
- **챌린지**: 클라우드 기반 시스템에서 보안 사고 발생 시 물리적 디스크 접근이 불가능하여 전통적 포렌식 방법이 적용되기 어렵다.
- **제언**: AWS CloudTrail, Azure Monitor, GCP Cloud Audit Logs를 불변 스토리지에 장기 보관하고, SIEM과 연동하여 사고 발생 시 신속히 로그를 추출할 수 있는 클라우드 포렌식 레디 아키텍처를 미리 구축해야 한다.
