## 해설

### 출제 배경 및 의도

QKD는 양자역학 원리를 이용해 물리적으로 안전한 키 분배를 실현하는 기술이다. 기존 공개키 암호의 수학적 난제 의존 방식과 달리 정보이론적 안전성을 제공한다. 국내 QKD 표준화 동향과 실제 적용 사례까지 제시해야 한다.

### 1. QKD 개요

| 항목 | 내용 |
|------|------|
| 정의 | 양자역학 원리를 이용해 도청 탐지가 보장된 방식으로 비밀키를 분배하는 기술 |
| 물리적 원리 | 양자 불확정성 원리 + 복제 불가 정리 |
| 안전성 | 정보이론적 안전성(ITS: Information-Theoretic Security) — 계산 능력에 무관 |
| 한계 | 거리 제한(광섬유 약 100~200km), 고비용, 전용 인프라 필요 |

### 2. QKD 주요 프로토콜

| 프로토콜 | 제안 | 특징 |
|----------|------|------|
| **BB84** | 1984, Bennett·Brassard | 편광 기반, 4개 상태 |
| **B92** | 1992, Bennett | 2개 상태만 사용, 단순화 |
| **E91** | 1991, Ekert | 양자 얽힘 기반, 벨 부등식 검증 |
| **MDI-QKD** | 2012, Lo 외 | 측정장치 독립, 측정기 공격 면역 |
| **TF-QKD** | 2018, Lucamarini 외 | 거리 한계 극복 (수백 km) |

### 3. QKD 시스템 구성 요소

```
Alice                    양자 채널(광섬유)          Bob
[광자 발생기]──────────────────────────────▶[광자 검출기]
[난수 생성기]                                [기저 선택]
      │                                           │
      └──────── 공개 채널(인증) ──────────────────┘
                 (기저 비교, 오류 정정, 비밀증폭)
```

### 4. QKD vs PQC (양자내성암호) 비교

| 구분 | QKD | PQC (Post-Quantum Cryptography) |
|------|-----|----------------------------------|
| 안전성 근거 | 양자역학(물리 법칙) | 수학적 난제(격자, 해시 등) |
| 안전성 유형 | 정보이론적(무조건적) | 계산론적 |
| 구현 방식 | 전용 광학 하드웨어 필요 | 소프트웨어 구현 가능 |
| 거리 제한 | 있음(현재 수백 km, 위성으로 확장) | 없음 |
| 비용 | 고비용 | 기존 인프라 활용 |
| 표준화 | ITU-T X.1700 시리즈 | NIST PQC(2024 공표) |

### 5. 국내외 QKD 동향

| 구분 | 내용 |
|------|------|
| 국내 | SK텔레콤·KT 상용 QKD 서비스, 국정원 QKD 보안요구사항 |
| 해외 | 중국 위성 QKD(묵자호, 2017), EU OpenQKD 프로젝트 |
| 표준 | ITU-T SG17, ETSI QKD 표준 그룹, IETF |


- QKD는 양자역학적 불확정성 원리를 활용해 도청 즉시 검출 가능한 키를 분배하며, BB84·E91 등 프로토콜과 ITU-T·ETSI 표준화가 진행 중이나 전송 거리 한계를 양자중계기로 극복하는 것이 상용화의 핵심 과제임.  "끝"

### 실무 제언

**하이브리드 QKD+PQC 전략**
- **챌린지**: QKD는 거리 제한과 고비용으로 단기간 전면 도입이 어렵다.
- **제언**: 핵심 보안 구간(국가기반시설, 금융망 코어 구간)에는 QKD를 우선 도입하고, 나머지 구간은 NIST PQC 표준 알고리즘(ML-KEM, ML-DSA)으로 대체하는 하이브리드 전략을 채택해야 한다.
