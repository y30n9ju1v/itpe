## 해설

### 출제 배경 및 의도

양자컴퓨팅의 발전으로 기존 공개키 암호(RSA, ECC)가 위협받고 있다. 양자암호의 물리적 원리(양자역학 기반)와 BB84 프로토콜의 단계별 동작 방식을 서술할 수 있어야 한다.

### 1. 양자암호시스템 개요

- **원리**: 양자역학의 두 특성을 활용
  - **불확정성 원리**: 양자 상태는 측정 시 반드시 변화 → 도청 탐지 가능
  - **복제 불가 정리(No-Cloning Theorem)**: 임의의 양자 상태를 완벽 복제 불가
- **목적**: 물리적으로 안전한 키 교환(QKD: Quantum Key Distribution)
- **특징**: 도청 시도가 반드시 오류를 발생시켜 탐지 가능

### 2. BB84 프로토콜 동작 방식

1984년 Bennett과 Brassard가 제안한 최초의 QKD 프로토콜.

**사용 기저(Basis):**
- ⊕ 기저: 0°, 90° 편광 (|0⟩, |1⟩)
- ⊗ 기저: 45°, 135° 편광 (|+⟩, |−⟩)

```
[단계 1] Alice가 랜덤 비트와 랜덤 기저로 광자 전송
비트:  0  1  0  1  0  1  1  0
기저:  ⊕  ⊗  ⊗  ⊕  ⊗  ⊕  ⊕  ⊗
편광: →  ↗  ↖  ↓  ↗  →  ↓  ↖

[단계 2] Bob이 랜덤 기저로 광자 측정
기저:  ⊗  ⊗  ⊕  ⊕  ⊗  ⊗  ⊕  ⊗
결과:  ?  1  ?  1  0  ?  1  0

[단계 3] 공개 채널로 기저 비교 (비트값은 공개 안함)
Alice: ⊕  ⊗  ⊗  ⊕  ⊗  ⊕  ⊕  ⊗
Bob:   ⊗  ⊗  ⊕  ⊕  ⊗  ⊗  ⊕  ⊗
일치:  ✗  ✓  ✗  ✓  ✓  ✗  ✓  ✓

[단계 4] 기저 일치 비트만 키로 사용
공유 키: 1   1  0   1  0
```

**[단계 5] 도청 탐지:**
- 일부 키 비트를 공개 채널로 비교
- 도청자(Eve)가 측정 시 25% 오류 발생 → QBER(Quantum Bit Error Rate) 임계값 초과 시 통신 폐기

### 3. QKD 시스템 구성

| 구성요소 | 역할 |
|----------|------|
| 광자 발생기 | 단일 광자 생성·편광 |
| 양자 채널 | 광섬유 또는 자유 공간 |
| 단일 광자 검출기 | 편광 측정 |
| 공개 채널(인증) | 기저 비교, 오류 정정 |

### 4. BB84 vs 현대 QKD 비교

| 항목 | BB84 | 현대 QKD(E91, MDI-QKD) |
|------|------|------------------------|
| 원리 | 단일 광자 편광 | 양자 얽힘 활용 |
| 거리 | 수백 km | 위성 QKD로 수천 km |
| 보안 증명 | 정보이론적 안전성 | 측정장치 독립 보안 |

### 실무 제언

**금융 네트워크 QKD 도입**
- **챌린지**: 양자컴퓨팅이 RSA-2048을 수시간 내 해독할 수 있게 되면 현재 금융 암호 인프라가 무력화된다.
- **제언**: 국내 통신사 QKD 서비스를 활용하여 금융망 핵심 구간(본·지점 간, DC 간)에 양자암호통신을 단계적으로 도입하고, 단기적으로는 PQC(양자내성암호)를 병행 적용하는 하이브리드 전략을 채택해야 한다.
