## 해설

### 1. ISA/IEC 62443 개요

- **정의**: 산업자동화제어시스템(IACS, Industrial Automation and Control System)의 사이버 보안을 위한 국제 표준 시리즈 (ISA99 위원회 개발 → IEC 62443으로 표준화)
- **대상**: SCADA, DCS, PLC 등 OT 시스템과 관련 네트워크·인원·프로세스

### 2. ISA/IEC 62443 표준 구성

| 시리즈 | 내용 | 대상 |
|--------|------|------|
| **IEC 62443-1.x** | 일반(General) — 용어, 개념, 보안 모델 | 공통 |
| **IEC 62443-2.x** | 정책 및 절차 — 운영자·서비스 제공자의 보안 관리 | Asset Owner, 서비스 제공자 |
| **IEC 62443-3.x** | 시스템 — Zone/Conduit 설계, 보안 위험 평가 | 시스템 통합자 |
| **IEC 62443-4.x** | 컴포넌트 — 제품 보안 요구사항, SDL | 제품 공급자 |

### 3. 보안 레벨(Security Level, SL)

| SL | 명칭 | 방어 대상 위협 |
|----|------|---------------|
| **SL 1** | 비의도적 위반(Casual) | 우발적 오류, 비전문 공격자 |
| **SL 2** | 의도적 단순(Simple) | 일반적인 사이버 공격자 |
| **SL 3** | 정교한 공격(Sophisticated) | 전문 해커, 내부자 위협 |
| **SL 4** | 국가 수준(Advanced) | 국가 지원 APT 그룹 |

### 4. Zone/Conduit 모델

```
┌───────────────────────────────────────┐
│         Enterprise Zone (SL1)         │
│  IT 네트워크, ERP                      │
│                  ↕ Conduit (SL2)      │
├───────────────────────────────────────┤
│        Control Zone (SL2)             │
│  SCADA, HMI, 히스토리안               │
│                  ↕ Conduit (SL3)      │
├───────────────────────────────────────┤
│        Safety Zone (SL3~4)            │
│  안전 계장 시스템(SIS), PLC           │
└───────────────────────────────────────┘
```

- **Zone**: 동일한 보안 요구사항을 가진 논리적·물리적 자산 그룹
- **Conduit**: Zone 간 데이터 흐름을 제어하는 통신 채널 (방화벽, 데이터 다이오드)

### 5. ISA/IEC 62443 적용 방안

| 단계 | 활동 |
|------|------|
| ① **위험 평가** | 62443-3-2: 자산 목록 작성 → 취약점 분석 → 위험 평가 |
| ② **Zone/Conduit 설계** | 기능·보안 요구사항 기반 Zone 분류, Conduit 설계 |
| ③ **보안 레벨 결정** | 각 Zone별 목표 SL(Target SL) 설정 |
| ④ **보안 대책 구현** | 방화벽, 단방향 게이트웨이, 패치 관리, 계정 관리 |
| ⑤ **운영 및 유지보수** | 취약점 모니터링, 정기 위험 재평가 |

### 실무 제언

**발전소·정수장 등 국가기반시설 ISA/IEC 62443 적용**
- **챌린지**: 국가기반시설의 OT 시스템은 24시간 365일 무중단 운영이 필수여서 보안 솔루션 도입을 위한 점검 시간 확보가 어렵고, 공급망(벤더) 보안까지 관리해야 한다.
- **제언**: ISA/IEC 62443-2-4 기준으로 유지보수 벤더의 보안 요구사항을 계약에 명시하고, 가상 패치(Virtual Patch) 방식의 네트워크 레벨 IPS를 적용하여 시스템 재기동 없이 알려진 취약점을 즉시 차단하며, SL 4가 요구되는 Safety Zone은 물리적 에어갭(Air Gap)을 유지해야 한다.
