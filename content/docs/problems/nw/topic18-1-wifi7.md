## 해설

> **Wi-Fi 7은 "무선 기가비트 광대역 인터넷"이다 — 유선 이더넷을 넘어서는 속도와 초저지연으로 AR/VR, 산업 자동화를 무선으로 실현한다**

### 1. 출제 배경 및 의도

Wi-Fi 7(IEEE 802.11be)은 2024년 상용화된 최신 Wi-Fi 표준입니다. 139회 관리 문제로 출제될 만큼 최신 기술 트렌드 파악이 중요합니다. 이론적 최대 속도 46Gbps, 지연 1ms 미만을 목표로 하는 Wi-Fi 7의 핵심 기술을 이전 세대와 비교하여 이해하는 것이 목표입니다.

### 2. 개념의 본질과 작동 메커니즘

#### Wi-Fi 세대별 진화

| 세대 | 표준 | 최대 속도 | 주파수 | 핵심 기술 |
|---|---|---|---|---|
| **Wi-Fi 4** | 802.11n | 600 Mbps | 2.4/5 GHz | MIMO, 40MHz 채널 |
| **Wi-Fi 5** | 802.11ac | 3.5 Gbps | 5 GHz | 8x8 MU-MIMO, 80/160MHz |
| **Wi-Fi 6** | 802.11ax | 9.6 Gbps | 2.4/5 GHz | OFDMA, TWT, BSS 색상 |
| **Wi-Fi 6E** | 802.11ax | 9.6 Gbps | 2.4/5/6 GHz | 6GHz 대역 추가 |
| **Wi-Fi 7** | 802.11be | 46 Gbps | 2.4/5/6 GHz | 320MHz, MLO, 4K-QAM |

#### Wi-Fi 7 핵심 기술

**1) 320MHz 채널 대역폭 (Channel Bandwidth)**

```
Wi-Fi 6:  80/160 MHz 채널
Wi-Fi 7:  320 MHz 채널 (6GHz 대역에서만)

채널 대역폭과 처리량 관계:
처리량 ∝ 채널 대역폭 × 변조 차수 × 공간 스트림 수
320MHz = 160MHz × 2배 → 처리량 2배
```

**2) 4K-QAM (4096-QAM)**

```
변조 방식 발전:
Wi-Fi 5: 256-QAM (8비트/심볼)
Wi-Fi 6: 1024-QAM (10비트/심볼)
Wi-Fi 7: 4096-QAM (12비트/심볼)

심볼당 더 많은 비트 → 같은 채널에서 더 많은 데이터
단, 신호 품질(SNR)이 높아야 작동 → 근거리 적합
```

**3) MLO (Multi-Link Operation)**

```
Wi-Fi 7의 가장 혁신적 기술!

기존: 한 번에 하나의 주파수 대역만 사용
      장치가 5GHz로 연결 중 → 2.4GHz, 6GHz 유휴

MLO: 여러 주파수 대역을 동시에 사용
     ┌── 2.4 GHz 링크 ───┐
     ├── 5 GHz 링크  ───┤ → 동시에 모두 사용!
     └── 6 GHz 링크  ───┘
     
장점:
- 처리량 합산 (최대 대역폭 총합)
- 로드밸런싱 (트래픽 분산)
- 지연 감소 (덜 혼잡한 링크 선택)
- 신뢰성 향상 (링크 장애 시 자동 전환)
```

**4) MRU (Multi-Resource Unit)**

```
Wi-Fi 6 OFDMA: 각 서브채널(RU)을 한 사용자에게만 할당
Wi-Fi 7 MRU: 비연속적 여러 RU를 하나의 사용자에게 할당

Wi-Fi 6:  [RU1: 사용자A] [RU2: 미사용] [RU3: 사용자B]
Wi-Fi 7:  [RU1+RU3: 사용자A] [RU2: 사용자B]
→ 비연속 채널 자원 효율적 활용
```

**5) 향상된 MU-MIMO**

```
Wi-Fi 6:  최대 8 공간 스트림 (8x8 MU-MIMO)
Wi-Fi 7:  최대 16 공간 스트림 (16x16 MU-MIMO)
→ 더 많은 동시 연결, 더 높은 공간 다중화
```

#### Wi-Fi 7 성능 요약

```
이론적 최대 처리량 계산:
4096-QAM × 5/6 코딩율 × 320MHz × 16 스트림 × ...
= 약 46 Gbps (이론값)
실제 환경: 10~20 Gbps 예상

목표 지연:
Wi-Fi 6: ~5ms
Wi-Fi 7: <1ms (MLO + 개선된 스케줄링)
```

#### Wi-Fi 7 활용 시나리오

| 시나리오 | 요구사항 | Wi-Fi 7 적용 |
|---|---|---|
| **AR/VR** | 초저지연(<1ms), 고처리량 | MLO + 6GHz 고속 |
| **8K 스트리밍** | 100Mbps+ per stream | 320MHz 고속 채널 |
| **산업 자동화** | 초신뢰, 결정론적 지연 | MLO 다중 경로 |
| **홈 네트워크** | 다수 디바이스 동시 | MU-MIMO 16개 스트림 |
| **게임** | 낮은 지터, 낮은 지연 | 6GHz 전용 + MLO |

### 3. 핵심 키워드 (Must-Have)

- **MLO (Multi-Link Operation):** Wi-Fi 7의 핵심 혁신, 다중 대역 동시 사용
- **4096-QAM:** 12비트/심볼, 1024-QAM 대비 20% 처리량 향상
- **320MHz:** Wi-Fi 7 최대 채널 폭 (Wi-Fi 6의 2배)
- **6 GHz 대역:** Wi-Fi 6E에서 도입, 혼잡 적고 광대역
- **MRU:** 비연속 주파수 자원 조합 활용

### 4. 맥락과 비교

**Wi-Fi 6 vs Wi-Fi 7 핵심 비교:**

| 항목 | Wi-Fi 6 (11ax) | Wi-Fi 7 (11be) |
|---|---|---|
| **최대 속도** | 9.6 Gbps | 46 Gbps |
| **채널 폭** | 160 MHz | 320 MHz |
| **변조** | 1024-QAM | 4096-QAM |
| **MIMO** | 8×8 | 16×16 |
| **다중 링크** | 없음 | MLO (혁신) |
| **목표 지연** | ~5ms | <1ms |
| **주파수** | 2.4/5 GHz | 2.4/5/6 GHz |

Wi-Fi 7은 5G NR의 경쟁 기술로, 기업 내 초고속 무선 환경에서 5G 대비 비용 효율적입니다.

### 5. 실무 제언

기업 Wi-Fi 7 도입 시 6GHz 대역을 적극 활용하기 위해 기존 2.4/5GHz AP를 교체해야 합니다. MLO의 이점을 극대화하려면 클라이언트 단말(노트북, 스마트폰)도 Wi-Fi 7 지원 NIC가 필요합니다. 현재(2026년) Wi-Fi 7 AP는 주요 벤더(Cisco, Aruba, TP-Link)에서 상용 출시되었으며, 고성능 기업망, 병원, 스마트팩토리 환경에 선도 적용 중입니다. 기술사 답안에서 MLO의 원리와 장점, 이전 세대 비교표, 실제 활용 시나리오를 체계적으로 서술하면 최신 기술 이해를 명확히 보여줄 수 있습니다.
