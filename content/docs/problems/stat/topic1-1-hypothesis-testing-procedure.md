## 문제

통계적 가설 검정(Statistical Hypothesis Test)의 절차

## 해설

### 출제 배경 및 의도

데이터 기반 의사결정 시대에 통계적 가설검정은 AI/ML 모델 검증, A/B 테스트, 품질 관리 등 실무 전반에 활용됩니다. 채점관은 단순한 절차 암기가 아닌, 각 단계의 의미와 상호 관계, 실무 적용 능력을 평가합니다. 특히 귀무가설과 대립가설의 설정 논리, 1·2종 오류의 트레이드오프 이해가 핵심입니다.

### 1. 모집단 가설의 표본 기반 검증, 통계적 가설검정 개요

정 의  • 모집단의 분포·모수에 대한 가설을 세우고, 표본에 기초하여
         가설의 채택·기각을 결정하는 통계적 기법
       - 데이터 기반 의사결정의 핵심 방법론으로 AI/ML·A/B 테스트에 활용

```
통계적     →  유의수준  →  검정통계량  →  기각역  →  검정통계량  →  결과
가설설정      결정          설정          설정       계산          해석
 (H0/H1)     (α)         (Z, t, X², F)             (표본관측)   (채택/기각)
```

- 6단계 절차는 순서대로 유기적으로 연결되어 통계적 의사결정의 기반을 형성.

### 2. 가설검정 절차 상세 (6단계)

| 단계 | 구분 | 주요기술 | 설 명 |
|------|------|----------|-------|
| 1단계 | (설정) 통계적 가설설정 | 귀무가설(H0), 대립가설(H1) | 우측·좌측·양측검정 설정 |
| 2단계 | (설정) 유의수준 결정 | α = 0.05 (5%) | 0.1, 0.05, 0.01 중 선택 |
| 3단계 | (설정) 검정통계량 설정 | Z, t, X², F 검정 | 데이터 특성에 따라 선택 |
| 4단계 | (설정) 기각역 설정 | 임계값 결정 | 유의수준을 만족하는 기각역 설정 |
| 5단계 | (계산) 검정통계량 계산 | 표본통계량 산출 | 모집단으로부터 확률표본 관측 후 계산 |
| 6단계 | (의사결정) 결과 해석 | H0 채택/기각 결정 | 검정통계량이 기각역에 속하면 H0 기각 |

1) 1단계: 통계적 가설 설정

| 가설 유형 | 설 명 |
|-----------|-------|
| 귀무가설(H0, Null Hypothesis) | 차이가 없다는 보수적 가정, 기각이 목표 |
| 대립가설(H1, Alternative Hypothesis) | 새로운 주장·연구자가 입증하고픈 가설 |
| 양측검정 | H1: μ ≠ μ₀, 차이의 방향 불특정 |
| 우측검정 | H1: μ > μ₀, 오른쪽 방향 |
| 좌측검정 | H1: μ < μ₀, 왼쪽 방향 (단축) |

2) 3단계: 검정통계량 유형

| 검정통계량 | 사용 조건 |
|------------|-----------|
| Z-검정 | 모분산 기지(旣知), n ≥ 30 |
| t-검정 | 모분산 미지(未知), 소표본 |
| X²-검정 | 범주형 데이터, 적합도·독립성 검정 |
| F-검정 | 두 집단 분산 비교, 분산분석(ANOVA) |

- 검정통계량은 데이터의 분포 특성과 표본 크기에 따라 선택.

### 3. 가설검정의 2가지 오류와 활용

1) 오류 분류 매트릭스

| 의사결정 | H0가 참 | H0가 거짓 |
|----------|---------|----------|
| H0 수락 | 옳은결정 (1-α) | β-오류 (2종) |
| H0 기각 | α-오류 (1종) | 옳은결정 (검정력, 1-β) |

2) 1종·2종 오류 상세

| 구분 | 설 명 | 관련 개념 |
|------|-------|----------|
| 1종 오류 (α오류, Type I Error) | H0이 참인데 기각하는 오류 / 거짓양성(False Positive) | 생산자 위험(Producer's risk) |
| 2종 오류 (β오류, Type II Error) | H0이 거짓인데 채택하는 오류 / 거짓음성(False Negative) | 소비자 위험(Consumer's risk) |
| 신뢰수준 (1-α) | 1종 오류를 범하지 않을 확률 | 주로 0.95 (95%) |
| 검정력 (1-β) | 대립가설이 참일 때 H0를 기각할 확률 | 높을수록 좋은 검정 |

- α를 줄이면 β가 커지고, β를 줄이면 α가 커지는 상충관계(Trade-off) 존재.

- 최근 AI/ML 모델 성능 검증에 가설검정 활용 증대. "끝"

### 실무 제언

**A/B 테스트 설계에서의 유의수준과 검정력**

- **챌린지**: 웹서비스 A/B 테스트 시 유의수준(α)을 0.05로 설정해도 검정력(1-β)이 낮으면 실제로 효과가 있는 변경사항을 놓치게 됩니다. 특히 표본 크기가 충분하지 않으면 2종 오류 위험이 증가합니다.
- **제언**: 실무에서는 유의수준(α = 0.05)과 검정력(1-β ≥ 0.8)을 동시에 만족하는 최소 표본 크기(Sample Size)를 사전에 계산하는 Power Analysis를 수행해야 합니다. 데이터가 충분히 수집되기 전에 중간 결과를 보고 실험을 조기 종료하면 1종 오류 확률이 급격히 상승합니다.
