## 해설

### 출제 배경 및 의도

p-value는 통계적 유의성을 판단하는 핵심 지표로, 학술 연구부터 AI 모델 성능 검증까지 광범위하게 활용됩니다. 채점관은 p-value의 정확한 개념(귀무가설이 참일 때 현재 관측치가 나올 확률)과 유의수준과의 비교를 통한 의사결정 과정을 체계적으로 서술할 수 있는 능력을 평가합니다.

### 1. 모집단에 대한 가설의 검증, 통계적 가설검정

1) 개념 및 가설

정 의  • 표본에서 얻은 사실을 근거로 하여 모집단에 대한 가설이
         맞는지 통계적으로 검정하는 분석 방법
       - 데이터 기반 의사결정의 핵심 도구로 AI·품질관리·A/B 테스트에 활용

```
[모집단]
   ↓ 표본추출(Sampling)
[표본 관측] → [검정통계량 계산] → [p-value 산출] → [유의수준 비교] → [의사결정]
                (Z, t, X², F)                          (p < α?)       (채택/기각)
```

2) 가설 체계

| 항목 | 상세설명 |
|------|----------|
| 귀무가설 (H0, Null Hypothesis) | 직접 검정대상이 되는 가설 (H0의 기각이 목표) / True 가능성이 적어 버릴 것이 예상 / 귀무가설은 옳다는 가정하에 시작 |
| 대립가설 (H1, Alternative Hypothesis) | 귀무가설에 대립되는 가설 (채택이 목표) / H0이 기각될 때 받아들여지는 가설 / 새로운 주장 또는 실제로 입증하고픈 가설 |

3) 가설검정 절차 및 용어 (6단계)

| 단계 | 구분 | 설명 |
|------|------|------|
| 1단계 | (통계) 가설 H1과 H0의 설정 | 귀무가설/연구가설 설정 |
| 2단계 | 통계분석방법 및 검정통계량 선택 | 표본통계량의 표본분포, 검정통계량 계산 방법 결정 |
| 3단계 | 통계적 유의수준(α) 결정 | 주로 0.05 설정 (0.1, 0.05, 0.01 중 선택) |
| 4단계 | 통계분석 (검정통계량 계산) | 모집단으로부터 확률표본 관측 후 계산 |
| 5단계 | 계산된 검정통계량의 p값을 유의수준(α)와 비교 | p < α이면 H0 기각, p ≥ α이면 H0 채택 |
| 6단계 | H0의 기각 혹은 수용 결정 | 귀무가설 채택/기각 최종 의사결정 |

- 6단계 절차에서 p-value는 5단계에서 의사결정의 핵심 기준으로 활용.

### 2. 통계적 가설 검정에서 활용되는 p-value

1) p-value 개념

| 항목 | 설 명 |
|------|-------|
| 정 의 | 귀무가설(H0)이 참이라는 전제 하에, 현재 표본에서 관측된 검정통계량 이상의 극단적인 값이 나올 확률 |
| 의미 해석 | p-value가 작을수록 귀무가설 하에서 현재 결과가 발생하기 어려움 → H0 기각 근거 강화 |
| 판단 기준 | p < α: H0 기각 (통계적으로 유의함) / p ≥ α: H0 채택 (통계적으로 유의하지 않음) |
| 주의사항 | p-value는 "귀무가설이 참일 확률"이 아닌 "H0 하에서 현재 관측치가 나올 확률" |

2) p-value와 1종·2종 오류 관계

| 구분 | 설명 |
|------|------|
| 1종오류 (α) | H0가 참임에도 기각 / α = P(type 1 error) = P(H0를 기각 \| H0는 참) |
| 2종오류 (β) | H0가 거짓임에도 채택 / β = P(type 2 error) = P(H0를 채택 \| H0는 거짓) |
| p-value < α | H0 기각, H1 채택 (통계적으로 유의함) |
| p-value ≥ α | H0 채택 (통계적으로 유의하지 않음) |

3) 검정 방향에 따른 p-value 해석

| 검정 유형 | 기각 조건 | 설 명 |
|----------|----------|-------|
| 양측검정 (Two-tailed) | p < α/2 (양쪽 합산) | H1: μ ≠ μ₀, 차이의 방향 불특정 |
| 우측검정 (Right-tailed) | p < α (우측 면적) | H1: μ > μ₀, 증가 방향 |
| 좌측검정 (Left-tailed) | p < α (좌측 면적) | H1: μ < μ₀, 감소 방향 |

### 3. 실제 사례: 파트타임 월급 가설검정

1) 문제 상황

| 항목 | 내용 |
|------|------|
| 사례 | A사 파트타임 평균월급 150만원, 표준편차 12만원 / 임의 파트타임 100명 조사 → 평균 153만원 |
| 귀무가설 (H0) | μ = 150만원 (파트타임 평균 월급은 150만원이다) |
| 대립가설 (H1) | μ > 150만원 (파트타임 평균 월급은 150만원보다 많다) |
| 검정 유형 | 우측검정 (단측검정) |
| 유의수준 | α = 0.05 |

2) 검정 과정

```
       (x̄ - μ)       (153 - 150)      3
Z₀ = ─────────── = ─────────────── = ─── = 2.5
        σ/√n          12/√100         1.2

임계값(Critical Value): Z_α = Z_0.05 = 1.645
```

3) 판정 결과

| 항목 | 내용 |
|------|------|
| 임계값 (C.V.) | Z_α = 1.645 → 월급 기준 151.98만원 |
| 검정통계량 | Z₀ = 2.5 |
| 비교 | Z₀(2.5) > 임계값(1.645) → 기각역에 속함 |
| 결론 | H0 기각, H1 채택 → "파트타임 평균 월급이 150만원보다 많다" |

- 통계적 독립성(어떤 관측값이 다른 관측값에 영향 미치지 않는 성질) 고려한 검정이 중요. "끝"

### 실무 제언

**p-value 오용과 통계적 유의성의 함정**

- **챌린지**: 실무에서 p < 0.05만으로 결과를 "유의하다"고 결론 내리는 오류가 빈번합니다. 표본 크기(n)가 매우 크면 실질적으로 의미 없는 미세한 차이도 통계적으로 유의하게 나타날 수 있습니다. 반대로 표본이 작으면 실제 효과가 있어도 유의하지 않게 나옵니다.
- **제언**: p-value와 함께 효과 크기(Effect Size: Cohen's d, η², r 등)를 반드시 보고해야 합니다. 또한 신뢰구간(Confidence Interval)을 제시하여 추정의 불확실성을 함께 전달하는 것이 올바른 통계적 커뮤니케이션입니다.
