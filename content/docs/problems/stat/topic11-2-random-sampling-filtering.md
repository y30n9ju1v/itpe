## 해설

### 출제 배경 및 의도

5G·IoT 산업기술 발전으로 빅데이터 스트림이 급증하면서 전체 데이터를 처리하는 것이 불가능한 환경이 됨에 따라, 데이터 대표성 확보를 위한 랜덤 샘플링과 유의미한 데이터 추출을 위한 필터링 기법의 이해도를 확인하기 위해 출제

### 1. 빅데이터 분석·처리에서 샘플링과 필터링의 필요성

정 의 • 샘플링: 매번 모집단의 각 구성 요소가 샘플에 포함될 확률을 동일하게 하는 방법
      - 필터링: 데이터 활용 목적에 맞는 유의미한 데이터를 추출하여 품질을 개선하는 과정

```
[빅데이터 스트림]          [데이터 정제]           [분석·처리]
 산업기술발전       →    샘플링 + 필터링    →   빅데이터
 5G / IoT               알고리즘 적용            분석/처리
```

### 2. 가. 랜덤 샘플링 - 샘플링 오차 감소를 통한 모집단 대표성 확보

정 의 • 모집단으로부터 샘플을 추출하는 방법으로, 샘플링 오차를 제거하기 위해 매번 모집단의 각 구성 요소가 샘플에 포함될 확률을 동일하게 하는 샘플링 방법

랜덤 샘플링 유형:

| 유형 | 설명 |
|------|------|
| 단순 랜덤 샘플링(Simple Random Sampling) | 모집단 모든 샘플링 단위가 동일 확률로 시료에 뽑힐 가능성. N개 크기 중 n개 시료 랜덤 취함 |
| 계통 샘플링(Systematic Sampling) | 추출들 처음 k개 단위 중 하나를 랜덤 출발점으로 정한 다음, 이후 그 점으로부터 매 k번째 떨어진 간격에 위치한 단위들을 표본으로 추출 |
| 2단계 샘플링(Two-Stage Sampling) | 모집단을 몇 개의 부분(1차 단위)으로 나누어 1단계로 그 중의 몇 개를 표본으로 추출하여 1차 시료로, 다음에 1차 시료 중에서 몇 개의 단위체를 추출 |
| 층별 샘플링(Stratified Sampling) | 모집단을 서로 겹치지 않는 몇 개의 층으로 나누어 각 층에서 샘플을 랜덤하게 추출(층화추출법). 층별샘플링(크기에 비례), 네이만샘플링(크기의 표준편차에 비례), 데밍샘플링(비용 고려) |
| 집락 샘플링(Cluster Sampling) | 모집단을 몇 개의 층으로 나누어 그 중에서 샘플(n)수에 알맞게 몇 개의 층을 랜덤 샘플링하여 그것을 취한 층 안의 모든 것을 측정 조사 |

```
[단순 랜덤]  모집단 → 동일확률 무작위 추출
[계통]       모집단 → k 간격마다 추출
[2단계]      모집단 → 1차 군 추출 → 2차 단위 추출
[층별]       모집단 → 층 분류 → 각 층에서 랜덤 추출
[집락]       모집단 → 층 분류 → 선택 층 전수 조사
```

### 3. 나. 필터링 - 유의미한 데이터 추출

가. 학습기반 필터링

| 기법 | 설명 |
|------|------|
| 베이지안 필터링 | 베이즈 정리 기반 조건부 확률로 스팸/정상 분류 |
| 내용기반 필터링 | 콘텐츠 속성 분석하여 유사 콘텐츠 추천 |
| 칼만 필터링 | 잡음이 있는 측정값에서 상태 추정(시계열/센서 데이터) |

나. 비학습기반 필터링

| 기법 | 설명 |
|------|------|
| 해시 필터링 | 해시 함수로 빠른 중복 제거 |
| 질의(Query) 필터링 | 조건식으로 특정 데이터 추출 |
| 블룸 필터링 | 확률적 자료구조로 원소 포함 여부 판단(공간 효율) |

```
[학습기반]                 [비학습기반]
베이지안 필터링    →      해시 필터링
내용기반 필터링    →      질의(Query) 필터링
칼만 필터링        →      블룸 필터링
(모델 학습 필요)          (규칙·구조 기반)
```

### 4. 결론 - 데이터 특성에 맞는 샘플링·필터링 기법 선택으로 빅데이터 품질 확보

전체 정보를 확인하는 것이 불가능 또는 비효율적일 때 데이터 특성을 잘 설명할 수 있는 샘플링 기법을 선택하고, 노이즈 제거를 위한 필터링을 적용하여 분석 품질을 높이는 것이 핵심

끝

### 실무 제언

스트리밍 데이터 환경(Kafka, Flink 등)에서는 저장소 부족 문제로 전체 데이터 보관이 어려우므로, 블룸 필터·해시 필터링을 통한 실시간 중복 제거와 계통 샘플링 기반 대표 데이터 추출이 빅데이터 파이프라인 설계의 핵심 요소로 활용
