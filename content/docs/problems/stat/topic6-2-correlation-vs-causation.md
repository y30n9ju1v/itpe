빅데이터 분석에서 **상관관계(Correlation)**와 **인과관계(Causation)**에 대하여 비교하여 설명하시오.

## 해설

### 출제 배경 및 의도

빅데이터 분석의 중요성이 부각되면서 통계분석 기법에 대한 지식 확인

### 1. 변수간 관계성 파악 방법, 상관관계와 인과관계의 개념 비교

```
[상관관계(Correlation)]          [인과관계(Causation)]
X 변수 ←──────→ Y 변수           X 변수 ──────→ Y 변수
(선형관계성 파악 / 대칭적)        (원인)          (결과 / 비대칭적)
```

| 비교 항목 | 상관관계(Correlation) | 인과관계(Causation) |
|-----------|----------------------|---------------------|
| 변수간 관계 | X가 증감할 때 Y가 얼마나 증감하는지 대칭적 관계 | X가 일어날 때 Y가 어떻게 변하는지 Y에 영향을 주는 X를 설명하는 비대칭적 관계 |
| 목적 | 서로 다른 변수간 관계 패턴 또는 경향 파악 | 변수간 연관성을 비교하여 원인과 결과 파악 |
| 특징 | 변수간 관련성과 함께 관련성 정도 파악에 효과적 | 하나 또는 동시에 두 개 변수의 효과 연구에 효과적 |
| 결과 관련성 | 원인과 결과의 관련성 無 | 원인과 결과는 관련성 필수 |
| 분석 기법 | 시계열분석, 생존분석 | 회귀분석, 이준치 분석, A/B테스트 |
| 도출 모형 | 상관계수(피어슨, 스피어만, 켄달) | 기준선 모형 |

### 2. 빅데이터 분석에서의 상관관계와 인과관계 활용

1) 혼동 변수(교란 변수) 주의

```
[스트레스]
    ↓        ↘
[커피 섭취] ──→ [심장병 발생]  ← 상관관계처럼 보이지만
                               실제 원인은 스트레스(교란 변수)
```

2) 유형별 활용 사례

| 데이터 유형 | 상관관계 분석 사례 / 기법 | 인과관계 분석 사례 / 기법 |
|-------------|--------------------------|--------------------------|
| 수치형 | 광고비용과 매출의 상관관계 / 피어슨 상관분석 | 코로나 확진자 수와 마스크 판매량 / 회귀분석 |
| 범주형 | 몸무게 순위와 키 순위 상관관계 / 스피어만·켄달 | 담배값과 흡연율 / 이준치 분석 |

- 빅데이터 분석 시 드러난 현상을 통해 상관관계를 우선 도출한 후, 데이터들간 인과관계를 파악해야 의도된 결과 유추 가능  "끝"

### 실무 제언

**상관관계에서 인과관계로의 전환 전략**

- **챌린지**: 빅데이터 환경에서는 수많은 변수 간 상관관계가 우연히 발견될 수 있으며(허위 상관), 이를 인과관계로 오해하면 잘못된 의사결정으로 이어질 수 있습니다.
- **제언**: 상관관계를 먼저 탐색적으로 파악한 후, 인과관계 검증을 위해 A/B테스트(무작위 배정)나 도구 변수(IV) 등의 인과추론 기법을 적용해야 합니다. 특히 교란 변수(Confounding Variable) 통제 여부를 반드시 확인하여 분석의 신뢰성을 확보하는 것이 권장됩니다.
