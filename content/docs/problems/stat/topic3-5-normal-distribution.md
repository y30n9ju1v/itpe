## 해설

### 1. 대표적인 연속 확률분포, 정규분포 개요

```
확률분포
├── 이산확률분포
│   ├── 이항분포
│   ├── 기하분포
│   └── 포아송분포
└── 연속확률분포
    ├── [정규분포]  ← 중심
    ├── 표준정규분포
    └── t분포/f분포
```

정 의  • 평균 μ와 분산 σ² 두 매개변수에 따라 구체적인 분포의 위치와
         모양을 결정하는 좌우대칭 종 모양 형태의 확률 분포

수식: f(x) = 1/(√2πσ) × e^(-(x-μ)²/2σ²),  -∞ < x < ∞

### 2. 정규분포의 확률함수 및 특징

1) 정규분포의 확률함수

```
                   99.7%
              ←────────────────→
                 95.4%
              ←──────────────→
                 68.3%
              ←────────────→
   │                  │
   μ-3σ μ-2σ μ-σ  μ  μ+σ μ+2σ μ+3σ
```

2) 정규분포의 특징

단순 특징:
① 평균을 기준으로 좌우대칭, 평균지점에서 값이 가장 높은 최고점이며 최빈값, 중앙값, 평균이 모두 같음
② x축을 점근선으로 사용함
③ 모든 곡선 아래의 전체 면적은 1
④ 곡선은 평균으로부터 멀어질수록 x축에 가까워지나, 결코 x축에 닿지 않음

복합 특징:
⑤ σ이 일정할 때, μ의 값이 변하면 대칭축의 위치가 변함 (μ₁ < μ₂ → 분포 이동)
⑥ μ의 값이 일정할 때, σ의 값이 커질수록 중앙 부분이 낮아지면서 양쪽으로 값이 작아지고, 값이 작을수록 곡선의 중앙부분이 높아지면서 좁아짐

### 3. 중심극한정리에서의 정규분포 사용

```
모집단 → 확률표본추출 → 표본 (X₁, X₂, ..., Xₙ)
         ↓
    X̄ = (X₁+X₂+...+Xₙ)/n
         ↓
모집단 분포 (σ)     →     표본평균 분포 (σ/√n)
```

- 중심극한정리에 의해 표본 크기가 커질수록(보통 30 이상), 표본평균들이 이루는 분포가 모집단의 평균 μ 그리고 표준편차 σ/√n인 정규분포에 가까워짐
- 최근 AI/ML 알고리즘의 데이터 분포 가정으로 정규분포 활용 증대.  "끝"
