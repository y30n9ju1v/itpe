## 해설

### 출제 배경 및 의도

데이터 분석은 빈출 도메인으로써 매 회차 분석 기법 문제에 따른 준비 필요

### 1. 두 변수간 선형 관계 분석, 상관 분석의 개념

정 의  • 두 변수간의 상관계수(correlation coefficient)를 통해 데이터 안의 선형 관계에 대한 통계적 분석 기법

```
     상관계수(r)
     -1          0          +1
      |----------|----------|
   강한음의    상관없음    강한양의
   상관관계              상관관계

피어슨 상관계수:
r = Σ(Xi-X̄)(Yi-Ȳ) / √[Σ(Xi-X̄)²·Σ(Yi-Ȳ)²]
```

전제 조건:
• 등분산성: 두 변수들을 대표하는 직선을 그어 독립변수의 어떤 지점에서도 종속변수의 흩어진 정도가 동일함
• 선형성: 연속형 두 변수 간에는 선형적인 관계가 있어야 함

### 2. 상관 분석의 개념도 및 상관계수의 유형

1) 상관 분석의 개념도 - 상관계수 해석 기준

| 상관계수(r) | 선형관계 해석 |
|-------------|---------------|
| r ≤ .1 | 거의 무시될 수 있는 선형관계 |
| .1 < r ≤ .3 | 약한 선형관계 |
| .3 < r ≤ .7 | 뚜렷한 선형관계 |
| r > .7 | 강한 선형관계 |

2) 상관계수의 유형

| 구분 | 척도 유형 | 설명 | 검정 방식 |
|------|-----------|------|-----------|
| Pearson | 간격/비율 척도 | 기본적으로 사용되는 피어슨 상관계수, 연속형 변수의 상관관계 측정 | 모수 검정 |
| Spearman | 서열-서열 척도 | 스피어만 상관계수(ρ:rho), 변수값 대신 순위로 바꿔서 이용 | 비모수 검정 |
| Kendall | 서열-서열 척도 | 켄달 상관계수(τ:tau), 변수값 대신 순위로 이용, 샘플사이즈가 적거나 동률이 많을 때 유용 | 비모수 검정 |

```
스피어만 공식: ρ = 1 - 6Σdi² / n(n²-1)
  - di = xi의 순위 - yi의 순위

켄달 타우: τ = (C - D) / (C + D)
  - C: 일치 쌍(Concordant pairs)의 수
  - D: 불일치 쌍(Discordant pairs)의 수
```

### 3. 상관 분석 활용 시 주의사항

| 구분 | 주의사항 | 설명 |
|------|----------|------|
| 해석 | 상관관계와 인과관계 혼동 방지 | 상관관계가 있다고 해서 반드시 인과관계가 성립하지는 않음 |
| 데이터 | 이상치(Outlier)의 영향 고려 | 이상치가 상관계수를 크게 왜곡할 수 있으므로 전처리 필요 |
| 한계 | 비선형 관계 처리 한계 | 피어슨 상관계수는 선형 관계만 측정하므로 비선형 관계는 별도 기법 필요 |

- 상관 분석은 변수간 관계 파악의 출발점이며, 인과관계 분석(회귀분석 등)과 병행하여 활용해야 신뢰성 있는 결론 도출 가능  "끝"

### 실무 제언

**척도 유형에 따른 상관계수 선택 전략**

- **챌린지**: 데이터 분석 시 척도 유형을 고려하지 않고 피어슨 상관계수를 무조건 적용하면 잘못된 결론을 도출할 위험이 있습니다.
- **제언**: 연속형(간격/비율) 변수에는 Pearson, 순서형(서열) 변수에는 Spearman 또는 Kendall을 선택하되, 샘플 수가 적거나 동률이 많은 경우 Kendall이 더 안정적입니다. 분석 전 정규성 검정(Shapiro-Wilk 등)을 통해 모수/비모수 방법론을 먼저 결정하는 것이 권장됩니다.
