## 문제

한 공장에서 스마트 팩토리(Smart Factory) 적용을 통한 생산시간 단축을 확인하려고 한다. 기존 공정에서의 제품 생산시간이 정규분포를 따르며 평균이 30분, 표준편차가 5분인 것으로 알려져 있다. 스마트 팩토리가 적용된 공정의 제품에 대한 생산시간을 49개 랜덤 추출하여 관측한 결과 평균이 28.5분이었다. 스마트 팩토리가 적용된 공정의 생산시간이 표준편차가 5분인 정규분포를 따른다고 할 때, 스마트 팩토리를 통해 생산시간이 단축되었는지 가설을 설정하고 검정하시오. (단, 유의수준 5%, Z_0.05=1.645)

## 해설

### 출제 배경 및 의도

스마트 팩토리 도입 효과를 통계적으로 검증하는 실전 문제입니다. 단순히 평균이 낮아졌다는 사실을 넘어, 그 차이가 통계적으로 유의한지(우연이 아닌지)를 Z-검정으로 판단하는 능력을 요구합니다. 채점관은 가설 설정의 논리(좌측 단측검정), Z-검정통계량 계산 과정, 임계치와의 비교 및 최종 해석까지 완결된 서술을 평가합니다.

### 1. 표본 기반 모집단 가설 검증, 통계적 가설검정 개요

정 의  • 표본에서 얻은 사실을 근거로 모집단에 대한 가설이 맞는지
         참/거짓을 통계적으로 검정하는 분석방법
       - 해당 사례는 생산시간 단축(미만) 검정이므로 좌측 단측검정 적용

1) 가설검정 구성요소

| 구성요소 | 설 명 |
|---------|-------|
| 귀무가설 (H0) | 영가설(Null), 차이가 없다는 가정하에 시작, 기각이 목표 |
| 대립가설 (H1) | 연구/실험가설, 차이가 있다는 가정하에 시작, 채택이 목표 |
| 유의수준 (α) | H0가 참일 때 H1을 채택하는 오류를 범할 최대 허용치 |
| p-value | H0가 참일 때 표본(관측치)가 발생할 확률 |
| 검정통계량 | 표본통계량의 모수 근접 수치 (본 사례: Z-score) |
| 임계치 | 기각/채택을 구분하는 검정통계량의 값 |
| 기각역 (C1) | 귀무가설을 기각하는 검정통계량의 범위 |
| 채택역 (C0) | 귀무가설을 채택하는 검정통계량의 범위 |

2) Z-검정 적용 조건 확인

| 조건 | 기준 | 사례 확인 |
|------|------|----------|
| 모분산 기지(旣知) | 모표준편차(σ) 알려짐 | σ = 5분 ✓ |
| 표본 크기 | n ≥ 30 (중심극한정리) | n = 49 ✓ |
| 정규분포 가정 | 모집단 정규분포 | 문제에서 명시 ✓ |

- 3가지 조건 모두 만족 → Z-검정 적용 타당.

### 2. 스마트 팩토리 사례에 대한 가설 설정

1) 문제 상황 파악

```
[기존 모집단]              [스마트 팩토리 표본 (n=49)]
 평균 μ₀ = 30분             표본평균 x̄ = 28.5분
 표준편차 σ = 5분            표준편차 σ = 5분 (동일 가정)

      ↓ 표본추출 (n=49)
      유의수준 α = 5%, Z_0.05 = 1.645
      검정 방향: 단축 여부 → 좌측 단측검정
```

2) 가설 설정

| 가설 | 수식 | 설 명 |
|------|------|-------|
| 귀무가설 (H0) | μ ≥ μ₀ (μ ≥ 30) | 스마트 팩토리 적용 후 생산시간이 단축되지 않았다 |
| 대립가설 (H1) | μ < μ₀ (μ < 30) | 스마트 팩토리 적용 후 생산시간이 단축되었다 |

3) 검정 방향 결정 근거

| 구분 | 설 명 |
|------|-------|
| 양측검정 해당 X | H1: μ ≠ 30 (증가 또는 감소, 방향 불특정) |
| 우측검정 해당 X | H1: μ > 30 (생산시간 증가 가설) |
| 좌측검정 적용 ✓ | H1: μ < 30 (생산시간 단축 가설) → 본 사례에 해당 |

- "단축"은 감소 방향이므로 좌측 단측검정(Left-tailed Test) 적용.

### 3. 스마트 팩토리 사례에 대한 통계적 가설 검정

1) Z-검정 통계량 계산

통계분석 방법: Z-검정 (모집단 평균과 표준편차가 알려진 경우, 표본평균이 모집단 평균과 다른지 검정)

```
       (x̄ - μ₀)        (28.5 - 30)     -1.5
Z₀ = ─────────── = ───────────────── = ───── = -2.1
        σ/√n             5/√49          5/7

  x̄  = 28.5분 (표본평균)
  μ₀ = 30분   (귀무가설 모평균)
  σ  = 5분    (모표준편차)
  n  = 49     (표본 크기)
  σ/√n = 5/7 ≈ 0.714 (표준오차)
```

2) 검정통계량과 임계치 비교

```
  대립가설(H1) 기각역     귀무가설(H0) 채택역
  ←──────────┤──────────────────────────→
             │         σ=1 정규분포
   기각역     │    채택역 (95%)
  (α=5%)     │
  -2.1     임계치(-1.645)    0(μ)
   Z₀         Z_α

  Z₀(-2.1) < 임계치(-1.645) → 기각역에 속함 → H0 기각
```

3) 판정 결과

| 항목 | 내용 |
|------|------|
| 귀무가설 채택 조건 | 검정통계량(Z₀) > 임계치(-1.645) |
| 귀무가설 기각 조건 | 검정통계량(Z₀) ≤ 임계치(-1.645) |
| 검정통계량 | Z₀ = -2.1 |
| 임계치 | -Z_0.05 = -1.645 |
| 비교 결과 | Z₀(-2.1) < 임계치(-1.645) → 기각역에 해당 |
| 최종 결론 | 귀무가설(H0) 기각, 대립가설(H1) 채택 |
| 해석 | 유의수준 5% 하에서 "스마트 팩토리 적용 후 생산시간이 단축되었다"는 주장을 통계적으로 지지 |

4) 검정 결과 요약

```
  검정통계량 Z₀ = -2.1
  임계치 -Z_α = -1.645
  ∵ Z₀(-2.1) ≤ -Z_α(-1.645) → H0 기각

  ∴ 유의수준 5%에서 스마트 팩토리 적용 후 생산시간이
    기존 30분에서 통계적으로 유의하게 단축되었다고 판단.
```

- Z-검정의 경우 중심극한정리에 의해 n≥30 이상의 표본 크기에서 정규분포 사용. "끝"

### 실무 제언

**표본 크기 설계의 중요성 (Power Analysis)**

- **챌린지**: 표본 크기가 검정 결과에 결정적인 영향을 미칩니다. 만약 n=25로 표본을 설정했다면 Z₀ = (28.5-30)/(5/√25) = -1.5가 되어 임계치(-1.645)보다 크므로 H0를 기각하지 못합니다. 즉, 실제로 생산시간이 단축되었음에도 통계적으로 입증하지 못하는 2종 오류(β오류)가 발생합니다.
- **제언**: 스마트 팩토리 도입 효과 검증을 위한 실험 설계 단계에서 Power Analysis를 수행하여 검정력(1-β ≥ 0.8)을 만족하는 최소 표본 크기를 사전에 산출해야 합니다. 또한 표준편차(σ)가 실제로 변할 수 있으므로 t-검정 병행 적용도 고려해야 합니다.
