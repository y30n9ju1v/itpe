## 해설

### 1. 통계학 4가지 척도 구분

가. 통계학 척도의 정의

정 의  • 어떠한 대상의 특성을 단위를 사용하여 정량화한 대상특성의 단위

나. 통계학 4가지 척도 구분표

| 구분 | 척도 | 설명 | 예시 |
|------|------|------|------|
| 범주형 변수 | 명목척도 | 사물을 지칭하거나 분류하기 위해 부여한 수치 기호를 사용한 척도 | 주민등록번호, (남, 여) |
| 범주형 변수 | 서열척도 | 명목척도의 특성을 가지면서, 대상의 순위정보를 나타내는 척도 | 소득순위 |
| 연속형 변수 | 등간척도 | 명목척도와 서열척도의 특성을 가지면서, 속성차이에 대한 간격(등간)정보를 포함하는 척도 | 선호점수 |
| 연속형 변수 | 비율척도 | 명목척도, 서열척도, 등간척도의 특성을 가지면서, 비율에 관한 정보까지 포함하는 척도 | 매출액 |

- 명목·서열척도는 범주형변수를 사용하는 로지스틱회귀분석에서 사용
- 등간·비율척도는 연속형변수를 사용하는 다중회귀분석에서 사용

### 2. 척도 관점 다중회귀분석과 로지스틱회귀분석 비교

1) 개념 비교

| 구분 | 다중회귀분석 | 로지스틱회귀분석 |
|------|------------|----------------|
| 개념 | 두 개 이상의 독립변수가 하나의 종속변수에 미치는 영향을 분석하고 예측 | 분석대상이 어느 집단에 분류될 수 있는가를 분석하고 예측 |

- 다중회귀분석은 연속형 결과 변수 값을 예측
- 로지스틱회귀분석은 비연속형 결과범주를 예측

2) 척도유형 측면 비교

| 구분 | 다중회귀분석 | 로지스틱회귀분석 |
|------|------------|----------------|
| 목적 | 예측문제 해결 | 분류문제 해결 |
| 변수 유형 | 독립변수: 연속형 / 종속변수: 연속형 | 독립변수: 연속형 / 종속변수: 범주형 |
| 독립변수 척도 | 명목, 서열, 등간, 비율척도 | 명목, 서열, 등간, 비율척도 |
| 종속변수 척도 | 명목, 서열, 등간, 비율척도 | 명목, 서열척도 |

- 로지스틱 회귀분석에서는 정교한 분석을 위해 Sigmoid, ReLU 등의 활성 함수를 활용하여 결과를 도출

### 3. 기법 관점 다중회귀분석과 로지스틱회귀분석 비교

| 구분 | 다중회귀분석 | 로지스틱회귀분석 |
|------|------------|----------------|
| 분석 기법 | 최소 제곱법 / 실제 값과 예측된 값 간 잔차(오차) 제곱이 최소가 되는 계수를 탐색 | 최대 우도법 / 가중 최소 제곱법 |
| 모델검증 기법 | F-test, t-test | 혼동 행렬, ROC Curve |
| 적용 예시 | 흡연율과 사망률 역학 조사 / 노동 수요/공급 예측 | 주식 매도/보유 판단 / 대출 승인/거절 판단 |

- 최근 AI/ML에서 다중회귀분석과 로지스틱회귀분석이 핵심 지도학습 알고리즘으로 활용 증대.  "끝"
