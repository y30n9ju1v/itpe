## 해설

### 출제 배경 및 의도

소셜미디어·IoT·웹 로그 등 비정형 데이터가 전체 데이터의 80% 이상을 차지함에 따라, 이를 체계적으로 분석하는 기법이 주목받고 있다. 이 문제는 비정형 데이터 마이닝의 개념·필요성을 시작으로, 텍스트 마이닝과 사회연결망 분석(SNA)의 세부 내용을 단계적으로 묻는다. 기술사 답안에서는 각 개념의 정의·구성·절차를 논리적으로 연결하고 실무 연계성을 제시하는 것이 핵심이다.

### 1. 비정형 데이터 마이닝 개요

가. 정의

정 의  • 자연어, 이미지, 음성, 영상 등 비정형 데이터에서 유용한 패턴 및 지식을 추출하는 마이닝 기법
       - 정형화된 스키마 없이 존재하는 다양한 형태의 데이터를 대상으로 인사이트를 도출

나. 비정형 데이터 마이닝 필요성

| 필요성 | 설명 |
|--------|------|
| 비정형 정보 기하급수적 증가 | 웹, 음성, 문서, 동영상 등 각종 온라인 소스의 데이터가 빅데이터 규모로 급증 |
| 비정형의 가시화 요구 | 자연어 처리 등을 통해 고객 감성·행동 분석 요구가 지속적으로 증가 |
| 수동 분석 어려움 | 방대한 비정형 데이터에서 유용한 정보를 효과적으로 추출하기 위한 자동화된 관리 필요 |

---

### 2. 텍스트 마이닝의 개념, 기능, 기본분석절차

가. 개념

정 의  • 자연어로 구성된 비정형화된 텍스트 데이터에서 패턴 또는 관계를 추출하여 가치와 의미 있는 정보를 찾아내는 기법
       - NLP(자연어처리) 기술에 기반하여 텍스트를 구조화하고 의미 있는 지식을 자동 발견

나. 텍스트 마이닝 처리 구조

```
사용자
  │
  ▼
텍스트 수집/전처리·변환
  │
  ▼
NLP 기반 온톨로지 / 색인 / 지식DB
  │
  ▼
텍스트 자료 처리 과정
  │
  ▼
텍스트 분석 가시화/평가
```

나. 텍스트 마이닝 기능

| 기능 구분 | 기능 상세 설명 | 관련 요소 |
|-----------|---------------|-----------|
| 텍스트 분류 | 텍스트 데이터 특성 정의, 데이터 간 유사도를 계산하여 분류 | 비지도 학습, 군집화 |
| 문서 군집(Document Clustering) | 각 지식 콘텐츠 특성 파악, 유사·상호관련성 높은 콘텐츠 군집 | 유사도 계산, 통계, 규칙 기반 알고리즘 |
| 정보 추출 | 텍스트 문서 내에서 중요한 의미 정보를 자동 추출 | 온톨로지, 시맨틱웹, 검색 성능 개선 |
| 문서 요약(Document Summarization) | 문서의 핵심 의미를 유지하면서 복잡도와 길이를 줄여 요약 | 추출 요약, 생성 요약 |

다. 텍스트 마이닝 기본 분석절차

| No | 분석 절차 | 텍스트 마이닝 분석 내용 설명 |
|----|-----------|------------------------------|
| 1 | 텍스트 문서 선정/수집 | 데이터베이스 콘텐츠 혹은 텍스트 기반의 문서들을 대상으로 선정 |
| 2 | 텍스트 전처리 | 문서 내 단어/구/절을 언어 분석 처리 과정을 통해 가공 가능한 데이터로 표현 |
| 3 | 의미정보 변환 | 전처리된 데이터 중 의미 있는 정보를 선별하여 저장 |
| 4 | 의미정보 추출 | 복잡한 의미정보 표현을 단순화하고, 도메인에 적합한 정보를 의미 데이터로 저장 |
| 5 | 패턴 및 경향 분석 | 의미 데이터를 기반으로 문서를 자동 군집화·분류하여 정보 재생산 |
| 6 | 정보 표현 및 평가 | 시각화 툴로 효과적으로 표현하고, 문제점 수정·보완을 통해 품질 및 성능 향상 |

### 3. 사회연결망 분석(SNA)의 개념, 분석 표현방법, 분석기법, 분석절차

가. 개념

정 의  • 사회연결망 이론을 바탕으로 소셜 네트워크의 형태 및 연결 구조, 연결 강도 등을 계량화·측정하여 특정 구성원의 영향력, 특정 네트워크의 영향력 등 유의미한 지식을 도출하는 기법

나. 분석 표현방법

| 표현 속성 | 표현 지표 | 내용 설명 |
|-----------|-----------|-----------|
| 응집력(Cohesion) | 밀도(Density) | 사회 네트워크에서 가능한 총 관계자 수 중 실제로 맺어진 관계 수의 비율 |
| 응집력(Cohesion) | 결속(Clique) | 모든 노드끼리 완전하게 연결된 하부 네트워크 |
| 구조적 등위성(Equivalence) | 유클리디안 거리 | 두 노드 간의 가장 짧은 물리적 거리 |
| 구조적 등위성(Equivalence) | 상관계수 | 두 행위자들 간 관계의 패턴 유사성 |
| 명성(Prominence) | 연결 정도 | 한 행위자가 다른 행위자와 맺는 관계의 수를 통해 네트워크 중심 위치 정도 측정 |
| 명성(Prominence) | 근접 중심성 | 직접·간접으로 연결된 모든 행위자들 간 거리의 역수 |
| 범위(Range) | 도달가능성(Reachability) | 두 행위자 간 연결이 얼마나 많이 설정되어 있는가 |
| 범위(Range) | 최단경로(Geodesic Distance) | 두 행위자 간 연결 중 가장 적은 수의 관계자를 거치는 경로 |

다. 분석기법

| 분석기법 | 설명 |
|----------|------|
| Density 분석 | 네트워크 내에 얼마나 많은 관계가 존재하는지 분석하는 기법 |
| Neighbor 분석 | 네트워크를 구성하는 각 노드들 간의 표면적인 연결 상태를 파악하는 기본 SNA 방법 |
| Centrality 분석 | 한 노드가 네트워크에서 얼마나 중심에 위치하는지 측정 (Degree/Closeness/Betweenness/Centralization) |
| Clique Cohesion 분석 | 결합력(Cohesion)을 바탕으로 군집 구조를 파악하는 방법 |
| Affinity 분석 | 공통점이 많은 정도를 분석 (예: 친구 관계 유사성, 동일 콘텐츠 선호도) |

라. SNA 분석절차

```
수집 및 전처리
  (비정형 데이터 수집 / 데이터 전처리·변환 / 주제어 선정)
  │
  ▼
연결망 분석
  (사회연결망 구축 / 관계 표현 및 밀도 측정 / 개체 정의 / 영향도 분석)
  │
  ▼
결과 평가 및 활용
  (분석 결과 가시화 / 패턴 및 경향 평가 / 분석 연계 및 활용)
```

### 4. 결론

비정형 데이터 마이닝에서 텍스트 마이닝과 사회연결망 분석은 빅데이터 분석의 핵심 도구로 부상하였다. 텍스트 마이닝은 NLP 기반으로 의미 있는 패턴을 추출하고, SNA는 관계·영향력 구조를 계량화하여 의사결정 지원에 활용된다. 두 기법의 결합은 고객 감성 분석, 바이럴 마케팅, 위험 인물 탐지 등 다양한 실무 영역에서 시너지를 창출한다. **끝**

### 실무 제언

- 텍스트 마이닝 전처리 단계에서 한국어 형태소 분석기(KoNLPy 등) 활용이 성능에 직결됨
- SNA 적용 시 중심성 지표(Degree/Betweenness) 선택이 분석 목적에 따라 달라짐을 유의
- 두 기법을 결합한 하이브리드 분석으로 SNS 영향력자 탐지, 허위정보 확산 경로 추적 등에 활용 가능
