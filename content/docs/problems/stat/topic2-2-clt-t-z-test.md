## 해설

### 출제 배경

생성형 AI 등 인공지능에 대한 관심이 대폭 증가하며 인공지능의 근간을 이루는 기초 통계 지식을 점검하는 문제.

### 1. 추론 통계의 기반, 중심극한정리

정 의: 모집단에서 크기가 n인 표본을 k번 복원 추출 시, n·k가 충분히 크면 각 표본 평균의 분포는 정규분포로 수렴한다는 이론

```
모집단        sampling
(평균μ,  ────→  표본1 → 표본평균1 ┐
 분산σ²)        표본2 → 표본평균2 │  → 정규분포 수렴
                 ⋮      ⋮         │     분산 σ²/n
                표본k → 표본평균k ┘     모평균 μ
```

- 모집단의 분포가 정규분포가 아니어도 표본 '평균'의 분포는 모평균을 중심으로 하는 정규분포로 수렴
- 이를 통해 표본만 갖고도 t-검정, Z-검정, 카이제곱검정 등 정규분포를 가정하는 검정기법들의 사용이 가능

### 2. 소표본 기반 모평균 추정, t-검정

1) t-검정의 정의 및 수행절차:

정 의: 모분산 σ²을 모르고 표본의 크기가 30보다 작을 시에 t분포를 활용해 가설을 검정하는 기법

수식:
```
       (X̄ - μ)
t  = ─────────
       (σ/√n)
```

t-분포 특징:
① 평균이 0인 좌우대칭 종모양
② 정규분포를 표준화한 분포
③ 모분산을 모르고 표본 크기가 작을 때 사용
④ 자유도(표본 수-1)가 커질수록 꼬리가 얇아짐
⑤ 자유도가 커질수록 Z-분포에 가까워짐

2) t-검정 유형별 상세 설명:

| 유형 | 개념 | 설명 |
|------|------|------|
| 1표본 t-검정 | 모집단1개 → 표본집단1개 추출 | 1개 모집단에서 1개 표본집단 추출, 모집단 평균에 대한 가설 검정, H0: "공감반 평균 점수는 60점이다" |
| 대응표본 t-검정 | 모집단1개 → 표본A, 표본B (전/후 비교) | 한 모집단의 두 대응 표본집단 차이 검정, H0: "특강 전후 공감반 평균 차이가 0이다" |
| 독립표본 t-검정 | 모집단A·B → 표본A, 표본B (집단간 비교) | 2개 모집단에서 각각 표본집단 추출, 두 모집단의 평균에 차이가 있는지 검정, H0: "공감반과 비상반의 평균은 같다" |

- t분포는 자유도 n이 커질수록 표준정규분포 N(0,1)에 수렴하므로 표본 크기가 30 이상 시 Z분포 사용

### 3. 대표본 기반 모평균 추정, Z-검정

1) Z-검정의 정의:

정 의: 모분산 σ²을 알거나, 표본의 크기가 30보다 큰 경우에 Z분포를 활용해 가설을 검정하는 기법

```
Z = (X - μ) / σ
```

Z-분포 특징:
① 평균 0, 분산 1인 좌우대칭 종모양
② 정규분포를 표준화한 분포
③ Z값: 특정 관측값이 평균으로부터 떨어진 정도
④ 모분산을 알거나 표본 크기가 클 때 사용
⑤ -3 < Z < 3 범위에 99.7%의 관측값 포함

2) Z-검정 절차 상세설명:

| 절차 | 상세활동 | 설명 |
|------|----------|------|
| ① 가설 수립 | 귀무가설(H0) 수립 / 대립가설(H1) 수립 | 1표본, 대응표본, 독립표본 검정 가설 / 양측 검정, 단측 검정 |
| ② 유의수준 결정 | 신뢰도 설정 / 유의수준 = 1-신뢰도 | 주로 95%로 설정 / 유의수준 = 0.05 |
| ③ 검정통계량 계산 | Z값 도출 / p값 계산 | Z = (X-μ)/σ / Z값에 해당하는 p값 도출 |
| ④ 결론 도출 | p와 유의수준 비교 / 가설 기각/채택 | p < 유의수준일 시 귀무가설 기각 / 귀무가설 기각 시 대립가설 채택 |

### 4. t-검정과 Z-검정 비교

| 구분 | t-검정 | Z-검정 |
|------|--------|--------|
| 검정 조건 | 모집단이 정규분포인 경우 / 모분산을 모르는 경우 | 표본 크기가 충분히 큰 경우 / 모분산을 알고 있는 경우 |
| 표본 크기 | 주로 n < 30 / 소표본 대상 정규분포 표준화 | 주로 n ≥ 30 / 대표본 대상 정규분포 표준화 |
| 특징 | Outlier 등에 비교적 Robust / 자유도(표본-1)에 따라 그래프 상이 | 충분히 큰 n을 가정 / 항상 평균 0, 분산 1의 동일 그래프 |

- 표본크기가 큰 경우는 중심극한정리에 따라 표본평균의 정규성이 보장되므로, 모집단이 정규분포가 아니어도 검정 수행 가능. "끝"
