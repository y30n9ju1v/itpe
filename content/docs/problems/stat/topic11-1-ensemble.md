## 문제

머신 러닝(Machine Learning)에서 활용되는 앙상블(Ensemble) 기법을 설명하시오.

## 해설

### 출제 배경 및 의도

빅데이터·AI 시대에 단일 예측 모델의 한계를 극복하기 위한 앙상블 기법의 중요성이 높아짐에 따라, 배깅·부스팅·랜덤포레스트 등 핵심 알고리즘의 원리와 절차 이해 여부를 확인하기 위해 출제

### 1. 다수 모형 결합으로 신뢰성을 높이는 기법, 앙상블(Ensemble)의 개요

정 의 • 기계 학습에서 하나 또는 다수의 모형을 통해 학습한 예측/분류 결과를 종합하여 최종적인 의사결정에 활용하는 기법
      - 다양한 모형의 예측 결과를 결합함으로써 단일 모형보다 높은 신뢰성 확보
      - 이상치 대응력 향상 및 전체 분산 감소를 통한 과적합(Overfitting) 최소화

```
     [훈련 데이터]
          │
    ┌─────┴─────┐
[모델 A]  [모델 B]  [모델 C]
    └─────┬─────┘
     [결과 조합]
          │
    [앙상블 최종 모델]
```

### 2. 앙상블 기법의 학습 절차

| 단계 | 절차 | 설명 |
|------|------|------|
| 1단계 | 훈련 집합 도출 | 원본 데이터로부터 다수의 훈련용 하위 집합 생성 |
| 2단계 | 집합별 모델 학습 | 각 훈련 집합에 대해 독립적인 학습 모델 구축 |
| 3단계 | 결과 조합 | 각 모델의 예측·분류 결과를 투표(Voting) 또는 평균으로 결합 |
| 4단계 | 앙상블 도출 | 조합된 결과를 기반으로 최종 앙상블 모델 확정 |

### 3. 앙상블 주요 알고리즘

| 구분 | 알고리즘 | 정의 | 절차 |
|------|----------|------|------|
| Sampling | 배깅(Bagging) | 학습 데이터에서 다수의 Bootstrap 자료를 생성하고, 각 자료를 모델링한 후 결합하여 최종 예측 모형을 만드는 알고리즘 | ①N개의 BootStrap 데이터 추출 → ②N개의 단일 분류자 생성 → ③Majority Voting으로 최종 모델 |
| Sampling | 부스팅(Boosting) | 이전의 분석용 데이터에 새로운 분류규칙을 만들어 가중치를 적용하는 과정을 반복해 최종 모형을 만드는 알고리즘 | ①동일 가중치 분석용 데이터로부터 분류자 생성 → ②이전 분류 결과에 따라 가중치 부여 → ③N회 반복 후 최종 분류자 |
| Variable | 랜덤 포레스트(Random Forest) | 배깅과 유사하게 BootStrap 자료를 도출하여 각 자료를 대표하는 변수 샘플을 의사결정 나무의 노드로 분류하는 알고리즘 | ①N개의 BootStrap 데이터 추출 → ②N개의 분류기 훈련 후 대표 변수 샘플 도출 → ③대표 변수 샘플을 의사결정 나무의 Leaf Node로 분류 → ④Leaf Node들의 선형결합으로 최종 모델 |

```
[배깅]                     [부스팅]
Bootstrap 복원 추출    →   가중치 순차 반영
병렬 학습                  직렬(순차) 학습
분산 감소 효과             편향 감소 효과
```

### 4. 앙상블 분류기 선택 시 고려사항

| 고려 측면 | 내용 |
|-----------|------|
| 신뢰성 확보 | 분류기의 신뢰도를 기반으로 좋은 분류기를 찾아 조합 |
| 성능 기준 | 훈련 집합의 오차 판단이 아닌 성능을 기준으로 예측하여 훈련 모형의 예측력 강화 |

끝

### 실무 제언

XGBoost, LightGBM 등 부스팅 계열 알고리즘은 Kaggle 등 실무 데이터 경진대회에서 높은 성능을 보이며, 랜덤포레스트는 피처 중요도 해석이 용이하여 금융·의료 분야 해석 가능한 AI(XAI) 구현에 활용
