## 해설

### 출제 배경

평균 검정 방법론의 주요 기법에 대한 이해. AI/ML 모델의 성능 비교 검정에 활용.

### 1. 가설 검정의 분류 체계

```
              [가설검정]
                  │
          ┌───────┴───────┐
        평균              분산
          │                │
    ┌─────┴─────┐     ┌────┴────┐
  모집단1개  2개  3개이상 1개    2개
    │       │      │     │       │
  t검정   t검정  분산  카이  F검정
               분석    제곱
             (ANOVA)  검정
```

- 검정 대상이 평균이냐 분산이냐에 따라 나뉘고, 모집단의 수에 의해 구체적으로 결정

### 2. 독립 표본 T-Test 및 Paired T-Test

1) 독립표본 T-Test:

정 의: 서로 다른 두 모집단으로부터 추출된 경우에 사용하는 분석방법

결과해석:
• T값이 클수록 두 집단의 평균은 차이가 있을 가능성 커짐
• P-value가 0.05보다 작을 경우 두 집단의 평균이 유의미하게 차이가 있다고 해석
• 신뢰구간이 0을 포함하지 않는 경우 두 집단의 평균은 차이가 있다고 판단
• 추출한 표본은 정규분포를 따르는 모집단으로부터 추출된 것을 가정
• 사례수가 최소 30 이상일 경우에 통계적으로 의미 있음

2) Paired T-Test:

정 의: 하나의 모집단으로부터 반복 추출된 경우에 사용하는 분석방법
- 데이터가 하나의 모집단으로부터 반복 추출, 사례수는 동일해야 함
- 결과 해석은 독립표본 T-Test와 동일한 방법 사용

### 3. ANOVA 및 T-Test와의 비교

1) ANOVA(Analysis Of Variance, 분산분석):

정 의: 서로 다른 세 개 이상의 모집단으로부터 추출된 경우 사용하는 분석방법
- F값이 클수록 귀무가설이 기각될 가능성이 커짐
- 분산분석 결과만으로는 한 그룹만 평균의 차이가 있는지, 모든 그룹이 차이가 있는지는 확인할 수 없음 → 다중 비교(Multiple Comparison)를 통해 확인 가능

2) T-Test와 ANOVA 비교표:

| 구분 | T-Test | ANOVA |
|------|--------|-------|
| 검정 방법 | 서로 독립된 두 집단간의 평균 차이 검정 (독립) 또는 하나의 집단 내에서 동일한 대상을 두 번 반복 측정하여 전/후, 좌/우의 평균 차이 검정 (Paired) | 서로 독립된 집단이 셋 이상인 경우, 집단간의 평균이 서로 차이가 있는지를 확인하기 위한 검정 방법 |
| 차이 비교 | 2집단 평균 비교 | 3집단 이상 평균 비교 |
| 검정 통계량 | T검정통계 | F검정통계 |
| 제시 통계량 | 평균, 표준편차 | 평균, 표준편차, 사후검정 결과 |
| 적용 예시 | T-Test: 인스턴트 커피 A에 대한 남녀간의 선호도 차이를 검정 / Paired-Test: 인스턴트 커피 A와 B의 단맛의 정도 차이를 검정 | 분산분석: 인스턴트 커피 A에 대한 연령별(20대, 30대, 40대, 50대 이상) 선호도 차이 유무를 검정 |
| 귀무가설 | 두 집단간 평균은 차이가 없음 | 셋 이상의 집단간 평균은 차이가 없음 |
| 대립가설 | 두 집단간 평균은 차이가 있음 | 적어도 하나 이상의 집단은 다른 집단과 평균의 차이가 있음 |

- 최근 A/B 테스트 고도화로 T-Test, ANOVA 등 검정 방법론 활용 증대. "끝"
