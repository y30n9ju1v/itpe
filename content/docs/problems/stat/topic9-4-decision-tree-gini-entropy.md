## 해설

### 출제 배경 및 의도

빈번하게 출제된 데이터 분석 기법에 이어 심화된 알고리즘이 출제. 105회 정보관리 기출문제

### 1. 분류(classification)와 회귀(regression) 분석 기법, 의사결정나무의 개요

정 의 • 모형의 구축과정을 나무형태로 표현하여 대상이 되는 집단을 몇 개의 소집단으로 구분하는 분류 및 예측 기법
      - 의사 결정 나무의 형성에 있어서 분석의 목적과 자료의 구조에 따라 적절한 분리기준(Split Criterion)과 정지규칙(Stopping Rules)을 정의
      - 엔트로피, 지니 지수의 불순도 수치화한 지표를 사용하여 측정 자식마디의 순수도를 가장 높이는 변수를 분리 기준으로 결정함

```
목표 변수 유형별 평가 기준:
이산형 목표변수 → 지니 지수, 엔트로피 지수 (CART, C4.5)
연속형 목표변수 → F통계, 분산의 감소량
```

### 2. 분류나무 불순도 측정, 지니 지수와 엔트로피 지수의 설명

가. 지니 지수(Gini Index)의 설명

| 항목 | 내용 |
|------|------|
| 수식 | G(A) = 1 - Σk=0 to n (pk²) |
| 개념 | CART에서 사용하는 불순도 알고리즘으로 복원추출 개념을 이용하여 집합내에 이질적인 것이 얼마나 섞였는지를 측정하는 지표 |
| 설명 | 집합에 있는 항목이 모두 같다면 지니 불순도는 최솟값(0) |
| 설명 | 지니 지수가 높을수록 노드 내의 이질성이 크고 순수도가 낮음 |
| 설명 | 지니 지수가 낮을수록 같은 특성을 가진 객체들로 분류가 잘 된 상태로 판단 |

성별 예시 계산:

```
G(상위) = 1-(5/10)²-(5/10)² = 0.5
G(남)   = 1-(5/6)²-(1/6)²   = 0.278
G(여)   = 1-(4/4)²-(4/4)²   = 0 (완전 순수)
G(성별) = (6/10)(0.278)+(4/10)(0) = 0.167
→ 부모 노드 불순도 0.5→0.167 감소로 성별 분리의 정확도가 높음
```

나. 엔트로피 지수(Entropy Index)의 설명

| 항목 | 내용 |
|------|------|
| 수식 | E(A) = -Σk=1 to m (pk·log₂(pk)) |
| 개념 | C4.5에서 사용하는 불순도 알고리즘으로 log를 사용하여 집합의 무질서한 정도를 측정하는 지표 |
| 설명 | 지수가 1에 가까울 경우 불순도가 높음 |
| 설명 | 0에 가까울 경우 불순도가 낮음 |
| 설명 | 지니 지수와 마찬가지로 불순도와 엔트로피(Entropy)는 비례관계 |

성별 예시:

```
E(상위)  = -(5/10)log₂(5/10)-(5/10)log₂(5/10) = 1
E(성별)  = (6/10)(0.65)+(4/10)(0)               = 0.39
IG(성별) = E(상위)-E(성별) = 1-0.39             = 0.61
→ IG(정보이득)=0.61로 값이 클수록 변별력이 좋음을 의미
```

### 3. 재귀적 분할 알고리즘에 따른 비교

| 구분 | CART | C4.5 | CHAID |
|------|------|------|-------|
| 분류나무(분류) | ○ | ○ | ○ |
| 회귀나무(예측) | ○ | ○ | X |
| 예측변수 | 범주, 수치 | 범주, 수치 | 범주 |
| 불순도 알고리즘 | 지니 지수 | 엔트로피지수, GI | Chi-square 통계량 |
| 분리 | 이지분리 | 다지분리 | 다지분리 |
| 나무성장(멈추기) | 완전모형 개발 | 완전형 개발 | 최적모형 개발 |
| 가지치기(교차검정) | 학습데이터>검증데이터 | 학습데이터 | X |
