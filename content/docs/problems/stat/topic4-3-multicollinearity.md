## 해설

### 출제 배경

AI·빅데이터 분석에서 다중회귀분석의 신뢰도를 저하시키는 다중공선성 문제에 대한 이해 확인.

### 1. 다중회귀분석의 신뢰성 위협, 다중공선성 개요

정 의  • 다중회귀분석에서 두 개 이상의 독립변수들 사이에 강한 선형관계가 존재하는 현상

```
[독립변수1]   ←── 강한 상관관계 ──→   [독립변수2]
      ↘                                     ↙
          ──────→  [종속변수]  ←──────
              
다중공선성 발생 → 회귀계수 추정 불안정 → 모델 신뢰성 저하
```

발생 원인:
• 상관관계 높은 독립변수를 함께 투입할 때
• 더미변수 처리 오류 (완전한 다중공선성)
• 변수의 수가 표본 수보다 많을 때
• 데이터 범위가 협소할 때

### 2. 다중공선성 탐지 방법 및 영향

1) 탐지 방법

| 구분 | 방법 | 설명 |
|------|------|------|
| 탐지 | 상관계수 행렬 분석 | 독립변수 간 상관계수가 0.7~0.9 이상이면 의심 |
| 탐지 | VIF(Variance Inflation Factor) | 분산팽창계수. VIF > 10이면 다중공선성 문제 심각 / VIF = 1/(1-R²ⱼ) |
| 탐지 | 상태지수(Condition Index) | 조건수 > 30이면 다중공선성 존재 의심 |
| 탐지 | 고유값(Eigenvalue) 분석 | 고유값이 0에 가까울수록 다중공선성 심각 |

2) 다중공선성의 영향

| 영향 | 설명 |
|------|------|
| 회귀계수 불안정 | 독립변수 하나 추가/제거 시 다른 회귀계수가 크게 변함 |
| 표준오차 증가 | 추정된 회귀계수의 표준오차가 크게 증가 |
| 통계적 유의성 왜곡 | t-통계량이 작아져 유의한 변수가 유의하지 않게 나타남 |
| 예측력 저하 | 개별 독립변수의 영향 분리가 불가능해 예측 모형 신뢰성 저하 |

### 3. 다중공선성 해결 방법

| 구분 | 해결방안 | 설명 |
|------|---------|------|
| 변수 조정 | 상관관계 높은 변수 제거 | 다중공선성이 높은 독립변수 중 하나를 제거 |
| 변수 조정 | 변수 결합 | 상관관계 높은 두 변수를 합산하거나 평균을 구하여 새로운 변수 생성 |
| 차원 축소 | 주성분 분석(PCA) | 독립변수들을 상호 독립적인 주성분으로 변환 |
| 정규화 | Ridge Regression (L2) | 다중공선성 존재 시 계수 안정화에 가장 효과적 |
| 데이터 | 표본 크기 증가 | 더 많은 데이터 수집으로 추정 안정성 향상 |
| 중심화 | 변수 중심화(Centering) | 변수에서 평균을 빼 공선성 감소 |

- 다중공선성은 다중회귀분석의 대표적인 문제점으로, Ridge Regression을 통한 정규화가 가장 효과적인 해결책
- 최근 AI/ML 모델에서도 Feature Selection과 PCA를 통한 다중공선성 제거가 모델 안정성 향상에 기여.  "끝"

### 실무 제언

**VIF 기반 변수 선택 전략**

- **챌린지**: 다중공선성 진단 시 VIF 값을 기준으로 단계적으로 변수를 제거하는 Stepwise 방법을 사용합니다. 그러나 무조건적인 변수 제거는 모델의 설명력을 저하시킬 위험이 있습니다.
- **제언**: VIF > 10인 변수 중 종속변수와 상관관계가 낮은 것을 우선 제거하되, 도메인 지식을 함께 고려해야 합니다. 무조건적인 변수 제거보다는 Ridge Regression 적용을 먼저 검토하는 것이 권장됩니다.
