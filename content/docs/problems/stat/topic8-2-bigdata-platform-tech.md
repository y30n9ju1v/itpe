### 1. 데이터 기반 4차 산업혁명 구현을 위한 빅데이터 플랫폼의 중요성

**정 의**
- 빅데이터에서 가치를 추출하기 위해 일련의 과정(수집→저장→처리→분석→시각화)을 규격화한 기술
  - 데이터 수집·저장·분석·활용 기술 최적화 및 플랫폼 연계 기술 최적화 제공

```
[빅데이터 플랫폼]          [통합 데이터지도/데이터 댐]     [기대효과]
데이터 수집·저장         →  국가 데이터 통합 제공      →  국가 데이터 산업 발전
분석·활용 기술 최적화       데이터 댐 구축                데이터 기반 4차 산업혁명
플랫폼 연계 기술 최적화                                   비즈니스 활용 가능
```

---

### 2. 빅데이터 플랫폼 구현을 위한 주요 기술 개요

**가. 빅데이터 플랫폼 개념도**

- 빅데이터 처리 플랫폼(데이터수집 → 빅데이터 분석/처리/저장관리) + 빅데이터 컴퓨팅 인프라(분산/고성능/인메모리)

**나. 빅데이터 플랫폼 구현을 위한 주요 기술**

| 기술 영역 | 주요 기술 | 설명 |
|-----------|-----------|------|
| 데이터 수집 | ETL, 크롤링, 스쿱, 플룸, 스크라이브 | 내부와 외부의 매체로부터 수동/자동으로 데이터 수집, 원천 데이터의 정형/반정형/비정형 데이터 수집 |
| 데이터 저장 | RDBMS, Hadoop(HDFS, 맵리듀스), NoSQL | 정형 데이터, 반정형 데이터, 비정형 데이터 저장, 데이터를 빠르고 쉽게 분석할 수 있도록 저장 형태를 미리 정의 |
| 데이터 분석 | Data Mining, 기계학습, SNS 분석 | 텍스트 분석, 머신러닝, 통계, 데이터 마이닝을 활용하여 목적에 맞도록 분석 |
| 데이터 활용 | 히스토그램, Data Visualization, 인포그래픽 | 데이터 가시화 및 BI, Open API 연계 |

---

### 3. 빅데이터 플랫폼 구현을 위한 주요 기술 상세

**가. 수집 및 저장 측면**

| 구분 | 기술 | 설명 |
|------|------|------|
| 데이터 수집 | ETL | 수집 대상 데이터를 추출, 가공하여 DW/데이터마트 저장 |
| | 크롤링/웹크롤링 | URL 리스트 통해 링크 존재와 HTML 코드 유효성 검사 |
| | Open API | 다양한 서비스와 데이터를 쉽게 이용할 수 있도록 공개한 개발자용 인터페이스 |
| 데이터 저장 | NoSQL | 확장 가능한 성능 및 스키마 없는 데이터 모델에 최적화된 비관계형 DB |
| | HDFS | 대용량 파일을 분산된 서버에 저장하고, 빠르게 처리할 수 있게 하는 하둡 분산 파일 시스템 |
| | 맵리듀스 | 대용량 데이터 세트를 분산 병렬 컴퓨팅에서 처리하거나 생성하기 위한 목적으로 만들어진 소프트웨어 프레임워크 |

**나. 분석 및 활용 측면**

| 기술 | 설명 |
|------|------|
| Data Mining | 대용량 데이터로부터 데이터 내에 존재하는 패턴, 관계 혹은 규칙 등을 탐색하고 통계적인 기법들을 활용하여 모델화하며 이를 통해 빅데이터 분석에 이용하는 기법 |
| 기계학습 | 인공지능의 분야 중 하나로, 인간의 학습 능력과 같은 기능을 컴퓨터에서 실현하고자 하는 기술 |
| SNS 분석 | 빅데이터를 비즈니스에 활용하기 위하여 사회 연결망 데이터를 활용하여 사회 연결망과 사회 구조 등을 사회과학적으로 분석하는 기법 |
| Data Visualization | 분석된 데이터의 의미 있는 정보를 보일 수 있도록 정보화 및 시각화 기술 |
| 인포그래픽 | 중요 정보를 하나의 그래픽으로 표현해서 보는 사람들이 쉽게 정보를 이해할 수 있도록 만드는 시각화 기술 |

### 4. 빅데이터 플랫폼을 통한 산업 발전 방안

```
데이터 댐 구축            →  통합 데이터 지도 활성화      →  AI 산업 발전
(4차 산업혁명 기반 조성)     (다양한 산업 군 연계 활용을      (민간 AI 활용
                             통한 일자리 창출)               비즈니스 창출)
```

- 높은 수준의 빅데이터 플랫폼 구현과 연계 활용을 통해 국가 발전 전략의 기반을 마련하고, 다양한 비즈니스 활용 가능 "끝"
