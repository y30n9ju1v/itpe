## 해설

### 출제 배경 및 의도

빅데이터·AI 시대의 핵심인 데이터 분석의 전반적 프로세스(EDA → 전처리 → 모델링 → 평가)와 지도·비지도 학습 기법, 분류·군집 모델의 정량 평가 방법을 종합적으로 서술할 수 있는지 확인하기 위해 출제

### 1. AI·빅데이터의 핵심 영역, 데이터 분석 개요

정 의  • 데이터 탐색·전처리·분석모델링·평가의 순환 과정을 통해 데이터로부터 인사이트를 도출하고 비즈니스 가치를 창출하는 체계적 분석 프로세스
       - 분석 기법(회귀/분류/군집) 선택 → 모델 평가(성능/정확도) 지표 활용
       - 단방향 순차 흐름이 아닌 각 단계가 유기적으로 연결된 순환과정

```
[데이터 분석 순환 구조]

  ┌──────────────────────────────────────┐
  │                                      │
  ▼                                      │
데이터 EDA ──▶ 데이터 전처리 ──▶ 분석 모델링 ──▶ 모델 평가
  ▲                                             │
  └─────────────────────────────────────────────┘
                  (순환·반복)
```

### 2. 데이터 분석 Flow 4단계

| 단계 | 활용 | 내용 |
|------|------|------|
| 데이터 EDA | 분석 대상 데이터 특성 이해 | 특별한 방법 없이 다양한 관점으로 깊이 있게 데이터 파악, 도구를 통한 시각화, 통계 지표 활용, 변수들 간 상관관계 이해 |
| 데이터 전처리 | 모델링 활용 수준으로 데이터 가공 | 누락값·이상치 등 결측치에 대한 대체 방안 적용, 변수 특성에 따라 단위·값을 변경(스케일링, 인코딩 등) |
| 분석 모델링 | 머신러닝 기법을 통한 분석 모델 생성 | 전체 데이터를 학습/테스트 데이터로 분할, 학습 데이터를 통해 예측 모델 생성 |
| 분석 모델 평가 | 분석 모델의 정확도 평가 | 테스트 데이터를 통해 예측 모델 적용 및 평가, 예측값 타입·분류 목적 등에 따라 다양한 평가 지표 활용 |

### 3. 데이터 분석 기법

#### 가. 지도학습(Supervised Learning) 분석 기법

| 구분 | 기법 | 내용 |
|------|------|------|
| 회귀분석 | 선형 회귀 | 독립변수와 종속변수 간의 선형 관계를 모델링 |
| 회귀분석 | 제약 선형 회귀 | Ridge(L2), Lasso(L1), Elastic Net — 규제항 추가로 과적합 방지 |
| 회귀분석 | 로지스틱 회귀 | 종속변수가 범주형일 때 사용, 시그모이드 함수로 확률 출력 |
| 분류분석 | 의사결정 나무 | 불순도(지니/엔트로피) 측정 기반으로 노드를 분할하여 분류 규칙 도출 |
| 분류분석 | SVM | 최적 경계선(Hyperplane)과 Margin을 최대화하는 초평면 탐색 |

```
[로지스틱 회귀 시그모이드]          [SVM Hyperplane]

  1 ┤     ╭─────────             │ ○ ○  /Margin/  ● ●
    │    ╱                       │    ╱           
0.5 ┼───╱──────────              │   ╱ Hyperplane
    │  ╱                         │  ╱           
  0 ┼╯                           │ ╱
    └─────────────               └───────────────
```

#### 나. 비지도학습(Unsupervised Learning) 분석 기법

| 구분 | 기법 | 내용 |
|------|------|------|
| 계층적 군집 | 계층적 군집분석 | 거리 기반 군집화. 합병(Bottom-up) / 분할(Top-down) 방식. 군집 연결 방법: 단일·완전·평균·가중·중심 연결 |
| 비계층적 군집 | K-Means | 군집 중심(Centroid)과 거리가 가까운 객체들을 묶어 군집 형성, K값 사전 지정 필요 |
| 비계층적 군집 | Mean-Shift Clustering | 초기점에서 평균점을 계산하며 슬라이딩 윈도우를 이동하여 밀도 높은 영역 탐색 |
| 비계층적 군집 | DBSCAN | 밀도 접근성 기반 Core Point 중심의 군집화, 노이즈 포인트 자동 처리 |
| 비계층적 군집 | EM 알고리즘 | K-Means와 유사하나 확률분포 기반. E-step(기대값 계산)·M-step(파라미터 최대화) 반복 |

### 4. 분류 및 군집 모델의 평가 방법

#### 가. 분류 모델 — Confusion Matrix

```
                   예측값
                 Positive  Negative
실제값  Positive │   TP   │   FN   │
        Negative │   FP   │   TN   │
```

| 평가지표 | 수식 | 설명 |
|----------|------|------|
| Accuracy(정확도) | (TP+TN) / (TP+TN+FN+FP) | 전체 중 올바르게 분류한 비율 |
| Precision(정밀도) | TP / (TP+FP) | Positive 예측 중 실제 Positive 비율 |
| Recall(재현율) | TP / (TP+FN) | 실제 Positive 중 올바르게 탐지한 비율 |
| F1 Score | 2 × (Precision × Recall) / (Precision + Recall) | 정밀도·재현율의 조화평균 |

```
[ROC Curve]

TPR(재현율)
  1 ┤    ╭────────────
    │   ╱  AUC
    │  ╱  (1에 가까울수록
    │ ╱    좋은 모델)
  0 ┼─────────────────
    0                1
         FPR(위양성률)
```

- **ROC Curve**: Y축 = True Positive Rate(재현율), X축 = False Positive Rate
- **AUC**: ROC 곡선 아래 면적, 1에 가까울수록 우수한 모델

#### 나. 군집 모델 — 실루엣 계수(Silhouette Coefficient)

```
Si = (bi - ai) / max(ai, bi)
```

| 기호 | 의미 |
|------|------|
| ai | 같은 군집에 속한 객체들 간의 평균 거리 (작을수록 군집 내 응집도 높음) |
| bi | 다른 군집 중 가장 가까운 군집까지의 평균 거리 (클수록 군집 간 분리도 높음) |
| Si | -1 ~ 1 범위, 1에 가까울수록 군집화 우수 |

- **ARI(Adjusted Rand Index, 조정랜드지수)**: 정답 레이블이 있을 때 군집 결과와의 일치도를 측정하는 보정 지수

끝

### 실무 제언

데이터 분석 Flow의 각 단계는 독립적이지 않고 상호 피드백이 이루어지는 순환 구조이므로, 실무에서는 EDA → 전처리 → 모델링 → 평가 결과를 바탕으로 전처리 재조정·특성 공학 반복이 일반적이다. 특히 불균형 데이터(Imbalanced Data)에서는 Accuracy보다 F1 Score·AUC를 우선 활용해야 하며, 군집 모델은 정답 레이블 없이 실루엣 계수로 최적 K값을 탐색하는 것이 핵심이다.
