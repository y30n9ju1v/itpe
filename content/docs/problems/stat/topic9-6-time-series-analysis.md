## 해설

### 출제 배경 및 의도

데이터를 이용해 시간에 흐름에 따라 미래의 변화에 대한 추세를 분석하기 위해 시계열 분석을 많이 사용하므로 출제 가능성이 있음

### 1. 시계열 분석에 사용되는 정상성

가. 시계열 분석 개념

정 의  • 연도별, 분기별, 월별 등 시계열로 관측되는 자료를 분석하여 미래를 예측하기 위한 분석 기법
       - AR, MA, ARIMA 모형을 통해 추세·계절·순환·불규칙 요인 분해 가능

나. 정상성(Stationary) 개념 및 조건

| 구분 | 내용 |
|------|------|
| 개념 | 시점에 상관없이 시계열의 특성이 일정한 특성 |
| 조건 ① | 평균이 일정 |
| 조건 ② | 분산이 시점에 의존하지 않음 |
| 조건 ③ | 공분산은 단지 시차에만 의존하고 시점 자체에는 의존하지 않음 |
| 비고 | 시계열 분석을 위해서는 정상성을 만족해야 함 |

### 2. 시계열 모형인 AR 모형, MA 모형

가. 자기 회귀 모형(AR 모형; Auto-Regressive Model)

```
Zt = φ₁Zt-1 + φ₂Zt-2 + ... + φpZt-p + at
```

- Zt-k: k시점 이전의 시계열 자료
- φp: p 시점이 현재 시점에 어느 정도 영향을 주는지 나타내는 모수
- at: 백색잡음과정(시계열 분석에서 오차항)
- 현시점의 자료가 p 시점 전의 유한개의 과거 자료로 설명될 수 있는 모형

나. 이동평균 모형(MA 모형; Moving Average Model)

```
Zt = at - θ₁at-1 - θ₂at-2 - ... - θqat-q
```

- Zt: 현시점 시계열 자료
- θq: q 시점이 현재 시점에 어느 정도 영향을 주는지 나타내는 모수
- 현 시점의 자료를 유한개의 백색 잡음의 선형결합으로 표현
- 현시점의 자료를 유한개의 백색 잡음의 선형결합으로 표현되었기 때문에 항상 정상성을 만족하므로 정상성 가정이 필요 없음

### 3. 시계열 모형인 ARIMA 모형과 차수

가. 자기 회귀 누적 이동평균 모형(ARIMA 모형; Auto Regressive Integrated Moving Average Model)

```
AR(p) 모형 ─┐
            ├→ ARMA(p,q) 모형 ──차분/변환──→ ARIMA(p,d,q) 모형
MA(q) 모형 ─┘
```

- 분기/반기/연간 단위로 다음 지표를 예측하거나 주간/월간 단위로 지표를 리뷰하여 트렌드를 분석하는 기법

나. ARIMA 모형 차수

| 차수 | 관련 모형 | 설명 |
|------|-----------|------|
| p | AR 모형과 관련이 있는 차수 | 차수=0일 때 IMA(d,q) 모형, 이 모형을 d번 차분하면 MA(q) 모형이 됨 |
| q | MA 모형과 관련이 있는 차수 | AR(p,d) 모형, 이를 d번 차분한 시계열 모형이 AR(p) 모형을 따르게 됨 |
| d | ARIMA에서 ARMA로 정상화할 때 차분 횟수 | ARMA(p,q) 모형, 이 모형은 정상성을 만족 |
| 비고 | ARIMA(p,d,q) 모형은 차수 p,d,q의 값에 따라 모형의 이름이 다르게 됨 | |

### 4. 시계열의 분해 요인

| 요인 | 개념 | 예시 |
|------|------|------|
| 추세 요인(Trend Factor) | 자료가 어떤 특정한 형태를 취함 | 선형적 추세, 이차식 형태, 지수적 형태 등 |
| 계절 요인(Seasonal Factor) | 고정된 주기에 따라 자료가 변화할 경우 | 요일마다 반복, 일 년 중 각 월에 의한 변화, 사분기 자료에서 각 분기에 의한 변화 등 |
| 순환 요인(Cyclical Factor) | 알려지지 않은 주기를 가지고 자료가 변화하는 요인 | 명백한 경제적이나 자연적인 이유가 없이 알려지지 않은 주기를 가지고 변화 |
| 불규칙 요인(Irregular Factor) | 추세, 계절, 순환 요인으로 설명할 수 없는 회귀분석에서 잔차에 해당하는 요인 | |

- 시계열 분해는 시계열에 영향을 주는 일반적인 요인을 시계열에서 분리해 분석하는 방법
