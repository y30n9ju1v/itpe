## 해설

### 출제 배경 및 의도

- **연계 토픽**: 123회 기출 주성분분석(PCA) 출제와 연계된 심화 토픽으로, PCA와 LDA의 차이를 비교·설명하는 능력을 검증
- **출제 의도**: 고차원 데이터의 차원의 저주(Curse of Dimensionality) 문제를 해결하기 위한 두 가지 대표적 차원 축소 기법의 원리·절차·활용 차이를 이해하고 있는지 평가

### 1. 변수 차원 축소, PCA와 LDA의 개념 비교

1) 정 의

정 의 (PCA)  • 다차원 공간에서 데이터가 분포되어 있을 때 가장 효율적인 최적의 축을 식별하여 차원을 축소하는 알고리즘 (비지도학습)

정 의 (LDA)  • 다차원 공간에서 데이터가 분포되어 있을 때 클래스를 최적으로 구분할 수 있는 특성 부분 공간을 찾는 알고리즘 (지도학습)

2) 구성도

```
[원본 고차원 데이터]
        │
        ├─── PCA (비지도) ──→ 분산 최대 방향의 축으로 차원 축소
        │                        (클래스 정보 미사용)
        │
        └─── LDA (지도)  ──→ 클래스간 분산↑ / 클래스내 분산↓ 방향으로 차원 축소
                                 (클래스 레이블 활용)
                                         │
                              [저차원 특성 공간]
```

3) PCA vs LDA 비교표

| 구분 | PCA | LDA |
|------|-----|-----|
| 학습 방식 | 비지도학습 | 지도학습 |
| 목적 | 데이터 분산 최대화 | 클래스 분리 최대화 |
| 레이블 사용 | 미사용 | 사용 (클래스 정보 필요) |
| 최적화 기준 | 공분산행렬의 고유벡터 | S(W)⁻¹·S(B) 의 고유벡터 |
| 주요 활용 | 노이즈 제거, 시각화, 압축 | 분류 전처리, 특성 추출 |
| 한계 | 클래스 구분 고려 안 함 | 클래스 수 − 1 이하로 축소 제한 |

### 2. PCA와 LDA의 개념도 및 분석절차 비교

가. 개념도 비교

```
[PCA 개념도]                          [LDA 개념도]
  데이터 포인트 분포                    클래스 A (●)  클래스 B (▲)

      ●  ●                                  ● ●  │  ▲ ▲
    ●  ●  ●   ← PC1(분산 최대 방향)         ● ●  │  ▲ ▲
      ●  ●                                       │
            ↕ PC2 (PC1과 직교)           ← 클래스내 분산(S(W)) 최소화
                                         → 클래스간 분산(S(B)) 최대화
                                                  │
                                          [판별 축(LD1)]
```

- PCA와 LDA는 모두 행렬을 고유값(Eigenvalue)과 고유벡터(Eigenvector)로 분해하여 저차원 특성 공간을 구성

나. 분석절차 비교

| 단계 | PCA | LDA |
|------|-----|-----|
| 1. 전처리 | 데이터 정규화 (평균=0, 분산=1) | 데이터 정규화 (평균=0, 분산=1) |
| 2. 행렬 계산 | 공분산행렬 계산 (평균값·편차를 통한 공분산) | 산포행렬 계산: 클래스간 S(B) 및 클래스내 S(W) 구성 |
| 3. 고유값 분해 | 공분산행렬 분해 → 고유값·고유벡터 계산 | S(W)⁻¹·S(B) 곱행렬을 통한 고유값·고유벡터 계산 |
| 4. 데이터 투영 | 상위 k개 고유벡터로 저차원 투영 | 상위 k개 판별벡터로 저차원 투영 |

- S(B): Between-class Scatter Matrix (클래스간 산포행렬)
- S(W): Within-class Scatter Matrix (클래스내 산포행렬)

### 3. PCA를 활용한 알고리즘

| 알고리즘 | 설명 |
|---------|------|
| 랜덤 PCA (Randomized PCA) | 확률적 알고리즘으로 주성분에 대한 근사값을 빠르게 찾는 알고리즘 (대용량 데이터에 유리) |
| 점진적 PCA (Incremental PCA) | 훈련 데이터셋을 미니배치로 나눈 뒤 분석을 수행하는 알고리즘 (메모리 효율적) |
| 커널 PCA (Kernel PCA) | 비선형 함수인 커널 함수를 이용하여 고차원 매핑 후 PCA 알고리즘을 적용하는 기법 (비선형 구조 처리 가능) |

- PCA와 LDA는 차원 축소를 진행하므로 차원의 저주(Curse of Dimensionality)를 해결하는 알고리즘으로 활용  "끝"

### 실무 제언

**PCA/LDA 선택 기준 및 활용 사례**

- **선택 기준**: 레이블(클래스 정보)이 존재하고 분류 성능 향상이 목적이면 LDA를, 레이블이 없거나 데이터 압축·시각화·노이즈 제거가 목적이면 PCA를 선택
- **복합 활용**: 고차원 이미지 데이터(예: 얼굴 인식)에서 먼저 PCA로 노이즈를 제거한 후 LDA로 클래스를 분리하는 PCA+LDA 파이프라인이 실무에서 효과적
- **주의사항**: LDA는 클래스 수 − 1 개 이하의 차원으로만 축소 가능하므로, 클래스 수가 적은 경우 PCA로 보완하거나 커널 LDA(KLDA)를 검토해야 함
