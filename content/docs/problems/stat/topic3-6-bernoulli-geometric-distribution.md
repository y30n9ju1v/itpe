## 해설

### 1. 베르누이 분포(Bernoulli Distribution) 개념

| 구분 | 설명 |
|------|------|
| 개념 | 상호 배반인 두 가지 가능한 결과 중 하나를 갖는 베르누이 시행을 나타내는 확률 분포 |
| 수식 | 성공확률: p, 실패확률: (1-p) / X의 표본공간 S={0,1} / P(X=0)=1-p, P(X=1)=p |
| 평균 | E(X) = p |
| 분산 | Var(X) = p(1-p) = pq |
| 사례 | 숫자 1부터 10까지 적혀 있는 카드가 10장 있고, 이 중 하나를 뽑았을 때 8이 적힌 카드가 나올 확률 실험 |

베르누이 분포 특징:
• 1회 시행에서 성공(1) 또는 실패(0) 두 가지 결과만 존재
• 각 시행은 독립적
• 이항분포(Binomial Distribution)의 특수한 경우 (n=1)

### 2. 기하분포(Geometric Distribution) 설명

| 구분 | 설명 |
|------|------|
| 개념 | 독립적인 베르누이 시행을 여러 번 수행할 때 처음 성공까지 시도한 횟수의 분포 |
| 수식 | 확률질량함수: P(X=x) = (1-p)^(x-1) × p / 누적분포: P(X≤x) = 1-(1-p)^x |
| 예상치 | E(X) = 1/p |
| 분산 | Var(X) = (1-p)/p² |
| 사례 | 어느 야구 선수가 홈런을 칠 확률은 5%이다. 이 선수가 6번째 타석에서 첫 홈런을 칠 확률 실험 |

기하분포 특징:
• 처음 성공이 일어날 때까지의 시행 횟수를 모델링
• 무기억성(Memoryless Property): 과거 실패 횟수와 무관하게 미래 성공 확률 동일
• 성공확률 p가 클수록 분포가 왼쪽으로 치우침 (빨리 성공)

### 3. 베르누이 분포와 기하분포 비교

| 구분 | 베르누이 분포 | 기하분포 |
|------|-------------|---------|
| 시행 횟수 | 1회 시행 | 첫 성공까지 반복 시행 |
| 관심 대상 | 1회 시행의 결과(성공/실패) | 첫 성공까지의 시행 횟수 |
| 평균 | p | 1/p |
| 분산 | pq | (1-p)/p² |
| 활용 | 단일 이벤트 성공/실패 확률 | 대기 시간, 신뢰도 분석 |

- 확률, 통계학, 정보 이론 등 다양한 분야에서 활용되며, 확률 모델링과 추론에 중요한 역할.  "끝"
