## 해설

### 1. 기술통계와 추론통계 개요

정 의  • 기술통계(Descriptive Statistics): 모집단으로부터 표본 추출 후
         표본의 중심위치, 산포, 빈도수 파악, 그래프 작성 등
       • 추론통계(Inferential Statistics): 기술통계 기반으로 모집단을
         추정하는 통계

```
[Population] ──샘플링──→ [Sample]
   모집단                   표본
              ←─추론──────
           Descriptive statistics (기술통계)
```

### 2. 표본 추출 방법

| 구분 | 추출 기법 | 설명 |
|------|-----------|------|
| 확률적 추출방법 | 단순임의 추출법(Simple Random Sampling) | 모든 표본추출방법의 기본. 크기 N인 모집단으로부터 크기 n인 표본을 추출할 때 NCn가지의 모든 가능한 표본을 동일한 확률로 추출하는 방법 |
| 확률적 추출방법 | 층화추출법(Stratified Sampling) | 유사한 단위들로 묶은 여러 개의 부모집단으로 나누어 층(Stratum)을 형성한 후 각 층으로부터 단순임의 추출법에 의해 표본을 추출하는 방법 |
| 확률적 추출방법 | 계통 추출법(Systematic Sampling) | 추출단위에 일련번호를 부여하고, 이를 등간격으로 나눈 후 첫 구간에서 무작위로 선정한 다음 등간격으로 떨어져 있는 번호들을 계속해서 추출해가는 방법 |
| 확률적 추출방법 | 집락(군집) 추출법(Cluster Sampling) | 모집단을 집락(Cluster)으로 나누고, 이들 집락들 중에서 일부의 집락을 추출한 후 추출된 집락에서 일부 또는 전부를 표본으로 추출하는 방법 |
| 비확률적 추출방법 | 유의추출법(Purposive Sampling) | 모집단 특성에 대해 조사원이 정확히 알고 있는 경우에 제한적으로 사용 |
| 비확률적 추출방법 | 판단추출법(Judgement Sampling) | 조사원이 자신의 지식과 경험에 의해 모집단을 가장 잘 대표한다고 여겨지는 표본을 주관적으로 판단하여 추출 |
| 비확률적 추출방법 | 할당추출법(Quota Sampling) | 모집단이 여러 특성으로 구성된 경우 각 특성에 따라 층을 구성 후 조사원이 직접 선정 |
| 비확률적 추출방법 | 편의추출법(Convenience Sampling) | 모집단에 대한 정보가 전혀 없는 경우 선정의 편의성에 기준을 두고 마음대로 선정 |
| 비확률적 추출방법 | 눈덩이 추출법(Snowball Sampling) | 소수의 응답자를 찾은 다음 이들과 비슷한 사람들을 소개받아가는 식으로 추출 |

### 3. 형태에 따른 데이터 구분

| 구분 | 데이터 유형 | 설명 |
|------|------------|------|
| 범주형 자료 | 명목형 자료 | 측정 대상의 특성을 분류한 변수(숫자로 바꾸어도 그 값이 크고 작음을 나타내는 것이 아니라 단순히 범주를 표시) / 예: 성별, 혈액형 |
| 범주형 자료 | 순서형 자료 | 측정 대상의 특성을 몇 개의 범주로 구분할 뿐만 아니라 그 범주들 사이에 순서 관계가 성립하는 변수 / 예: 우선순위, 등수, 학점, 선호도 |
| 수치형 자료 | 이산형 자료 | 셀 수 있는 형태의 자료(countable data) / 예: 멤버의 수 |
| 수치형 자료 | 연속형 자료 | 연속적 속성을 가지는 자료. 이산화를 통해 자연수 형태로 표시되는 경우가 많음 / 예: 신장, 체중 |

### 4. 왜도(Skewness)와 첨도(Kurtosis) 비교

| 구분 | 왜도(Skewness) | 첨도(Kurtosis) |
|------|---------------|---------------|
| 개념 | 중심위치로부터 분포가 기울어진 방향과 정도를 나타내는 척도 | 자료의 분포가 중심 경향값을 중심으로 집중적으로 분포되어 있는 정도. 분포의 뾰족한 정도 |
| 판단기준 | S=0: Mode=Median=Mean (좌우대칭) / S>0: 우측으로 긴 꼬리(Positive Skew) / S<0: 좌측으로 긴 꼬리(Negative Skew) | K=0: 표준정규분포와 뾰족한 정도가 같음 / K>0: 표준정규분포보다 뾰족함 / K<0: 표준정규분포보다 납작함 |

### 5. Percentile을 이용한 Outlier 검출 방법

```
Outliers                IQR(Interquartile Range)              Outliers
  ←●●                    │◀────────────────▶│                    ●●→
"Minimum"               Q1     Median       Q3               "Maximum"
(Q1-1.5*IQR)        (25th%)           (75th%)           (Q3+1.5*IQR)
```

검출 방법:
① 자료의 최소값, 최대값을 이용하여 선 표시
② Q1, Q3를 이용하여 Box 그림
③ 상자 가운데 중앙값(Median) 표시하고, 평균은 (+)로 표시
④ IQR 이용하여 가상선을 그림
  - Inner fence = (Q1-1.5IQR, Q3+1.5IQR) → Mild Outlier
  - Outer fence = (Q1-3IQR, Q3+3IQR) → Extreme Outlier
⑤ Inner fence를 넘지 않는 최대, 최소값까지 수염을 그리고, 미달하거나 초과하는 이상치 값은 점이나 원, 별표 등으로 따로 표시

- IQR(Interquartile Range) = Q3 - Q1
- 최근 AI 데이터 전처리에서 IQR 기반 이상치 검출 방법 활용 증대.  "끝"
