## 해설

### 출제 배경 및 의도

데이터 분석의 중요성 증가 및 기출인 CRISP-DM과 함께 미출제인 SEMMA 분석 방법에 대한 이해

### 1. 데이터분석의 중요성 및 데이터 분석방법론의 개요

정 의  • 대규모 데이터로부터 숨겨진 패턴과 알려지지 않은 정보를 찾아내기 위한 과정
       - 빅데이터 분석은 기존의 데이터마이닝 기법과 예측분석 방법 존재

```
[데이터 분석 중요성]
축적가능한 데이터량         분석가능한 데이터량
     ↑ (증가 중)                ↑ (느린 증가)
→ 데이터 GAP(합리적/효율적 의사결정의 기회 손실) 발생
→ AI기술 발전으로 빅데이터 분석의 역할 중요성 증대
```

나. 빅데이터 분석 목적과 역할

| 구분 | 내용 |
|------|------|
| 목적 | 데이터 사이언티스트들에 의해 분석된 정보를 토대로 각 분야의 의사 결정을 수행 |
| 역할 | 의사결정 시 최선의 대안을 선택할 수 있도록 근거를 제시 |

### 2. 데이터 분석 방법론, SEMMA(SAS Enterprise Miner) 개요

가. SEMMA의 개념

- SAS 기업에서 개발한 데이터 마이닝 표준 가이드로써 Sample, Explore, Modify, Model, Assess의 단계로 구성, 각 5단계 프로세스를 순차적으로 실행하는 데이터 분석 방법론

나. SEMMA의 단계

| 단계 | 주요기술 | 설 명 |
|------|---------|-------|
| Sampling | 통계적 추출, 조건 추출 | 분석 데이터 생성, 비용절감 및 모델 평가를 위한 데이터 준비 |
| Explore | 기초 통계, 클러스터링 | 분석 데이터 탐색, 데이터 조감을 통한 데이터 오류 검색 |
| Modify | 수량화, 표준화, 그룹화 | 분석 데이터 수정, 변환, 데이터가 지닌 정보의 표현 극대화 |
| Modeling | 뉴트럴 네트워크, 결정 트리 | 모델 구축, 데이터의 숨겨진 패턴 발견 |
| Assessment | 피드백 | 모델 평가 및 검증(텍스트 교본), 서로 다른 모델을 동시에 비교 |

### 3. 계층적 프로세스 모델, CRISP-DM(Cross Industry Standard Process for Data Mining) 개요

가. CRISP-DM의 개념

- 유럽연합에서 시작된 프로젝트에 SPSS, NCR 등이 참여한 첫 버전, 계층적 프로세스 모델로써 4개 레벨과 6단계로 수행되는 데이터 마이닝 방법론

나. CRISP-DM의 단계

| 단계 | 주요기술 | 설 명 |
|------|---------|-------|
| 비즈니스 이해 | 문제점 파악 | 고객의 요구사항과 비즈니스 목표를 이해, 영향 요인 도출 및 예비 계획 수립 |
| 데이터 이해 | 데이터 셋 수집 | 초기 데이터를 수집, 이해, 데이터의 품질 정의, 가설 검증하기 위한 데이터셋 |
| 데이터 준비 | 사전처리 데이터 | 모델링 도구의 입력데이터를 원시 데이터로부터 추출, 전처리 통해 목표 데이터 추출 |
| 모형 | 모델링(데이터 처리분석) | 모델링 기법 선택 및 최적의 파라미터 변수 설정 |
| 평가 | 검증 | 충분한 평가를 통해 최적의 모델 선정 |
| 적용 | 정리 및 시각화 | 프로세스를 통해서 획득한 지식을 가공하여 고객에게 제시 |

### 4. 데이터 분석 방법론, SEMMA와 CRISP-DM의 비교

가. SEMMA와 CRISP-DM의 분석방법론 비교

| 비교항목 | SEMMA | CRISP-DM |
|---------|-------|---------|
| 개념 | SAS 등에서 추진된 5개 프로세스 기반 데이터마이닝 분석 방법론 | SPSS 등에서 추진된 4 레벨, 6단계 프로세스 기반 데이터마이닝 분석 방법론 |
| 특징 | 개발자 관점에서 데이터 마이닝 집중으로 인해 비즈니스 관점이 미흡 | 비즈니스 문제 인식과 상황을 고려한 데이터 마이닝 결과 해석에 집중함 |
| 활용 TOOL | SAS Enterprise Miner | SPSS Modeler |

나. SEMMA와 CRISP-DM의 데이터 분석 관점 비교

| 분석 관점 | SEMMA | CRISP-DM |
|---------|-------|---------|
| 비즈니스 이해 | 없음 | Business Understanding(고객 요구사항과 비즈니스 목표 이해) |
| 필요 데이터 | Sample(분석 데이터 생성 및 준비), Explore(분석 데이터를 탐색해 기초 통계 변수 유의성 및 상관분석) | Data Understanding(초기 데이터 수집 및 이해, 데이터 품질 정의) |
| 데이터 설계 | Modify(분석 데이터 수정/변환) | Data Preparation(모델링 대상 데이터 준비) |
| 분석 모델링 | Model(모델구축, 데이터의 숨겨진 패턴 발견) | Modeling(모델링 기법을 선택 및 최적 값 가진 파라미터 변수 설정) |
| 분석 평가 | Assessment(모델 평가 및 검증) | Evaluation(평가를 통한 비즈니스 목표 달성 모델 선정) |
| 액션 플랜도출 및 모델개선 | 없음 | Deployment(프로세스를 통한 획득한 지식을 실행 계획을 수립하고 고객에게 제시) |

- 두 방법론의 가장 큰 차이점은 CRISP-DM에서는 프로세스에 비즈니스적/기술적 관점으로 접근하고, SEMMA는 기술적인 작업에 더 많은 관심을 가지고 접근
- 비즈니스 특성과 고객의 요구사항에 따라 적절한 데이터 분석 방법론을 선정해서 활용해야 함
