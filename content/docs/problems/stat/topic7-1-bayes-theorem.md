## 문제

베이즈 정리에 대해 설명하고 다음 문제를 풀이하시오.

미 연준 의장을 선출하는데 후보로 A, B, C 세명이 거론되고 있으며 이들이 선출될 확률은 각각 0.3, 0.5, 0.2 라고 한다. A가 선출되었을 때 이자율을 올릴 확률이 0.8, B는 0.1, C는 0.4라고 한다.

가. 의장 선출이 끝난 후 이자가 오를 확률은 얼마인가?

나. 의장 선출 후 이자가 올랐다면, A가 선출되었을 확률은 얼마인가?

## 해설

### 출제 배경 및 의도

새로운 증거를 기반으로 사전 신념을 업데이트하는 베이즈 추론은 AI/ML에서 핵심 개념

### 1. 데이터를 이용한 사후확률의 추정, 베이즈 정리(Bayes' Theorem)의 개념

정 의  • 두 확률 변수의 사전확률(Prior Probability)과 사후확률(Posterior Probability) 사이의 관계를 나타내는 정리
       - 새로운 정보를 토대로 어떤 사건이 발생했다는 주장에 대한 신뢰도를 갱신해 나가는 방법
       - 이전의 경험과 현재의 증거를 토대로 어떤 사건의 확률을 추론하는 알고리즘

```
P(H|E) = P(E|H)·P(H) / P(E)
  ↑사후확률    ↑우도  ↑사전확률
         ↑전체확률의 법칙
```

전제조건: 증상에 해당하는 n개의 요소 사건 A1,A2,...,An이 서로 배반임, A1,A2,...,An이 Σj=1(Aj)=1을 만족

### 2. 베이즈 정리에 대한 설명

가. 베이즈 정리의 수식

| 구성요소 | 수식 | 설 명 |
|----------|------|-------|
| 베이즈 정리 | P(H\|E) = P(E\|H)P(H)/P(E) | 사전확률과 우도를 이용해 사후확률 산출 |
| 사전확률(Prior Probability) | P(A1), P(A2), P(A3)... | 관측자가 이미 알고 있는 사건(A)의 확률, 현재 가지고 있는 정보를 기초로 하여 정한 초기확률 |
| 우도(Likelihood) | P(B\|A1), P(B\|A2), P(B\|A3)... | 이미 알고있는 사건(A)이 발생했다는 조건하에 다른 사건(B)이 발생할 확률, 사건 발생 후에 어떤 원인으로부터 일어날 것이라고 생각되는 확률 |
| 사후확률(Posterior Probability) | P(Ai\|B) | 사전확률(A)과 우도를 통해 알게 되는 조건부 확률, 추가된 정보로부터 사전정보를 새롭게 수정한 확률 |

나. 베이즈 정리의 증명을 위한 이론

| 이론 | 수식 | 설 명 |
|------|------|-------|
| 조건부 확률 | P(A\|B) = n(A∩B)/n(B) = P(A∩B)/P(B) | B가 발생한 조건하에서 A가 발생할 확률 |
| 확률의 곱셈공식 | P(B∩A) = P(A)P(B\|A) = P(B)P(A\|B) = P(A∩B) | 두 사건의 교집합 확률 표현 |
| 전체 확률의 법칙 | P(A) = P(A\|B1)P(B1)+...+P(A\|Bn)P(Bn) | 배반 사건들의 합산으로 전체 확률 산출 |

### 3. 주어진 문제풀이

가. 주어진 문제 정리

| 구분 | 후보 A | 후보 B | 후보 C |
|------|--------|--------|--------|
| 선출확률 | P(A)=0.3 | P(B)=0.5 | P(C)=0.2 |
| 이자율 인상확률 | P(R\|A)=0.8 | P(R\|B)=0.1 | P(R\|C)=0.4 |
| 이자율 상승확률 | P(A)·P(R\|A)=0.24 | P(B)·P(R\|B)=0.05 | P(C)·P(R\|C)=0.08 |

나. 문제풀이

1) 이자가 오를 확률 P(R):

```
P(R) = P(R∩A) + P(R∩B) + P(R∩C)
     = P(A)P(R|A) + P(B)P(R|B) + P(C)P(R|C)
     = (0.3)(0.8) + (0.5)(0.1) + (0.2)(0.4)
     = 0.24 + 0.05 + 0.08
     = 0.37 (37%)
```

2) 이자가 올랐다면 A가 선출되었을 확률 P(A|R):

```
P(A|R) = P(A∩R) / P(R)
        = P(A)P(R|A) / P(R)
        = (0.3)(0.8) / (0.37)
        = 24/37
        ≈ 64.9%
```

- 사전적 조건확률을 알고 있을 때 조건의 순서를 바꾸어 사후적인 조건확률을 구할 때 베이즈 정리 사용
