## 문제

데이터 마이닝과 통계분석의 차이점을 설명하고, 정형/비정형 데이터 마이닝을 비교하시오.

## 해설

### 출제 배경 및 의도

빅데이터 시대에 데이터 마이닝과 전통적 통계분석의 경계가 모호해지면서 두 방법론의 철학적 차이를 명확히 이해하는 것이 중요해졌다. 이 문제는 귀납적 접근(데이터 마이닝)과 연역적 접근(통계분석)의 차이, 그리고 정형·비정형 데이터 각각에 적합한 마이닝 기법을 비교 서술하도록 요구한다. 오피니언 마이닝과 텍스트 마이닝의 구분까지 포함하면 고득점이 가능하다.

---

### 1. 데이터 마이닝 개요

가. 정의

정 의  • 대규모 데이터에서 패턴·규칙·관계를 자동으로 탐색하는 지식 발견(KDD: Knowledge Discovery in Databases) 과정
       - 통계, 머신러닝, 데이터베이스 기술을 융합하여 숨겨진 인사이트를 추출

나. 데이터 마이닝 주요 기법

```
┌─────────────────────────────────────────────┐
│              데이터 마이닝 기법               │
├──────────────┬──────────────┬───────────────┤
│  분류(Classification) │ 군집(Clustering) │ 연관(Association) │
│  의사결정나무, 신경망  │ K-Means, DBSCAN  │ Apriori, FP-Growth │
└──────────────┴──────────────┴───────────────┘
```

---

### 2. 데이터 마이닝 vs 통계분석 비교

| 구분 | 데이터 마이닝 | 통계분석 |
|------|--------------|----------|
| 목적 | 패턴·규칙 발견 | 가설 검정·추론 |
| 접근 방식 | 데이터 주도(귀납적) | 가설 주도(연역적) |
| 데이터 규모 | 대용량·비정형 처리 가능 | 정형화된 소~중규모 |
| 모델 해석성 | 블랙박스(예측 중심) | 화이트박스(설명 중심) |
| 사전 지식 | 도메인 지식 불필요 | 분포 가정 필요 |
| 주요 기법 | 의사결정나무, 신경망, 군집 | 회귀분석, 검정, ANOVA |

---

### 3. 정형 vs 비정형 데이터 마이닝 비교

| 구분 | 정형 데이터 마이닝 | 비정형 데이터 마이닝 |
|------|------------------|-------------------|
| 데이터 형태 | 구조화된 DB, 스프레드시트 | 텍스트, 이미지, 음성, 영상 |
| 주요 기법 | 연관규칙, 분류, 군집 | 텍스트마이닝, 오피니언마이닝, SNA |
| 전처리 방식 | ETL, 정규화 | NLP, 형태소분석, 이미지처리 |
| 주요 활용 | 판매 예측, 고객 세분화 | 감성분석, 소셜분석, 이상탐지 |

---

### 4. 오피니언 마이닝 vs 텍스트 마이닝 차이

| 구분 | 텍스트 마이닝 | 오피니언 마이닝 |
|------|--------------|---------------|
| 분석 범위 | 비정형 텍스트 전반 분석 | 감성·의견 분석에 특화 |
| 주요 기법 | 분류, 군집, 정보추출 | 긍/부정 판별, 감성사전 활용 |
| 출력 결과 | 주제 분류, 요약, 관계 맵 | 감성 점수, 감성 트렌드 |
| 활용 사례 | 뉴스 요약, 문서 분류 | 제품 리뷰 분석, 브랜드 모니터링 |

나. 결론

빅데이터 시대에 전체 데이터의 약 90%가 비정형 데이터로 추정되므로, 비정형 데이터 마이닝 역량 확보가 기업의 핵심 경쟁우위 원천이다. 정형·비정형을 통합하는 하이브리드 마이닝 파이프라인 구축이 중요하다. 끝

---

### 실무 제언

- 통계분석과 데이터 마이닝을 상호 보완적으로 활용해야 하며, 탐색적 분석(EDA) → 마이닝 → 통계 검정 순서로 신뢰도를 높이는 것이 바람직하다.
- 비정형 마이닝 결과를 정형 데이터베이스에 구조화하여 저장하는 스키마 설계가 후속 분석의 일관성을 결정한다.
- 감성사전(Sentiment Lexicon)은 도메인별로 어휘 극성이 달라지므로 금융·의료·커머스 등 분야에 맞는 전문 사전을 별도 구축해야 정확도를 확보할 수 있다.
