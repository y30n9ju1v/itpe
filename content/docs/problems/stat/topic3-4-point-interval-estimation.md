## 문제

모집단의 특성을 추론하는 점추정과 구간추정에 대하여 비교 설명하시오.

|------|------|
| 토픽(대) | 기술통계 |
| 출제 | 132회 |
| 유형 | 관리 |
| 번호 | 1 |

## 해설

### 1. 통계적 추론 기법, 점추정과 구간추정 정의

| 구분 | 점추정 | 구간추정 |
|------|--------|---------|
| 정의 | 모수를 하나의 값으로 추정하는 기법 | 모수가 포함될 것이라고 기대하는 구간을 추정하는 기법 |

### 2. 점추정과 구간추정 개념도 및 상세특징 비교

1) 개념도 설명

```
점추정:                    구간추정:
  ╭──────╮                   ╭──────╮
 ╱   분포   ╲               ╱   분포   ╲
│           │              │  α/2      α/2 │
│     •θ̂   │              │◀──[─────구간─────]──▶│
 ╲         ╱               ╲             ╱
  ╰─────────╯                ╰─────────╯
   E(θ̂)=θ (불편추정량)         신뢰구간: 모수 포함 구간

- 점추정은 불확실한 정도를 표현하지 못하는 단점 존재 → 구간추정 병행하여 사용 필요
```

2) 점추정과 구간추정 상세 비교

| 구분 | 점추정 | 구간추정 |
|------|--------|---------|
| 주요특징 | 모수에 가장 접근하는 표본 추정치로서 하나의 값을 표현 / 불편성, 효율성, 일치성 만족 필요 | 신뢰수준, 표본오차 등을 통해서 추정값의 정확도 표현 / 신뢰구간, 신뢰수준(90%,95%,99%) 주로사용 |
| 장점 | 직관적 해석 가능 | 신뢰성 정도를 표현할 수 있음 |
| 단점 | 신뢰성 정도를 표현할 수 없음 | 상대적으로 많은 양의 표본이 필요함 |

### 3. 추정과 가설검정 비교

| 구분 | 추정 | 검정 |
|------|------|------|
| 특징 | 모집단의 특성치에 대한 추측값 제공 / 불편성, 효율성, 일치성 만족 필요 | 주장 또는 단순 추측 등의 옳고 그름에 대한 결정을 하는 과정 / 신뢰구간, 신뢰수준(90%,95%,99%) 주로 사용 |
| 종류 | 점추정, 구간추정 | 가설검정, 차이검정 |
| 활용기술 | T분포, 표준정규분포 | 귀무가설, 대립가설, 양측검정, 단측검정, ANOVA, T-test |

- 추정, 검정 등의 통계적 분석 방법을 활용하여 데이터 분석 등 수행.  "끝"
