## 문제

텍스트 마이닝(Text Mining)의 개념, 기능, 기본 분석절차를 설명하고, 이를 활용한 사회연결망 분석과 비정형 데이터 마이닝의 시사점을 설명하시오.

## 해설

### 출제 배경 및 의도

소셜미디어·웹 로그 등 비정형 텍스트 데이터가 폭발적으로 증가함에 따라 NLP 기반의 텍스트 마이닝이 데이터 분석의 핵심 역량으로 부각되었다. 이 문제는 텍스트 마이닝의 개념과 절차를 체계적으로 정리하고, SNA 및 비정형 데이터 마이닝으로의 응용 범위를 묻는다. 기술사 답안에서는 기능·절차·시사점을 논리적으로 연결하여 서술하는 것이 중요하다.

### 1. 텍스트 마이닝 개요

가. 정의

정 의  • 자연어로 구성된 비정형화된 텍스트 데이터에서 패턴 또는 관계를 추출하여 가치 있는 정보를 찾는 기법
       - NLP(자연어처리) 기술을 기반으로 텍스트를 구조화하고, 의미 있는 지식을 자동 발견

나. 텍스트 마이닝 처리 구조

```
┌─────────┐   텍스트 수집/전처리   ┌──────────────────────┐
│  사용자  │ ──────────────────▶  │  자연어처리(NLP) 기반  │
└─────────┘                      │  온톨로지 / 색인 / 지식DB│
                                  └──────────┬───────────┘
                                             │ 텍스트 자료 처리
                                             ▼
                                  ┌──────────────────────┐
                                  │  텍스트 분석·가시화·평가 │
                                  └──────────────────────┘
```

### 2. 텍스트 마이닝 기능

| 기능 구분 | 기능 상세 설명 | 관련 요소 |
|-----------|---------------|-----------|
| 텍스트 분류 | 텍스트 데이터 특성 정의, 데이터 간 유사도 계산하여 분류 | 비지도학습, 군집화 |
| 문서 군집 (Document Clustering) | 각 지식 콘텐츠 특성 파악, 유사·상호관련성 높은 콘텐츠를 군집화 | 유사도계산, 통계, 규칙기반 알고리즘 |
| 정보 추출 | 텍스트 문서 내 중요한 의미 정보를 자동 추출 | 온톨로지, 시맨틱웹, 검색성능개선 |
| 문서 요약 (Document Summarization) | 핵심 의미를 유지하면서 복잡도와 길이를 줄임 | 추출요약, 생성요약 |

### 3. 텍스트 마이닝 기본 분석절차

| 단계 | 절차 | 설명 |
|------|------|------|
| 1단계 | 텍스트 문서 선정/수집 | 분석 목적에 맞는 원천 데이터 수집 (웹크롤링, API, 내부 문서 등) |
| 2단계 | 텍스트 전처리 | 불용어 제거, 형태소 분석, 토큰화, 정규화 |
| 3단계 | 의미정보 변환 | TF-IDF, Word2Vec 등을 통해 텍스트를 수치 벡터로 변환 |
| 4단계 | 의미정보 추출 | 개체명 인식(NER), 핵심어 추출, 감성 분류 수행 |
| 5단계 | 패턴 및 경향 분석 | 연관규칙, 군집분석, 분류 모델 적용하여 패턴 발견 |
| 6단계 | 정보 표현 및 평가 | 워드클라우드, 시각화, 정밀도·재현율로 결과 평가 |

### 4. 비정형 데이터 마이닝의 필요성 및 시사점

| 구분 | 내용 |
|------|------|
| 비정형 정보의 기하급수적 증가 | 웹, 음성, 문서, 동영상 등 온라인 소스로부터 빅데이터 급증 |
| 비정형 데이터의 가시화 요구 | 자연어처리를 통해 고객 감성·의견 분석 요구 증가 |
| 수동 분석의 한계 | 유용한 정보를 효과적으로 추출하기 위한 비정형 데이터 관리 도구 필요 |

나. 사회연결망 분석(SNA)과의 연계

- 텍스트 마이닝으로 추출한 개체·관계 정보를 노드·엣지로 모델링하여 SNA 수행
- 소셜미디어 데이터에서 영향력 있는 사용자(허브)·커뮤니티 구조 탐지에 활용
- 구전 효과, 정보 확산 경로 분석으로 마케팅·위기관리에 기여

다. 결론

텍스트 수집·전처리 → 패턴 분석 → 정보 생성의 순환 구조로 지속적 피드백이 이루어지며, 비정형 데이터의 가치를 극대화하는 핵심 기술로 자리매김하고 있다. 끝

### 실무 제언

- 형태소 분석기(Mecab, Kiwi 등) 선택이 한국어 텍스트 품질에 결정적 영향을 미치므로 도메인 맞춤 사전을 함께 구축해야 한다.
- SNA 적용 시 노이즈 엣지 제거를 위한 임계값(threshold) 설정이 분석 결과의 신뢰도를 좌우한다.
- 오피니언 마이닝에서 부정어·반어 표현 처리 로직을 별도 설계하지 않으면 감성 분류 정확도가 크게 떨어질 수 있다.
