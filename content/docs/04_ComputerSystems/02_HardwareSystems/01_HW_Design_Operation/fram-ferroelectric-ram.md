## 개요

FRAM(Ferroelectric RAM, 강유전체 RAM)은 강유전체(Ferroelectric) 소재의 분극(polarization) 상태를 이용하여 데이터를 저장하는 비휘발성 메모리다. DRAM의 속도와 Flash의 비휘발성을 결합한 특성으로 IoT·자동차·의료 기기에 활용된다.

## 핵심 내용

### 동작 원리

강유전체 커패시터(PZT 소재 등)에 전압을 가하면 내부 분극 방향이 두 가지 안정 상태 중 하나로 전환된다.

- **분극 방향 ↑ → 1 저장**
- **분극 방향 ↓ → 0 저장**
- 전원이 꺼져도 분극 상태 유지 → 비휘발성

### 주요 메모리 기술 비교

| 구분 | DRAM | SRAM | Flash | FRAM |
|------|------|------|-------|------|
| **휘발성** | 예 | 예 | 아니오 | 아니오 |
| **읽기 속도** | 빠름 | 매우 빠름 | 느림 | 빠름 |
| **쓰기 속도** | 빠름 | 매우 빠름 | 매우 느림 | 빠름 |
| **쓰기 내구성** | 높음 | 높음 | 낮음 (~10만 회) | 높음 (~10조 회) |
| **소비전력** | 중간 | 높음 | 낮음(대기) | 매우 낮음 |
| **가격** | 낮음 | 높음 | 낮음 | 높음 |
| **집적도** | 높음 | 낮음 | 매우 높음 | 낮음 |

### FRAM 장점

1. **빠른 쓰기 속도**: Flash 대비 1,000배 이상 빠른 쓰기
2. **높은 내구성**: Flash의 10만 회 대비 10조(10¹³) 회 쓰기 가능
3. **초저전력**: 분극 전환에만 전력 소요, 대기 전력 거의 없음
4. **비휘발성**: 전원 차단 후에도 데이터 유지

### FRAM 단점

1. **낮은 집적도**: Flash 대비 저장 밀도 낮아 고용량 구현 어려움
2. **높은 단가**: 특수 소재(PZT) 사용으로 비용 증가
3. **파괴 읽기**: 일부 FRAM은 읽기 시 데이터 파괴 후 재기록 필요

### 활용 분야

- **IoT 센서 노드**: 저전력 + 빈번한 쓰기 환경
- **자동차 ECU**: 엔진 제어 데이터 즉시 저장
- **스마트 미터**: 전력량 데이터 실시간 기록
- **웨어러블 의료기기**: 신체 신호 지속 기록

## 출제 포인트

- 강유전체 분극 상태로 데이터 저장 (원리)
- Flash 대비 쓰기 속도 1,000배 빠름, 내구성 10조 회
- DRAM/SRAM/Flash/FRAM 4가지 비교표
- 활용 분야: IoT·자동차·의료
