## 개요

유전 알고리즘(Genetic Algorithm)은 다윈의 진화론(자연선택, 유전, 돌연변이)을 모방한 메타휴리스틱 최적화 알고리즘이다. 복잡한 탐색 공간에서 전역 최적해(Global Optimum)를 근사적으로 탐색하며, 그래디언트 정보 없이도 동작한다.

## 핵심 내용

### 유전 알고리즘 개념 및 절차

**기본 용어**:
- **염색체(Chromosome)**: 해(Solution)의 인코딩 (비트열, 실수 벡터)
- **유전자(Gene)**: 염색체의 개별 요소
- **적합도(Fitness)**: 해의 품질을 나타내는 목적 함수 값
- **개체군(Population)**: 현재 세대의 해 집합

**절차**:

```
1. 초기 개체군 생성 (무작위)
         ↓
2. 적합도 평가 (목적 함수 계산)
         ↓
3. 선택 (Selection) — 우수 개체 선택
         ↓
4. 교차 (Crossover) — 유전자 교환
         ↓
5. 돌연변이 (Mutation) — 일부 유전자 무작위 변경
         ↓
6. 새 개체군 생성
         ↓
종료 조건 만족? → 아니오: 2번으로 / 예: 최적해 반환
```

### 주요 연산 상세

**선택 (Selection)**:
- 룰렛 휠 선택: 적합도 비례 확률
- 토너먼트 선택: 무작위 k개 중 최우수 선택
- 엘리트 선택: 상위 일부 다음 세대로 직접 전달

**교차 (Crossover)**:
- 1점 교차: 한 지점을 기준으로 부모 유전자 교환
- 2점 교차: 두 지점 사이의 구간 교환
- 균일 교차: 각 유전자를 확률적으로 부모 중 하나에서 선택

**돌연변이 (Mutation)**:
- 낮은 확률(1~5%)로 유전자 무작위 변경
- 지역 최적해(Local Optimum) 탈출 수단

### 최적화 방법

| 기법 | 설명 |
|------|------|
| **엘리트주의** | 최우수 개체를 다음 세대에 보존 (수렴 보장) |
| **적응적 연산자** | 세대별로 교차·돌연변이 확률 동적 조정 |
| **다목적 최적화** | NSGA-II 등으로 여러 목적 함수 동시 최적화 |
| **하이브리드** | 그래디언트 기반 지역 탐색과 결합 (Memetic Algorithm) |

### 활용 분야

- 스케줄링 최적화 (공장 생산계획, 시간표)
- 신경망 구조 탐색 (NAS: Neural Architecture Search)
- 라우팅 최적화 (TSP: 외판원 문제)
- 게임 AI, 금융 포트폴리오 최적화

## 출제 포인트

- 돌연변이가 지역 최적해 탈출 수단임을 강조
