## 개요

PET(Privacy Enhancing Technology)는 개인정보를 보호하면서도 데이터 활용성을 극대화하는 기술의 총칭이다. AI 학습, 의료 데이터 공유, 금융 분석 등 데이터 가치가 높은 분야에서 프라이버시-유용성 트레이드오프를 해결하기 위해 주목받고 있다.

## 핵심 내용

### PET 정의 및 등장 배경

- **정의**: 개인정보 수집·사용·공유 시 프라이버시를 기술적으로 보호하면서 데이터 분석·활용을 가능하게 하는 기술 집합
- **배경**:
  - 개인정보보호법 강화 (GDPR, 개인정보보호법 개정)
  - AI 학습을 위한 대규모 개인 데이터 필요성 증가
  - 데이터 유출·오남용 사고 증가
  - 데이터 경제 활성화와 프라이버시 보호의 균형 필요

### PET 주요 기술 분류

| 분류 | 기술 | 설명 |
|------|------|------|
| **암호화 기반** | 동형암호(HE) | 암호화된 채로 연산 |
| **통계적 기반** | 차분 프라이버시(DP) | 통계 결과에 노이즈 추가 |
| **분산 학습** | 연합학습(FL) | 데이터 이동 없이 모델 학습 |
| **익명화** | 가명처리, k-익명성 | 식별 불가능하게 변환 |
| **합성 데이터** | GAN 기반 합성 | 실제와 통계적 특성 유사한 가상 데이터 |
| **보안 다자간 계산** | MPC | 여러 주체가 데이터 비공개로 공동 연산 |

### PET 3가지 핵심 기술 상세 비교

| 구분 | 동형암호 (Homomorphic Encryption) | 차분 프라이버시 (Differential Privacy) | 합성 데이터 (Synthetic Data) |
|------|----------------------------------|----------------------------------------|------------------------------|
| **원리** | 암호화 상태에서 연산 수행 후 복호화 | 통계 결과에 수학적 노이즈(Laplace/Gaussian) 추가 | GAN·VAE로 원본 분포 학습 후 가상 데이터 생성 |
| **데이터 공유** | 불필요 (암호화 상태 유지) | 결과값만 공유 | 합성 데이터 공유 |
| **프라이버시 보장** | 매우 강함 | 수학적으로 증명 가능 | 원본 미노출 |
| **정확도** | 완전 (노이즈 없음) | 노이즈로 인한 오차 발생 | 통계적 유사성 (완전 일치 아님) |
| **성능** | 매우 느림 (수천 배 오버헤드) | 빠름 | 학습 시간 필요 |
| **적용 사례** | 의료 데이터 안전 분석 | Apple iOS 키보드 학습 | 금융 사기 탐지 학습 데이터 |
| **한계** | 계산 비용, 적용 연산 제한 | ε(프라이버시 예산) 설정 어려움 | 데이터 품질 보장 어려움 |

### 연합학습 (Federated Learning)

- 데이터를 중앙 서버로 이동하지 않고, 각 기기에서 모델 학습 후 그래디언트만 공유
- 구글 Gboard 키보드 예측에 최초 적용
- 그래디언트 역추론 공격 위험 → 차분 프라이버시와 결합 사용

## 출제 포인트

- PET 등장 배경 3가지 서술
- 주요 기술 6가지 분류 (암호화·통계·분산·익명화·합성·MPC)
- 동형암호·차분 프라이버시·합성 데이터 3가지 비교표 작성
- 정확도, 성능, 적용 사례 차이 강조
