## 개요

ISMS-P 간편인증은 중소·영세 정보통신서비스 사업자가 기존 ISMS-P 인증의 높은 비용·복잡성 없이 핵심 보안 항목만으로 인증을 취득할 수 있는 간소화 제도다. 개인정보보호위원회·한국인터넷진흥원(KISA)이 2023년 도입을 추진하였다.

## 핵심 내용

### 가. ISMS-P 간편인증 목적

| 목적 | 내용 |
|------|------|
| **인증 사각지대 해소** | 소규모 사업자의 인증 참여 장벽 제거 |
| **보안 수준 향상** | 기본 보안 항목 이행을 통한 최소 보안 수준 확보 |
| **비용·기간 단축** | 기존 인증(6개월~1년) → 간편 심사(2~3개월) |
| **생태계 확장** | 인증 의무 대상 외 자율 참여 확대 |

### 나. ISMS-P 간편인증 대상

- **주요 대상**: 매출액 100억 미만 정보통신서비스 사업자
- **자율 참여**: 의무 인증 대상이 아닌 중소 쇼핑몰, 앱 서비스, 핀테크 스타트업
- **제외 대상**: ISMS-P 의무 인증 대상 (연 매출 100억 이상, 이용자 100만 명 이상 등)

### 다. ISMS-P 간편인증 기준

**기존 ISMS-P(102개) 대비 축소된 핵심 항목으로 구성:**

| 영역 | 주요 항목 |
|------|---------|
| **관리체계 수립** | 정보보호 정책, 조직·책임, 위험 관리 |
| **보호대책** | 접근 통제, 암호화, 취약점 관리, 백업 |
| **개인정보 처리** | 수집·이용 동의, 보관·파기, 개인정보 처리방침 |

**심사 방식**:
- **서면 심사 중심**: 현장 방문 최소화
- **자가진단 도구**: KISA 제공 체크리스트 활용
- **유효 기간**: 1년 (기존 3년 대비 단축, 재심사 용이)

### ISMS-P 간편인증 vs 일반 인증 비교

| 구분 | 일반 ISMS-P | 간편인증 |
|------|-----------|---------|
| **통제항목** | 102개 | 핵심 항목 축소 |
| **심사 기간** | 6개월~1년 | 2~3개월 |
| **비용** | 수천만 원 | 대폭 절감 |
| **현장 심사** | 필수 | 최소화 |
| **유효 기간** | 3년 | 1년 |
| **대상** | 의무/자율 | 소규모 자율 참여 |

## 출제 포인트

- 도입 목적: 소규모 사업자 인증 장벽 제거, 보안 사각지대 해소
- 간편인증 대상: 의무 대상 외 소규모 사업자
- 기준: 102개 → 핵심 항목 축소, 서면 심사 중심
