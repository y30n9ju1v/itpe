## 개요

딥페이크(Deepfake)는 딥러닝(Deep Learning)과 가짜(Fake)의 합성어로, AI를 이용해 실제처럼 보이는 가짜 이미지·영상·음성을 생성하는 기술 및 그 결과물이다. GAN·VAE·확산 모델이 핵심 기술이다.

## 핵심 내용

### 개념 및 핵심 기술

| 기술 | 설명 |
|------|------|
| **GAN(생성적 적대 신경망)** | 생성자-판별자 경쟁으로 실사 수준 이미지 생성 |
| **VAE(변분 오토인코더)** | 잠재 공간 기반 얼굴 특징 인코딩·합성 |
| **확산 모델(Diffusion)** | 노이즈 제거 과정으로 고품질 이미지 생성 |
| **Face Swap** | 영상에서 얼굴 영역 감지·교체 |
| **Voice Cloning** | 소량 음성 샘플로 화자 음성 복제 |
| **립싱크(Lip Sync)** | 오디오와 입 모양 동기화 |

### 딥페이크의 문제점

| 분야 | 문제 |
|------|------|
| **개인 피해** | 비동의 음란물(IBSA), 명예훼손, 스토킹 |
| **정치·사회** | 선거 허위 정보, 정치인 허위 발언 영상 |
| **금융 사기** | CEO 사칭 음성으로 자금 이체 지시 |
| **보안** | 생체 인증(얼굴·음성) 우회 |
| **법적 증거** | 영상 증거의 신뢰성 훼손 |

### 딥페이크 대응방안

**기술적 대응**:

| 방안 | 설명 |
|------|------|
| **딥페이크 탐지 AI** | GAN 생성 흔적(아티팩트) 탐지, 눈깜빡임·맥박 분석 |
| **디지털 워터마킹** | 원본 콘텐츠에 불가시 워터마크 삽입 |
| **콘텐츠 인증(C2PA)** | 생성 출처·이력을 메타데이터로 기록·검증 |
| **생체 인증 강화** | 3D 얼굴 인식, 라이브니스 검사(Liveness Detection) |

**제도적 대응**:
- 딥페이크 성범죄물 처벌: 성폭력처벌법 개정 (2020)
- 선거 딥페이크 규제: 공직선거법 개정
- AI 생성 콘텐츠 표시 의무화 (EU AI Act)

## 출제 포인트

- 핵심 기술: GAN·VAE·확산 모델·Face Swap·Voice Cloning
- 문제점: 개인 피해·정치적 악용·생체인증 우회 각각 사례
- C2PA(Content Provenance and Authenticity): 콘텐츠 출처 검증 표준
