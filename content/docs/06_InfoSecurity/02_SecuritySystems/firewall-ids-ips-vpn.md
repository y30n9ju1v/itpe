## 개요

방화벽(Firewall), IDS(침입탐지시스템), IPS(침입방지시스템), VPN(가상사설망)은 네트워크 보안의 4대 핵심 장비다. 각각 차단·탐지·방지·암호화 역할을 담당하며 계층적으로 운용된다.

## 핵심 내용

### 4대 보안 장비 비교

| 구분 | 방화벽 (Firewall) | IDS | IPS | VPN |
|------|-----------------|-----|-----|-----|
| **역할** | 패킷 차단/허용 | 침입 탐지·경보 | 침입 탐지+자동 차단 | 암호화 터널 통신 |
| **동작 방식** | 규칙 기반 패킷 필터링 | 트래픽 분석, 알림 | IDS + 인라인 차단 | 터널링+암호화 |
| **위치** | 네트워크 경계 | 네트워크 경유 지점 | 인라인(트래픽 통과) | 클라이언트-서버 간 |
| **응답** | 수동(규칙 정의) | 알림(수동 대응) | 자동 차단 | 해당 없음 |
| **오탐 영향** | 낮음 | 알림 과부하 | 정상 트래픽 차단 위험 | 해당 없음 |

### 방화벽 유형

| 유형 | 설명 |
|------|------|
| 패킷 필터링 | IP/포트 기반 허용·차단 (3계층) |
| 상태 기반(Stateful) | 세션 상태 추적 (4계층) |
| 애플리케이션 방화벽(WAF) | HTTP 페이로드 분석 (7계층) |
| 차세대 방화벽(NGFW) | IPS+앱 식별+SSL 검사 통합 |

### IDS 탐지 방식

| 방식 | 설명 | 장단점 |
|------|------|--------|
| **서명 기반** | 알려진 공격 패턴(시그니처) 매칭 | 정확, 신규 공격 미탐 |
| **이상 기반** | 정상 행동 프로파일 대비 이상 탐지 | 신규 공격 탐지, 오탐 높음 |

### VPN 주요 프로토콜

- **IPsec VPN**: 네트워크 계층 암호화, 사이트 간 연결
- **SSL/TLS VPN**: 브라우저 기반, 원격 접속
- **WireGuard**: 최신 경량 프로토콜, 고성능

### 구축 예시 (다층 방어)

```
인터넷 → 방화벽(경계) → DMZ[IDS/IPS] → 내부망 → 호스트 기반 방화벽
                ↕
            VPN 터널 (재택 근무자)
```

## 출제 포인트

- 4대 장비의 역할 차이: 차단(방화벽)·탐지(IDS)·방지(IPS)·암호화(VPN)
- IDS vs IPS 차이: 탐지 후 알림 vs 자동 차단, 오탐 영향
- 다층 방어(Defense in Depth) 적용
