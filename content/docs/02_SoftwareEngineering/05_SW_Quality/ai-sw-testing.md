## 개요


AI 소프트웨어는 결정론적 동작이 아닌 확률적·데이터 의존적 특성으로 인해 기존 테스트 기법의 적용이 어렵다. 메타모픽 테스트·뉴런 커버리지 테스트·안전 반경 최대화 테스트는 AI 특화 품질 보증 기법이다.

## 핵심 내용

### 메타모픽 테스트 (Metamorphic Test)

**개념**: 테스트 오라클(정답)이 없을 때, 입력 간 관계(메타모픽 관계)를 이용해 출력 일관성을 검증하는 기법

**원리**:
- 기준 테스트 입력 x → 출력 f(x)
- 변환 입력 x' (x의 변형) → 출력 f(x')
- f(x)와 f(x') 간 예상 관계 성립 여부 검증

**예시 (이미지 분류기)**:
- 메타모픽 관계: "이미지를 좌우 반전해도 분류 결과는 동일해야 함"
- 원본: 고양이 → 고양이 (90%)
- 반전: 반전 고양이 → 고양이 (88%) → 관계 성립

**활용**: 자율주행(동일 도로 다른 날씨), 의료 진단(이미지 회전)

### 뉴런 커버리지 및 딥러닝 테스트 커버리지 기준 (DeepGauge 프레임워크 핵심)

**개념**: 딥러닝 모델에서 테스트 케이스가 활성화하는 뉴런의 상태 및 출력 범위를 다각도로 측정하여 모델의 품질 수준을 보증하는 화이트박스 테스트 기법.

**DeepGauge 프레임워크의 4대 핵심 커버리지 지표 (고득점 필수 키워드!)**:

| 커버리지 유형 | 약어 | 상세 설명 |
|---------------|------|-----------|
| **k-Multisection Neuron Coverage** | **KMNC** | 각 뉴런의 활성화 값 범위를 $k$개의 균등한 구간으로 분할하고, 테스트 케이스가 이 중 몇 개의 구간을 활성화했는지 비율 측정. 뉴런 출력 분포의 세밀한 탐색 보장. |
| **Neuron Boundary Coverage** | **NBC** | 사전 훈련 데이터셋에서 나타난 뉴런의 활성화 값 범위를 벗어나는 **극단적인 예외 조건(상한/하한 외부 영역)**을 얼마나 자극하는지 측정. Corner-case 탐지용. |
| **Strong Neuron Activation Coverage** | **SNAC** | 훈련 과정의 상한 활성화 값보다 훨씬 강하게 자극되는 **초과 활성화 상태**를 뉴런이 얼마나 경험했는지 측정. 과도 자극 노이즈 저항성 평가. |
| **Top-k Neuron Coverage** | **TKNC** | 각 계층(Layer) 내에서 가장 높은 활성화 수준을 보인 **상위 $k$개의 지배적인 뉴런**들을 얼마나 활성화했는지 비율 측정. 핵심 패턴 검증. |

*   **기본 뉴런 커버리지 (Neuron Coverage - DeepXplore)**: 단순히 $\text{활성화된 뉴런 수} / \text{전체 뉴런 수}$의 비율을 뜻하며, 품질과의 엄밀한 상관성이 부족하여 위 DeepGauge 지표와 병행 사용함.

**한계**: 높은 커버리지가 모델의 100% 결함 없음을 보장하지 못함 ➡️ 메타모픽 및 안전 검증 기법과 결합 필수.

### 안전 반경 최대화 테스트 및 형식 검증 (Safety & Robustness)

**개념**: 입력 데이터 주변의 미세한 노이즈(ε)에도 모델 출력이 오염되지 않는 최대 반경인 '안전 반경(Safety Radius)'을 보증하여 모델의 적대적 노이즈 강건성(Robustness)을 검증함.

**검증 및 보증 기법 2대 축**:
1. **안전 반경 최대화 테스트**:
   * **원리**: 적대적 예제(Adversarial Example) 생성 기법(예: FGSM, PGD 공격)을 인위적으로 수행하여 출력이 반전되는 최소 임계 거리($\epsilon$)를 구하고 이를 최대화함.
2. **형식 검증 (Formal Verification - 고득점 핵심 키워드)**:
   * **원리**: 수치적 테스트에 의존하지 않고, 입력 영역의 섭동 범위($||x - x'|| \le \epsilon$)에 대해 출력 오차가 절대 발생하지 않음을 **수학적으로 증명**하는 기법.
   * **검증 도구**: **SMT Solver** 기반 기법 (Reluplex, Marabou, DeepG, ERAN 등 사용). 비선형 Activation 함수(ReLU 등)의 제약조건을 논리식으로 풀어내어 반례(Counterexample)가 존재하지 않음을 수학적으로 입증함.

### 비교

| 기법 | 목적 | 특징 |
|------|------|------|
| 메타모픽 테스트 | 오라클 없는 검증 | 관계 기반, 블랙박스 |
| 뉴런 커버리지 | 모델 내부 탐색 | 화이트박스, 화이트박스 |
| 안전 반경 최대화 | 견고성 측정 | 적대적 공격 저항성 |

## 출제 포인트

- **메타모픽 테스트**: 테스트 오라클 문제의 해결 원리 및 메타모픽 관계(MR) 설계 기법
- **딥러닝 커버리지 (DeepGauge)**: **KMNC, NBC, SNAC, TKNC**의 구체적 정의 및 기존 뉴런 커버리지와의 차이
- **안전 반경 최대화와 적대적 견고성**: FGSM, PGD 공격 저항력 평가
- **형식 검증 (Formal Verification)**: **SMT Solver(Reluplex, Marabou 등)**를 통한 수학적 안전성/강건성 증명 체계
