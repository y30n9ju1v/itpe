## 개요

AI 소프트웨어는 결정론적 동작이 아닌 확률적·데이터 의존적 특성으로 인해 기존 테스트 기법의 적용이 어렵다. 메타모픽 테스트·뉴런 커버리지 테스트·안전 반경 최대화 테스트는 AI 특화 품질 보증 기법이다.

## 핵심 내용

### 메타모픽 테스트 (Metamorphic Test)

**개념**: 테스트 오라클(정답)이 없을 때, 입력 간 관계(메타모픽 관계)를 이용해 출력 일관성을 검증하는 기법

**원리**:
- 기준 테스트 입력 x → 출력 f(x)
- 변환 입력 x' (x의 변형) → 출력 f(x')
- f(x)와 f(x') 간 예상 관계 성립 여부 검증

**예시 (이미지 분류기)**:
- 메타모픽 관계: "이미지를 좌우 반전해도 분류 결과는 동일해야 함"
- 원본: 고양이 → 고양이 (90%)
- 반전: 반전 고양이 → 고양이 (88%) → 관계 성립

**활용**: 자율주행(동일 도로 다른 날씨), 의료 진단(이미지 회전)

### 뉴런 커버리지 테스트 (Neuron Coverage Test)

**개념**: 딥러닝 모델에서 테스트 케이스가 활성화하는 뉴런 비율을 커버리지 지표로 사용

**주요 커버리지 기준**:

| 기준 | 설명 |
|------|------|
| **뉴런 커버리지** | 활성화된 뉴런 수 / 전체 뉴런 수 |
| **강한 뉴런 커버리지** | 상위 k% 활성화 뉴런 커버 |
| **계층 커버리지** | 각 계층별 활성화 패턴 다양성 |

**도구**: DeepXplore, Keras 기반 커버리지 측정

**한계**: 높은 커버리지가 반드시 높은 품질을 보장하지 않음 → 다른 기법과 병행

### 안전 반경 최대화 테스트 (Safety Radius Maximization Test)

**개념**: 입력 데이터 주변의 '안전 반경(Safety Radius)'을 측정하여 모델의 견고성(Robustness) 검증

- **안전 반경**: 입력 x 주변에서 모델 출력이 변하지 않는 최대 입력 변화량(ε)
- 안전 반경이 클수록 적대적 공격·노이즈에 강건

**검증 방법**:
- 적대적 예제(Adversarial Example) 생성 후 분류 변화 측정
- FGSM(Fast Gradient Sign Method)으로 최소 교란 계산

**목표**: 안전 반경을 최대화하는 방향으로 모델 학습·검증

### 비교

| 기법 | 목적 | 특징 |
|------|------|------|
| 메타모픽 테스트 | 오라클 없는 검증 | 관계 기반, 블랙박스 |
| 뉴런 커버리지 | 모델 내부 탐색 | 화이트박스, 화이트박스 |
| 안전 반경 최대화 | 견고성 측정 | 적대적 공격 저항성 |

## 출제 포인트

- 메타모픽 테스트: 오라클 문제 해결, 메타모픽 관계 정의
- 뉴런 커버리지: 기존 코드 커버리지의 AI 확장
- 안전 반경: 적대적 공격 저항성 측정 지표
