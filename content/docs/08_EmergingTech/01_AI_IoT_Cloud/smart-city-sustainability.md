## 개요

지속 가능한 스마트시티는 ICT(IoT·AI·빅데이터·클라우드)를 도시 인프라에 통합하여 인구 고령화, 기후변화, 에너지 문제 등 도시 문제를 해결하고 경제·환경·사회적 지속가능성을 달성하는 도시 모델이다.

## 핵심 내용

### 스마트시티 추진 배경

| 문제 | 스마트시티 해결 방안 |
|------|---------------------|
| 인구 고령화 | 스마트 헬스케어, 원격 의료, 실버 케어 플랫폼 |
| 기후변화·탄소중립 | 스마트 에너지 그리드, RE100, 탄소 모니터링 |
| 교통 혼잡 | C-ITS, 자율주행, 실시간 교통 최적화 |
| 자원 낭비 | 스마트 수자원·폐기물 관리 |

### 스마트시티 핵심 구성요소

| 영역 | 구성 요소 |
|------|-----------|
| **데이터 플랫폼** | 도시 데이터 허브, 디지털 트윈 |
| **IoT 인프라** | 센서 네트워크, LPWAN(LoRa, NB-IoT) |
| **AI·분석** | 실시간 데이터 분석, 예측 모델 |
| **거버넌스** | 데이터 공유 협약, 개인정보보호 |
| **시민 서비스** | 전자정부, 모바일 시민 앱 |

### 지속 가능성 3대 축 (ESG)

| 축 | 내용 |
|----|------|
| **환경(E)** | 탄소 배출 모니터링, 신재생에너지, 녹색 건물 |
| **사회(S)** | 디지털 포용, 취약계층 서비스, 시민 참여 |
| **거버넌스(G)** | 데이터 투명성, 스마트시티 거버넌스 위원회 |

### 국내 스마트시티 정책

- **세종 5-1생활권**: 국가 시범도시
- **부산 에코델타시티**: 물·에너지·생활 혁신
- **스마트시티 통합플랫폼**: 행안부 주도 공통 플랫폼 구축

## 출제 포인트

- 스마트시티 추진 배경: 인구 구조 변화 + 환경 문제
- ESG 관점의 지속 가능성 3대 축
- 핵심 기술: 디지털 트윈, IoT, AI, 데이터 거버넌스
- 국내 사례: 세종 5-1, 부산 에코델타시티
