## 개요

VAE(Variational AutoEncoder)는 입력 데이터를 잠재 공간(Latent Space)의 확률 분포로 인코딩하고, 그 분포에서 샘플링하여 새로운 데이터를 생성하는 생성 모델이다. 기존 AutoEncoder에 확률적 샘플링을 도입하여 생성 능력을 부여한다.

## 핵심 내용

### AutoEncoder vs VAE

| 구분 | AutoEncoder | VAE |
|------|------------|-----|
| 잠재 공간 | 고정된 벡터 | 확률 분포 (μ, σ) |
| 목적 | 압축·복원 | 생성·잠재 표현 학습 |
| 새 데이터 생성 | 불가 | 가능 (샘플링) |
| 잠재 공간 연속성 | 불연속 | 연속 (보간 가능) |

### VAE 구조

```
입력 X
   ↓ 인코더
잠재 분포 파라미터: μ (평균), σ (표준편차)
   ↓ 재매개변수화 트릭 (z = μ + ε·σ, ε~N(0,1))
잠재 벡터 z (샘플링)
   ↓ 디코더
재구성 X'
```

### 손실 함수

```
L = 재구성 손실 + KL 발산(KL Divergence)
```

- **재구성 손실**: 원본 X와 복원 X' 간 차이 (MSE 또는 BCE)
- **KL 발산**: 잠재 분포 q(z|x)와 사전 분포 N(0,I) 간 거리 → 잠재 공간 정규화

### 재매개변수화 트릭 (Reparameterization Trick)

샘플링 연산은 역전파 불가 → z = μ + ε·σ 형태로 분리하여 μ, σ에 대한 그래디언트 계산 가능하게 만드는 핵심 기법

### GAN과의 비교

| 구분 | VAE | GAN |
|------|-----|-----|
| 학습 방식 | 변분 추론 | 적대적 학습 |
| 생성 품질 | 다소 흐림 | 선명 |
| 잠재 공간 | 구조적, 연속 | 비구조적 |
| 학습 안정성 | 안정적 | 불안정(모드 붕괴) |

## 출제 포인트

- 재매개변수화 트릭: 샘플링의 역전파 문제 해결 방법
- 손실 = 재구성 손실 + KL 발산
- VAE vs GAN: 잠재 공간 연속성·생성 품질 비교
