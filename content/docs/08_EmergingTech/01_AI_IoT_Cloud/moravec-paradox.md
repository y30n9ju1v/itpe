## 개요

모라벡의 역설(Moravec's Paradox)은 인간이 쉽다고 느끼는 감각·운동 능력(걷기, 물체 잡기, 얼굴 인식)은 컴퓨터가 모방하기 매우 어렵고, 반대로 인간이 어렵다고 느끼는 논리·추론·체스 같은 고차원 인지 작업은 컴퓨터가 훨씬 잘한다는 역설이다. 1988년 한스 모라벡이 주장했다.

## 핵심 내용

### 역설의 핵심

- **인간에게 쉬운 것** → AI에게 어려움: 걷기, 물체 잡기, 표정 인식, 자연어 이해
- **인간에게 어려운 것** → AI에게 쉬움: 체스, 수학 계산, 논리 추론, 데이터 검색

### 역설의 원인

| 이유 | 설명 |
|------|------|
| **진화적 기원** | 감각·운동 능력은 수억 년 진화를 통해 최적화된 복잡한 신경 회로의 산물 |
| **암묵지 (Tacit Knowledge)** | 인간이 의식하지 못하는 방대한 경험과 문맥 지식 (명시화 어려움) |
| **높은 차원의 복잡성** | 물리적 세계 인식은 수백만 개의 암묵적 규칙 포함 |
| **명시적 논리** | 체스·수학은 규칙이 명확히 정의되어 있어 알고리즘으로 표현 가능 |

### AI 기술과의 연계

- **딥러닝의 발전**: 이미지 인식·음성 인식에서 인간 수준 달성 → 역설의 일부 해소
- **로봇공학의 한계**: 보스턴 다이나믹스의 고난도 운동 로봇도 여전히 비용 높고 범용성 부족
- **LLM의 역설**: GPT-4는 변호사 시험 합격, 단순 물리적 판단(컵이 기울면 물이 쏟아짐)은 여전히 오류

### 활용 시사점

1. **AI 설계 전략**: 고차원 인지는 AI에 위임, 물리적 상호작용은 인간-AI 협력
2. **로봇 공학 투자**: 임베디드 센서·액추에이터 고도화 필요
3. **인간-AI 협력 모델**: 인간의 신체적 능력 + AI의 계산 능력 결합

## 출제 포인트

- 역설의 정의: 인간에게 쉬운 것이 AI에게 어렵고, 어려운 것은 쉬움
- 원인 2가지: 진화적 최적화 + 암묵지
- LLM/현대 AI 기술과 연결하여 역설의 현재 상황 서술
- 활용 시사점으로 인간-AI 협력 모델 언급
