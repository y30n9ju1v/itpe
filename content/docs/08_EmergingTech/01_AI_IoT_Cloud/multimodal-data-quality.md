## 개요

「생성형 인공지능 학습을 위한 멀티모달 데이터의 품질검증 방법(TTAK.KO-10.1558)」은 텍스트·이미지·음성·영상 등 여러 유형의 데이터를 학습에 활용하는 생성형 AI 모델의 학습 데이터 품질을 검증하는 TTA 정보통신단체표준이다.

## 핵심 내용

### 멀티모달 데이터 품질특성 (6개 특성)

| 품질특성 | 정의 |
|----------|------|
| **정확성** | 데이터가 실제 현상을 정확하게 표현하는 정도 |
| **완전성** | 학습에 필요한 필수 데이터 항목이 모두 존재하는 정도 |
| **일관성** | 동일 의미의 멀티모달 데이터 간 상호 모순 없는 정도 |
| **다양성** | 모델 일반화를 위한 데이터 분포의 균형·다양성 |
| **적합성** | 학습 목적에 맞는 데이터 내용·형식의 적절성 |
| **윤리성** | 편향·혐오·저작권 위반 등 AI 윤리 기준 준수 |

### 데이터 유형별 유효성 검증 방법

#### 텍스트 데이터

| 검증 항목 | 방법 |
|-----------|------|
| 언어 유효성 | 언어 감지 도구로 목표 언어 확인 |
| 품질 필터링 | Perplexity 점수로 저품질 텍스트 제거 |
| 중복 제거 | MinHash, SimHash 유사도 기반 중복 탐지 |
| 편향 검출 | 감성 분석, 공정성 지표 측정 |

#### 이미지 데이터

| 검증 항목 | 방법 |
|-----------|------|
| 해상도·형식 | 최소 해상도·파일 포맷 규격 확인 |
| 캡션 일치성 | CLIP Score: 이미지-텍스트 의미 유사도 측정 |
| 품질 지표 | BRISQUE, NIQE (무참조 이미지 품질 지수) |
| 저작권 | 라이선스 정보 메타데이터 확인 |

#### 음성/오디오 데이터

| 검증 항목 | 방법 |
|-----------|------|
| 음질 | SNR(신호 대 잡음비) 측정 |
| 전사 정확도 | WER(Word Error Rate) 측정 |
| 형식 유효성 | 샘플링 레이트, 비트레이트 규격 확인 |

#### 영상(Video) 데이터

| 검증 항목 | 방법 |
|-----------|------|
| 프레임 품질 | 해상도·FPS 규격 확인 |
| 시간적 일관성 | 프레임 간 연속성 분석 |
| 자막 동기화 | 오디오-텍스트 타임스탬프 정합성 확인 |

## 출제 포인트

- 6개 품질특성: 정확성·완전성·일관성·다양성·적합성·윤리성
- 유형별 검증: 텍스트(Perplexity), 이미지(CLIP Score), 음성(WER)
- 윤리성: 편향·저작권 이슈가 생성 AI 학습 데이터의 핵심 쟁점
