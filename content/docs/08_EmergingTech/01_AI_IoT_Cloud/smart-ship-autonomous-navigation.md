## 개요

스마트선박(Smart Ship)은 IT·AI·IoT 기술을 선박에 융합하여 운항 효율·안전성·환경 친화성을 높인 선박이다. 자율운항선박(MASS: Maritime Autonomous Surface Ship)은 IMO가 정의한 자율화 등급에 따라 최소 선원 또는 무인으로 운항 가능한 선박이다.

## 핵심 내용

### IMO 자율운항선박 자율화 등급 (MASS 코드)

| 등급 | 명칭 | 설명 |
|------|------|------|
| **Level 1** | 자동화 프로세스 보조 | 자율 시스템이 의사결정 지원, 선원이 최종 결정 |
| **Level 2** | 원격 제어 (선원 승선) | 원격 제어 가능, 선원이 선내 대기 |
| **Level 3** | 원격 제어 (선원 미승선) | 육상 원격 운항센터에서 제어, 무인 운항 |
| **Level 4** | 완전 자율운항 | 시스템이 모든 결정, 비상 시에도 자율 대응 |

### SAN (Ship Area Network, 선박 통합 네트워크) 모듈 구조

SAN은 선박 내 모든 시스템을 통합 연결하는 선내 네트워크 인프라:

```
+------------------+
|   선박 통합 네트워크 (SAN)     |
|                              |
| ┌────────────┐  ┌──────────┐ |
| │항법 시스템  │  │기관 시스템│ |
| │(INS)      │  │(EMS)     │ |
| └────────────┘  └──────────┘ |
| ┌────────────┐  ┌──────────┐ |
| │화물 관리   │  │안전/보안  │ |
| │시스템      │  │시스템     │ |
| └────────────┘  └──────────┘ |
+------------------+
         ↓
   위성통신 (VSAT) / 육상 원격운항센터
```

**SAN 주요 구성요소:**
- **항법 네트워크**: GPS, AIS, RADAR, 전자해도(ECDIS)
- **기관 네트워크**: 엔진 모니터링, 연료 관리
- **화물 네트워크**: 컨테이너 위치, 냉동 화물 온도
- **안전 네트워크**: CCTV, 화재감지, 조난신호(EPIRB)
- **통신 네트워크**: VSAT 위성, 4G/5G, V2X

### INS (Integrated Navigation System, 통합항해시스템) 모듈 구조

INS는 다양한 항법 장비를 통합하여 선교(Bridge)에서 일원화된 항법 정보를 제공:

| 모듈 | 설명 |
|------|------|
| **센서 통합** | GPS, AIS, 자이로컴패스, 풍속/풍향계 데이터 통합 |
| **ECDIS** | 전자해도정보시스템 (종이 해도 대체) |
| **ARPA/RADAR** | 목표물 자동 추적 레이더 |
| **경보 관리** | 항해 경보 우선순위 통합 표시 |
| **자동 조타** | 경로 계획·유지 자동 제어 |
| **출력/표시** | 통합 항해 정보 단일 화면 표시 |

### 스마트선박 보안 (Cyber Security)

- IMO MSC-FAL.1/Circ.3: 선박 사이버 위험 관리 지침
- IT 네트워크(업무)와 OT 네트워크(운항) 분리
- 외부 연결 포트(USB, 위성통신) 접근 제어

## 출제 포인트

- IMO MASS 자율화 4단계 레벨과 각 특징
- SAN: 선박 내 모든 시스템 통합 네트워크
- INS: 항법 장비 통합, ECDIS·RADAR·자동조타 포함
- 사이버 보안: IT/OT 네트워크 분리
