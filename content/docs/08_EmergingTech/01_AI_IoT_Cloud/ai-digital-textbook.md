## 개요

AI 디지털교과서는 인공지능 기술을 활용하여 학생 개인의 학습 수준·속도·방식에 맞춰 최적화된 학습 경험을 제공하는 지능형 교육 플랫폼이다. 2025년부터 국내 초·중·고에 단계적으로 도입된다.

## 핵심 내용

### 개념 및 특징

- **기존 디지털교과서 vs AI 디지털교과서**
  - 기존: 콘텐츠 디지털화 수준 (PDF·동영상 제공)
  - AI: 학생별 학습 데이터 분석 → 맞춤형 콘텐츠 동적 제공

- **핵심 특징**:

| 특징 | 내용 |
|------|------|
| 개인화 학습 | 학습 이력 분석으로 난이도·경로 자동 조정 |
| 실시간 피드백 | 오답 패턴 분석, 즉각적 보완 학습 제공 |
| 교사 지원 | 학습 대시보드로 취약 학생 조기 발견 |
| 멀티미디어 통합 | AR/VR, 시뮬레이션, 인터랙티브 실험 |

### 플랫폼 구조

```
학습자 계층 (학생 디바이스)
    ↓↑
서비스 계층
├── 학습 콘텐츠 서비스
├── AI 추천 엔진 (학습 경로 개인화)
├── 평가 서비스
└── 협업·상호작용 서비스
    ↓↑
데이터 계층
├── 학습 데이터 저장소 (xAPI/LRS)
├── AI/ML 파이프라인
└── 콘텐츠 저장소
    ↓↑
인프라 계층 (클라우드/온프레미스)
```

### 기능 및 핵심 서비스

| 기능 | 서비스 내용 |
|------|-----------|
| **지능형 튜터링** | AI 기반 1:1 맞춤 설명, 질의응답 |
| **학습 경로 추천** | 학생별 최적 학습 시퀀스 자동 생성 |
| **자동 채점·피드백** | 서술형 포함 자동 채점, 오류 분석 |
| **학습 분석 대시보드** | 교사: 학급 전체·개별 학습 현황 시각화 |
| **디지털 실험실** | 과학·수학 시뮬레이션 인터랙티브 실험 |

### 도입 과제

- 학습 데이터 보안·개인정보보호 (학생 데이터의 민감성)
- 교사 디지털 역량 강화 필요
- 디지털 기기 접근성 불평등 해소
- AI 추천의 편향·알고리즘 투명성

## 출제 포인트

- AI 디지털교과서의 핵심: 개인화 학습 + 실시간 피드백
- 플랫폼 3계층 구조: 학습자-서비스-데이터-인프라
- xAPI/LRS(Learning Record Store)와 학습 데이터 표준
