## 개요

지식재산권(Intellectual Property Rights)은 인간의 지적 창작물에 부여되는 법적 권리로, 산업재산권·저작권·신지식재산권의 3대 분류로 구성된다. IT 분야에서는 소프트웨어, 데이터베이스, 반도체 배치설계 등이 주요 보호 대상이다.

## 핵심 내용

### 지식재산권 분류 체계

| 분류 | 종류 | 보호 대상 | 존속 기간 |
|------|------|-----------|-----------|
| **산업재산권** | 특허권 | 기술적 발명 | 출원 후 20년 |
| | 실용신안권 | 물품의 형상·구조 | 출원 후 10년 |
| | 디자인권 | 물품의 외관·심미적 창작 | 등록 후 20년 |
| | 상표권 | 상품·서비스 식별 표지 | 등록 후 10년(갱신 가능) |
| **저작권** | 저작권 | 문학·음악·SW·DB 등 창작물 | 사후 70년 |
| | 저작인접권 | 실연자·음반제작자·방송사업자 | 공표 후 70년 |
| **신지식재산권** | 반도체 배치설계권 | 반도체 집적회로 배치설계 | 등록 후 10년 |
| | 영업비밀 | 기술·경영 상의 비공개 정보 | 비밀 유지 기간 |
| | 데이터베이스권 | DB 제작에 투자된 성과 | 제작 완료 후 5년 |
| | 도메인이름 | 인터넷 주소 | - |

### SW 지식재산권 특성

**소프트웨어 저작권**:
- 소스코드·목적코드 모두 보호 (등록 불필요, 창작과 동시 발생)
- 프로그램 기능·로직 자체는 보호 안 됨 (아이디어는 특허로 보호)

**SW 특허**: 컴퓨터로 구현된 발명(비즈니스 방법 특허 포함)

**오픈소스 라이선스**: 저작권 포기가 아니라 이용 허락 조건을 명시한 것
- GPL: 파생물도 동일 라이선스 (강한 카피레프트)
- MIT/Apache: 소스 공개 의무 없음 (약한 카피레프트)

## 출제 포인트

- 산업재산권·저작권·신지식재산권 3분류와 각 종류 나열
- SW는 저작권(소스코드)과 특허권(발명) 이중 보호 가능
- 데이터베이스권, 반도체 배치설계권이 신지식재산권에 해당
- 오픈소스 라이선스는 저작권 위에 구축된 이용 허락 체계
