## 개요

디지털서비스 전문계약제도는 클라우드·SaaS 등 디지털서비스의 특성(구독형, 표준약관)에 맞게 기존 SI 중심 공공 조달 방식을 개편한 제도다. 2021년 전자정부법 개정으로 도입되었으며, 민간 클라우드 서비스를 공공기관이 신속·유연하게 구매할 수 있도록 한다.

## 핵심 내용

### 주요 특징

| 특징 | 내용 |
|------|------|
| **이용약관 기반 계약** | 맞춤 개발 없이 표준약관으로 계약 (기존 SI 계약 방식 탈피) |
| **구독형 서비스 허용** | 월정액·종량제 등 구독 과금 모델 공공 조달 적용 |
| **간소화된 입찰** | 기술평가 중심, 복잡한 제안서 불필요 |
| **신속 도입** | 디지털서비스 이용지원시스템(DSIS)을 통한 직접 구매 |
| **수시 업데이트** | 계약 기간 중 서비스 기능 개선 허용 |

### 디지털서비스 종류

1. **클라우드 컴퓨팅 서비스**: IaaS, PaaS, SaaS
2. **인공지능 서비스**: AI API, 머신러닝 플랫폼
3. **블록체인 서비스**: 분산원장 기반 서비스
4. **데이터 분석 서비스**: 빅데이터 분석 플랫폼
5. **기타 디지털 전환 서비스**: IoT 플랫폼, 협업 도구 등

### 기존 계약 vs 디지털서비스 계약 비교

| 구분 | 기존 SI 계약 | 디지털서비스 전문계약 |
|------|------------|-------------------|
| **계약 방식** | 맞춤 개발 계약 | 이용약관 기반 |
| **과금** | 프로젝트 완료 후 일시 지급 | 구독·종량제 |
| **납품물** | 소스코드·문서 | 서비스 이용권 |
| **업데이트** | 별도 유지보수 계약 | 서비스 내 자동 갱신 |
| **소요 시간** | 수개월 | 수일~수주 |

### 기대 효과

- 혁신 민간 SaaS·AI 서비스의 공공 신속 도입
- 디지털 전환 가속화
- 스타트업·중소 SaaS 기업의 공공 시장 진입 확대
