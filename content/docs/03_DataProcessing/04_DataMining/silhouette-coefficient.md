## 개요

실루엣 계수(Silhouette Coefficient)는 클러스터링 결과의 품질을 평가하는 지표로, 각 데이터 포인트가 자신의 클러스터에 얼마나 잘 속하고(응집도), 이웃 클러스터와 얼마나 잘 분리되어 있는지(분리도)를 측정한다.

## 핵심 내용

### 계산 공식

데이터 포인트 i에 대해:

- **a(i)**: i와 같은 클러스터 내 다른 점들과의 평균 거리 (응집도, 낮을수록 좋음)
- **b(i)**: i와 가장 가까운 이웃 클러스터 점들과의 평균 거리 (분리도, 높을수록 좋음)

```
s(i) = (b(i) - a(i)) / max(a(i), b(i))
```

### 해석

| 값 범위 | 의미 |
|---------|------|
| 0.7 ~ 1.0 | 매우 좋은 클러스터링 |
| 0.5 ~ 0.7 | 적절한 클러스터링 |
| 0.25 ~ 0.5 | 약한 클러스터링 |
| 0 이하 | 잘못된 클러스터 할당 |

- **전체 실루엣 계수**: 모든 데이터 포인트 s(i)의 평균
- 범위: -1 ≤ s(i) ≤ 1

### 활용: 최적 K 결정

K-Means에서 K 값별로 실루엣 계수를 계산하여 최댓값을 갖는 K를 최적 군집 수로 선택.

### 엘보우법(Elbow Method)과 비교

| 구분 | 실루엣 계수 | 엘보우법 |
|------|------------|---------|
| 기준 | 응집도+분리도 | 군집 내 분산(WCSS) |
| 해석 용이성 | 수치 명확 | 꺾임점 주관적 판단 필요 |
| 계산 비용 | 높음 | 낮음 |

## 출제 포인트

- a(i), b(i) 정의와 공식 s(i) = (b-a)/max(a,b)
- 최적 K 결정 수단으로서의 활용
- 값이 0 이하인 경우의 의미
