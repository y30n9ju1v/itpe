## 개요

<!--more-->

데이터 어노테이션(Data Annotation)은 AI/ML 모델 학습을 위해 원시 데이터에 레이블·메타데이터·주석을 부착하는 작업이다. 이미지 어노테이션은 컴퓨터 비전 모델(분류·탐지·분할)의 품질을 결정하는 핵심 전처리 과정이다.

## 핵심 내용

### 이미지 어노테이션 유형

| 유형 | 설명 | 대표 포맷 | 활용 태스크 |
|------|------|-----------|-------------|
| **이미지 분류** | 이미지 전체에 단일 레이블 부착 | JPEG+CSV | 개/고양이 분류 |
| **바운딩 박스** | 객체 주위 직사각형 박스 표시 | COCO JSON, PASCAL VOC | 객체 탐지(YOLO) |
| **폴리곤** | 객체 외곽선을 다각형으로 표시 | GeoJSON | 자율주행 도로 구분 |
| **시맨틱 분할** | 픽셀별 클래스 레이블 할당 | PNG 마스크 | 의료 영상 분석 |
| **인스턴스 분할** | 같은 클래스 내 개별 객체 구분 | COCO Mask | 군중 분석 |
| **키포인트** | 신체 관절 등 특정 점 표시 | COCO Keypoints | 자세 추정 |
| **3D 어노테이션** | LiDAR 포인트클라우드에 3D 박스 | PCD, Velodyne | 자율주행 3D 탐지 |

### 어노테이션 기법

**도구 기반 기법**
- **수동 라벨링**: 전문 어노테이터가 직접 표시 (고품질, 고비용)
- **반자동 어노테이션**: AI 모델이 초안 생성 후 사람이 수정 (효율 향상)
- **크라우드소싱**: Amazon MTurk 등 다수 작업자 활용 (저비용, 품질 편차 있음)

**품질 보증 기법**
- **Inter-Annotator Agreement (IAA)**: 복수 어노테이터 결과 일치도 측정 (Cohen's Kappa)
- **골든 셋(Golden Set)**: 정답 레이블 포함 검증 샘플로 어노테이터 정확도 평가
- **액티브 러닝**: 모델이 불확실한 샘플만 선별하여 어노테이션 비용 절감

### 어노테이션 품질 이슈

- **레이블 노이즈**: 오류 레이블이 모델 성능 저하 → 노이즈 라벨 학습(Label Noise Learning) 기법 적용
- **클래스 불균형**: 희귀 객체 데이터 부족 → 오버샘플링, 합성 데이터(GAN)로 보완
- **일관성 문제**: 어노테이터 간 기준 차이 → 상세한 어노테이션 가이드라인 필수

## 실무 적용 예시

- 자율주행: 카메라+LiDAR 융합 어노테이션으로 보행자·차량·도로 인식 모델 학습
- 의료 AI: 방사선 전문의가 CT 영상에 종양 세그멘테이션 → 암 진단 보조 모델

## 출제 포인트

- 7가지 이미지 어노테이션 유형 분류와 각 활용 태스크
- 수동·반자동·크라우드소싱 기법 차이
- 품질 보증 방법(IAA, 골든 셋, 액티브 러닝)
- 데이터 품질이 AI 모델 성능에 미치는 영향
