## 개요

과적합(Overfitting)은 모델이 훈련 데이터의 노이즈까지 학습하여 새로운 데이터에 일반화되지 않는 현상이다. 과소적합(Underfitting)은 모델이 너무 단순하여 훈련 데이터의 패턴조차 제대로 학습하지 못하는 현상이다.

## 핵심 내용

### 발생 원인

| 구분 | 과적합 (Overfitting) | 과소적합 (Underfitting) |
|------|---------------------|------------------------|
| 원인 | 모델 복잡도 과다, 훈련 데이터 부족, 피처 수 과다 | 모델 복잡도 부족, 피처 부족, 과도한 규제 |
| 특징 | 훈련 오차 낮음, 검증 오차 높음 | 훈련·검증 오차 모두 높음 |
| 편향-분산 | 낮은 편향(Bias), 높은 분산(Variance) | 높은 편향, 낮은 분산 |

### 해결 방안

**과적합 해결:**
- **규제(Regularization)**: L1(Lasso), L2(Ridge) — 가중치 크기 제한
- **드롭아웃(Dropout)**: 뉴런 무작위 비활성화
- **조기 종료(Early Stopping)**: 검증 오차 상승 시점에 학습 중단
- **데이터 증강(Data Augmentation)**: 훈련 데이터 인위적으로 확대
- **앙상블(Ensemble)**: 여러 모델 결합으로 분산 감소
- **교차 검증(k-Fold CV)**: 과적합 탐지 및 모델 선택

**과소적합 해결:**
- 더 복잡한 모델 사용 (계층 추가, 파라미터 증가)
- 피처 엔지니어링으로 입력 변수 품질 향상
- 규제 강도 완화
- 더 많은 에포크로 학습

### 편향-분산 트레이드오프

총 오차 = 편향² + 분산 + 불가피한 노이즈
- 복잡한 모델: 편향↓, 분산↑ → 과적합 위험
- 단순한 모델: 편향↑, 분산↓ → 과소적합 위험
- **목표**: 편향과 분산을 동시에 최소화하는 최적 복잡도 탐색
