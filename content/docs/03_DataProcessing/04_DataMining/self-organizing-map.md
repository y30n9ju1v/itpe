## 개요

SOM(Self Organizing Map)은 핀란드 학자 Teuvo Kohonen이 제안한 비지도 신경망 기반 군집화 알고리즘이다. 고차원 입력 데이터를 2차원 격자 지도(Map)로 투영하여 위상 구조(Topology)를 보존하면서 시각화·군집화한다.

## 핵심 내용

### 가. SOM 정의 및 특징

**정의**: 경쟁 학습(Competitive Learning)을 통해 입력 벡터와 가장 유사한 뉴런(BMU)을 선택하고, 이웃 뉴런들의 가중치를 함께 갱신하여 입력 데이터의 위상적 구조를 2D 격자에 표현하는 비지도 신경망

**특징**

| 특징 | 설명 |
|------|------|
| **위상 보존** | 입력 공간에서 유사한 데이터가 출력 격자에서도 인접 위치에 매핑 |
| **비지도 학습** | 레이블 없이 데이터 자체의 패턴으로 학습 |
| **차원 축소** | n차원 → 2차원 격자로 투영 (PCA와 유사하나 비선형) |
| **시각화** | U-Matrix로 군집 경계 시각적 확인 가능 |
| **승자 독식** | BMU(Best Matching Unit)만 가장 크게 갱신 |

### 나. SOM 구성요소

**구조**

```
입력층 (n차원 입력 벡터)
    ↕ (완전 연결)
경쟁층/출력층 (m×m 격자, 각 노드가 n차원 가중치 벡터 보유)
```

| 구성요소 | 설명 |
|---------|------|
| **입력층** | n차원 입력 벡터 |
| **경쟁층(출력층)** | m×m 2D 격자의 뉴런들 (각각 n차원 가중치 벡터 보유) |
| **BMU (Best Matching Unit)** | 입력 벡터와 유클리드 거리가 가장 가까운 뉴런 |
| **이웃 함수 h(t)** | BMU 주변 뉴런 갱신 범위 (가우시안, 학습 진행 시 축소) |
| **학습률 α(t)** | 가중치 갱신 크기 (시간에 따라 감소) |

**학습 알고리즘 (4단계)**
1. **초기화**: 가중치 벡터 무작위 초기화
2. **경쟁**: 입력 벡터와 모든 뉴런 거리 계산 → BMU 선정
3. **협력**: BMU와 이웃 뉴런들의 가중치 갱신 (이웃 함수 적용)
4. **수렴**: 학습률·이웃 범위 점진적 축소 → 반복 수행

**가중치 갱신 공식**
```
w(t+1) = w(t) + α(t) × h(c,i,t) × [x - w(t)]
```
- c: BMU, i: 갱신 대상 뉴런, h: 이웃 함수값

### 다. SOM과 신경망 분석기법의 차이점

| 구분 | SOM | 일반 신경망(MLP) | BP 신경망 |
|------|-----|----------------|----------|
| **학습 방식** | 비지도 (경쟁 학습) | 지도 학습 | 지도 학습 |
| **출력** | 2D 위상 지도 | 분류/회귀 결과 | 분류/회귀 결과 |
| **목적** | 군집화, 시각화, 차원 축소 | 예측, 분류 | 예측, 분류 |
| **레이블 필요** | 불필요 | 필요 | 필요 |
| **활성화 함수** | 없음 (거리 기반 경쟁) | ReLU, Sigmoid 등 | Sigmoid |
| **역전파** | 없음 | 있음 | 있음 |

## 실무 적용 예시

- **고객 세분화**: 구매 패턴 데이터를 SOM으로 군집화하여 타겟 마케팅 세그먼트 도출
- **이상 탐지**: 정상 패턴의 SOM 학습 후, 맵에서 이탈한 데이터를 이상으로 탐지
- **텍스트 마이닝**: 문서 유사도 기반 SOM으로 주제 클러스터 시각화
