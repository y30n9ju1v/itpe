## 개요

연관 규칙 분석(Association Rule Analysis)은 대규모 트랜잭션 데이터에서 항목 간의 동시 출현 패턴을 발견하는 데이터마이닝 기법이다. 마트의 장바구니 분석("기저귀를 사면 맥주도 산다")이 대표적 사례로, 추천 시스템·재고 관리·프로모션 기획에 활용된다.

## 핵심 내용

### 연관 규칙의 표현

```
A → B (지지도: 0.3, 신뢰도: 0.8, 향상도: 1.5)
"A를 구매한 사람은 B도 구매한다"
```

### 3가지 핵심 지표

**① 지지도 (Support)**
```
Support(A → B) = P(A ∩ B) = (A와 B 동시 포함 거래 수) / (전체 거래 수)
```
- 규칙의 빈도·중요도 측정
- 최소 지지도(min_support) 이상만 고려

**② 신뢰도 (Confidence)**
```
Confidence(A → B) = P(B|A) = P(A ∩ B) / P(A)
```
- A 구매 시 B도 구매할 조건부 확률
- 규칙의 정확도 측정
- 단점: P(B) 자체가 높으면 신뢰도도 높게 나옴 (편향)

**③ 향상도 (Lift)**
```
Lift(A → B) = Confidence(A → B) / P(B) = P(A ∩ B) / (P(A) × P(B))
```
- A와 B의 독립성 대비 실제 연관 강도
- Lift > 1: 양의 연관 (A가 B 구매 증가에 기여)
- Lift = 1: 독립 (연관 없음)
- Lift < 1: 음의 연관 (A가 B 구매 감소)

### 예시 계산

| 거래 | 항목 |
|------|------|
| T1 | 빵, 우유, 기저귀 |
| T2 | 맥주, 기저귀, 콜라 |
| T3 | 빵, 우유, 맥주, 기저귀 |
| T4 | 빵, 우유 |

- Support(기저귀 → 맥주) = 2/4 = 0.5
- Confidence(기저귀 → 맥주) = 2/3 ≈ 0.67
- Lift = 0.67 / (2/4) = 1.33 > 1 → 양의 연관

### Apriori 알고리즘

빈발 항목 집합(Frequent Itemset) 생성 후 연관 규칙 도출:
1. 단일 항목 중 min_support 이상인 것 선택
2. 선택 항목으로 2-항목 집합 생성 → min_support 필터
3. 반복하여 빈발 항목 집합 확장 (상향식)
4. 빈발 항목 집합에서 min_confidence 이상 규칙 추출

**선험적 원리(Apriori Principle)**: 빈발하지 않은 항목 집합의 모든 상위 집합도 빈발하지 않음 → 탐색 공간 대폭 축소

## 출제 포인트

- 지지도·신뢰도·향상도 각각의 수식과 의미 서술
- 신뢰도 단독의 한계(P(B) 편향) → 향상도가 필요한 이유
- 향상도 해석 (>1, =1, <1)
- Apriori의 선험적 원리 설명
