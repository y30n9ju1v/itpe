## 개요

데이터 가치평가는 데이터 거래·활용 시 데이터 자산의 경제적 가치를 객관적으로 산정하는 체계다. 데이터산업진흥법(2022) 시행과 함께 데이터 거래소 활성화를 위한 가치평가 방법론이 제도화되고 있다.

## 핵심 내용

### 데이터 재화와 가치의 특징

**데이터의 재화적 특성**:

| 특성 | 설명 |
|------|------|
| **비경합성** | 여러 사용자가 동시에 사용해도 소모되지 않음 |
| **비배제성 극복** | 기술적 접근 제어(DRM)로 배제 가능 |
| **복제 가능성** | 한계비용 거의 0에 무한 복제 가능 |
| **네트워크 효과** | 데이터 결합·축적 시 가치 기하급수적 증가 |
| **시간 가치 감소** | 구축 데이터는 시간에 따라 가치 하락 |

### 데이터 가치평가 모델

| 모델 | 방법 | 적용 상황 |
|------|------|---------|
| **비용 접근법** | 데이터 생성·수집·처리 비용 합산 | 시장 가격 없는 신규 데이터 |
| **시장 접근법** | 유사 데이터 거래 사례 비교 | 활성 거래 시장 존재 시 |
| **수익 접근법** | 데이터 활용으로 기대되는 미래 수익의 현재가치(DCF) | 활용 수익 예측 가능 시 |
| **옵션 가치법** | 미래 활용 가능성을 실물옵션으로 평가 | 잠재적 가치 높은 데이터 |

**가치평가 절차**:
1. 데이터 식별 및 범위 설정
2. 품질 평가 (정확성·완전성·최신성)
3. 가치평가 방법 선택
4. 가치 산정 및 검증
5. 보고서 작성

### 데이터 가치평가 활용방안

- **데이터 거래**: 한국데이터거래소(K-DATA) 등 거래 플랫폼에서 공정 가격 산정
- **기업 자산 인식**: 무형자산으로 재무제표 반영
- **담보·투자**: 데이터 자산 기반 금융 조달
- **M&A 실사**: 데이터 보유 기업 인수 시 자산 평가
- **공공데이터 개방**: 개방 우선순위 결정 및 민간 활용 촉진

## 출제 포인트

- 데이터 재화의 비경합성·복제가능성·네트워크효과
- 가치평가 4모델: 비용·시장·수익·옵션
- 데이터산업법과 가치평가 제도화 연계
