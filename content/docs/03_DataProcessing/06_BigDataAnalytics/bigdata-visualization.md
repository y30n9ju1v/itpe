## 개요

빅데이터 시각화는 대용량·다양·고속의 빅데이터를 인간이 직관적으로 이해할 수 있도록 시각적 형태(차트·그래프·지도·인터랙티브 대시보드)로 표현하는 기술과 방법론이다.

## 핵심 내용

### 개념 및 절차

**개념**:
- 탐색적 시각화: 데이터 내 패턴·이상치 발견 (분석가 대상)
- 설명적 시각화: 인사이트를 청중에게 전달 (의사결정자 대상)

**절차**:

| 단계 | 활동 |
|------|------|
| **1. 목적 정의** | 분석 목적·대상 독자·전달 메시지 명확화 |
| **2. 데이터 수집·전처리** | 결측치 처리, 정규화, 집계 |
| **3. 시각화 유형 선택** | 데이터 특성에 맞는 차트 선택 |
| **4. 시각화 구현** | 도구 활용, 인터랙티브 요소 추가 |
| **5. 검증·배포** | 사용자 피드백 반영, 대시보드 배포 |

### 방법 및 도구

**시각화 유형별 적합 데이터**:

| 유형 | 적합 데이터 | 예시 |
|------|-----------|------|
| 막대 차트 | 범주형 비교 | 지역별 매출 |
| 선 차트 | 시계열 추이 | 일별 방문자 수 |
| 산점도 | 두 변수 상관관계 | 키·몸무게 관계 |
| 히트맵 | 2D 밀도·패턴 | 시간대별 트래픽 |
| 지리 시각화 | 공간 데이터 | 지역별 인구 분포 |
| 트리맵 | 계층적 비율 | 카테고리별 매출 비중 |
| 네트워크 그래프 | 관계·연결 구조 | SNS 영향도 분석 |
| 워드 클라우드 | 텍스트 빈도 | 키워드 분석 |

**빅데이터 시각화 도구**:

| 유형 | 도구 |
|------|------|
| BI(비즈니스 인텔리전스) | Tableau, Power BI, Looker |
| 오픈소스 라이브러리 | D3.js, Plotly, ECharts |
| Python 기반 | Matplotlib, Seaborn, Bokeh |
| 스트리밍 대시보드 | Grafana, Kibana (ELK 스택) |
| 지리 시각화 | Kepler.gl, Leaflet |

### 빅데이터 시각화 특수 과제

- **볼륨**: 수십억 건 데이터의 실시간 렌더링 → 집계·샘플링·GPU 가속
- **속도**: 스트리밍 데이터의 실시간 시각화 → Lambda/Kappa 아키텍처
- **다양성**: 비정형(텍스트·이미지) 포함 → 특수 시각화 기법

## 출제 포인트

- 시각화 절차 5단계
- 데이터 유형별 적합 차트 매핑 (시계열→선, 범주→막대, 상관→산점도)
- Grafana/Kibana: 실시간 로그·메트릭 시각화 도구
