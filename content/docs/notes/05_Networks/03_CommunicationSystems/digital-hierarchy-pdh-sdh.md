## 개요

디지털 계위(Digital Hierarchy)는 디지털 통신에서 여러 채널을 계층적으로 다중화하여 고속 전송을 실현하는 표준 체계다. PDH(비동기식)에서 SDH(동기식), SONET(북미 동기식)으로 발전하였으며, 현대 광통신 네트워크의 기반이다.

## 핵심 내용

### 가. PDH (Plesiochronous Digital Hierarchy)

- **정의**: 각 노드가 독립적인 클럭을 사용하여 준동기(Plesiochronous)로 동작하는 계위 체계
- **표준**: ITU-T G.700 시리즈
- **기본 단위**: DS0 (64Kbps, 단일 음성 채널)

**PDH 계층 구조 (유럽 E-계위)**:

| 계위 | 명칭 | 속도 | 채널 수 |
|------|------|------|--------|
| E0 | DS0 | 64Kbps | 1 |
| E1 | Primary | 2.048Mbps | 30 음성 + 2 제어 = 32 |
| E2 | - | 8.448Mbps | 4×E1 |
| E3 | - | 34.368Mbps | 4×E2 |
| E4 | - | 139.264Mbps | 4×E3 |

**단점**:
- 독립 클럭 → 주파수 편차 발생 → 포인터 바이트(정렬) 필요
- 다중화/역다중화 반복 → 유연성 부족
- 네트워크 관리 기능 취약

### 나. SDH (Synchronous Digital Hierarchy)

- **정의**: 모든 노드가 단일 기준 클럭에 동기화하여 동작하는 디지털 전송 계위
- **표준**: ITU-T G.707
- **기본 단위**: STM-1 (155.52Mbps)

**SDH 계층 구조**:

| 계층 | 속도 |
|------|------|
| STM-1 | 155.52 Mbps |
| STM-4 | 622.08 Mbps |
| STM-16 | 2.5 Gbps |
| STM-64 | 10 Gbps |
| STM-256 | 40 Gbps |

**SDH 장점**:
- **단일 클럭**: 모든 노드 동기화 → 정렬(stuffing) 불필요
- **직접 추출**: 중간 역다중화 없이 특정 채널 직접 접근
- **OAM(운용·관리·유지보수)**: 헤더에 OAM 바이트 내장, 원격 관리 가능
- **링 구조**: BLSR/UPSR으로 자동 복구(50ms 이내)

### 다. SONET (Synchronous Optical NETwork)

- **정의**: 미국·캐나다의 동기식 광전송 표준 (SDH의 북미 버전)
- **표준**: ANSI T1.105
- **기본 단위**: STS-1 / OC-1 (51.84Mbps)

**SONET vs SDH 대응표**:

| SONET | SDH | 속도 |
|-------|-----|------|
| OC-1/STS-1 | - | 51.84 Mbps |
| OC-3/STS-3 | STM-1 | 155.52 Mbps |
| OC-12/STS-12 | STM-4 | 622.08 Mbps |
| OC-48/STS-48 | STM-16 | 2.5 Gbps |
| OC-192/STS-192 | STM-64 | 10 Gbps |

### PDH vs SDH 비교

| 구분 | PDH | SDH |
|------|-----|-----|
| **동기 방식** | 준동기(독립 클럭) | 완전 동기 |
| **기본 속도** | 64Kbps (DS0) | 155.52Mbps (STM-1) |
| **채널 추출** | 역다중화 필요 | 직접 추출 |
| **OAM** | 제한적 | 풍부한 OAM 바이트 |
| **복구** | 느림 | 링 보호(50ms 이내) |
| **사용 현황** | 레거시 | 광전송 백본 |

## 출제 포인트

- PDH의 준동기 개념과 단점(클럭 편차·역다중화 필요)
- SDH 기본 속도 STM-1 = 155.52Mbps
- SONET = SDH 북미판, OC-3 = STM-1 대응 관계
- SDH 장점: 단일 클럭·직접 추출·OAM·링 보호
