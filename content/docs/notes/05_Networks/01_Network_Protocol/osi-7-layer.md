## 개요

OSI 7계층은 ISO가 제정한 네트워크 통신 표준 참조 모델(ISO/IEC 7498)이다. 이기종 시스템 간 통신을 위해 7개 계층으로 역할을 분리하여 각 계층의 독립적 발전과 상호운용성을 보장한다.

## 핵심 내용

### OSI 7계층 전체 요약

| 계층 | 이름 | 기능 | 프로토콜 | 데이터 단위 | 주요 장비 |
|------|------|------|---------|-----------|---------|
| 7 | **응용(Application)** | 사용자 인터페이스, 서비스 | HTTP, FTP, SMTP, DNS, SNMP | 메시지 | - |
| 6 | **표현(Presentation)** | 데이터 형식 변환·암호화·압축 | SSL/TLS, JPEG, MPEG | 메시지 | - |
| 5 | **세션(Session)** | 연결 수립·유지·종료, 동기화 | NetBIOS, RPC, SQL | 메시지 | - |
| 4 | **전송(Transport)** | 종단 간 신뢰성·흐름제어·오류제어 | TCP, UDP, SCTP | 세그먼트 | L4 스위치 |
| 3 | **네트워크(Network)** | 라우팅, 논리 주소(IP) 지정 | IP, ICMP, OSPF, BGP | 패킷 | 라우터, L3 스위치 |
| 2 | **데이터링크(Data Link)** | 물리 주소(MAC), 프레임 오류 검출 | Ethernet, PPP, HDLC, ARP | 프레임 | 스위치, 브리지 |
| 1 | **물리(Physical)** | 비트 전기·광 신호 변환, 전송 매체 | RS-232, USB, IEEE 802.3 | 비트 | 허브, 리피터, 케이블 |

### 계층별 주요 기능 상세

**4계층 (전송)**:
- TCP: 연결 지향, 신뢰성, 흐름제어(슬라이딩 윈도우), 혼잡제어
- UDP: 비연결, 속도 우선, 스트리밍·DNS에 사용

**3계층 (네트워크)**:
- IP 주소 기반 경로 선택(라우팅)
- 단편화(Fragmentation)·재조립

**2계층 (데이터링크)**:
- LLC(논리 링크 제어) + MAC(미디어 접근 제어)
- CRC 오류 검출, ARP(MAC 주소 해석)

### 캡슐화/역캡슐화

```
송신: 데이터 → [7→6→5→4: 세그먼트] → [3: 패킷] → [2: 프레임] → [1: 비트]
수신: 비트 → 프레임 → 패킷 → 세그먼트 → 데이터
```

### TCP/IP 4계층과 매핑

| TCP/IP 계층 | OSI 계층 |
|-----------|---------|
| 응용 계층 | 7 + 6 + 5 |
| 전송 계층 | 4 |
| 인터넷 계층 | 3 |
| 네트워크 접근 계층 | 2 + 1 |

## 출제 포인트

- 7계층 이름·기능·프로토콜·데이터 단위·장비 정확히 암기
- 캡슐화 방향: 송신 7→1, 수신 1→7
- TCP(신뢰성·연결 지향) vs UDP(속도·비연결) 차이
