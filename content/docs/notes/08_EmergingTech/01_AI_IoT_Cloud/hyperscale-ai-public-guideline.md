## 개요

「공공부문 초거대 AI 도입·활용 가이드라인」(과기정통부·행안부)은 공공기관이 대형 언어 모델(LLM) 등 초거대 AI를 도입할 때 준수해야 할 원칙·사전 고려사항·운영 기준을 제시한다. AI 윤리·보안·데이터 주권 등 공공 부문 특수성을 반영한 지침이다.

## 핵심 내용

### 가. 초거대 AI 개념

- **정의**: 수십억~수조 개의 파라미터를 갖는 대규모 딥러닝 모델로, 언어·이미지·코드 등 다양한 태스크를 범용적으로 수행
- **특징**:
  - 사전학습(Pre-training) + 미세조정(Fine-tuning) 패러다임
  - 프롬프트 기반 제로샷·퓨샷 학습
  - 멀티모달(텍스트·이미지·음성) 통합 처리

| 구분 | 소형 모델 | 초거대 AI |
|------|---------|---------|
| **파라미터** | 수백만~수억 | 수십억~수조 |
| **학습 비용** | 소규모 GPU | 수천억 원 수준 |
| **활용 방식** | 단일 태스크 | 범용 (Zero-shot) |
| **대표 모델** | BERT (110M) | GPT-4 (~1T), Claude |

### 나. 초거대 AI 도입 원칙

| 원칙 | 내용 |
|------|------|
| **책임성** | AI 의사결정에 대한 명확한 책임 주체 지정 |
| **투명성** | AI 활용 사실 공개, 판단 근거 설명 가능 |
| **공정성** | 편향 없는 데이터 사용, 차별 방지 |
| **안전성** | 오작동·오남용 방지, 인간 최종 검토 |
| **데이터 주권** | 공공 데이터의 외부 유출 방지 |
| **사람 중심** | AI 보조 도구, 최종 판단은 인간 |

### 다. 초거대 AI 도입 시 사전 고려사항

| 분야 | 고려사항 |
|------|---------|
| **보안** | 학습 데이터·프롬프트에 포함된 민감 정보 유출 방지 |
| **데이터 거버넌스** | 개인정보 포함 데이터 처리 기준, DPA 체결 |
| **저작권** | AI 생성물의 저작권 귀속, 학습 데이터 라이선스 |
| **공급망** | 외산 LLM API 의존 위험, 대안 모델 검토 |
| **성능 검증** | 공공 업무 적합성 평가, 환각(Hallucination) 위험 |
| **예산** | API 비용, 전용 인프라 vs SaaS 비용 비교 |

## 출제 포인트

- 초거대 AI 6대 도입 원칙: 책임성·투명성·공정성·안전성·데이터 주권·사람 중심
- 사전 고려사항 6가지 (보안·데이터·저작권·공급망·성능·예산)
- 환각(Hallucination) 문제와 인간 검토 필수성
