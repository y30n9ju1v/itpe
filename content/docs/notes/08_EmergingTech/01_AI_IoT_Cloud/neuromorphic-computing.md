## 개요


뉴로모픽 컴퓨팅(Neuromorphic Computing)은 인간 뇌의 신경망 구조를 모방한 컴퓨팅 패러다임으로, 뉴런과 시냅스를 하드웨어 수준에서 구현하여 초저전력·이벤트 기반 처리를 가능하게 한다. 기존 폰 노이만 아키텍처의 메모리 병목(Memory Wall)과 높은 전력 소비 문제를 해결하는 대안으로 주목받고 있다.

## 핵심 내용

### 뉴로모픽 컴퓨팅 vs 폰 노이만 아키텍처 비교

| 구분 | 폰 노이만 아키텍처 | 뉴로모픽 컴퓨팅 |
|------|-------------------|----------------|
| **구조** | CPU + 메모리 분리 | 연산·메모리 통합 (In-Memory Computing) |
| **처리 방식** | 클럭 기반 동기 처리 | 이벤트(스파이크) 기반 비동기 처리 |
| **병목** | 메모리 버스(폰 노이만 병목) | 없음 (지역 처리) |
| **전력 소비** | 높음 (항상 동작) | 매우 낮음 (이벤트 발생 시만 동작) |
| **병렬성** | 제한적 (코어 수) | 대규모 병렬 (수백만 뉴런) |
| **프로그래밍** | 명령형·순차적 | 스파이크 기반·확률적 |
| **대표 사례** | Intel x86, ARM | Intel Loihi, IBM TrueNorth |

### SNN (Spiking Neural Network) 동작 원리

SNN은 뇌의 실제 뉴런처럼 전기 스파이크(Spike)를 통해 정보를 전달하는 신경망이다.

**동작 메커니즘:**
1. **입력 스파이크 수신**: 이전 뉴런에서 스파이크(0/1 신호) 도착
2. **막전위(Membrane Potential) 누적**: 수신 스파이크의 가중치 합산
3. **임계값(Threshold) 비교**: 막전위가 임계값 초과 시 스파이크 발생
4. **불응기(Refractory Period)**: 스파이크 후 일정 시간 동안 반응 억제
5. **시간적 부호화**: 스파이크 발생 타이밍으로 정보 인코딩

**기존 ANN과의 차이:**
| 구분 | ANN (일반 신경망) | SNN |
|------|-----------------|-----|
| 신호 | 연속 실수값 | 이진 스파이크 |
| 시간 | 레이어별 동기 | 시간축 포함 비동기 |
| 전력 | 높음 | 매우 낮음 |
| 학습 | 역전파(Backprop) | STDP (Spike-Timing Dependent Plasticity) |

### 활용 분야

- **엣지 AI**: 스마트폰·IoT 기기에서 초저전력 추론
- **로봇공학**: 실시간 감각-운동 제어
- **뇌-컴퓨터 인터페이스**: 신경 신호 실시간 해석
- **자율주행**: 이벤트 카메라 + SNN 기반 물체 인식
- **신약 개발**: 신경 질환 시뮬레이션

## 출제 포인트

- 폰 노이만 병목(메모리 버스 병목) 설명 후 뉴로모픽의 In-Memory Computing 대조
- SNN 동작 4단계: 수신→누적→임계→스파이크 서술
- 전력 소비 차이 강조 (이벤트 기반 처리)
- 활용 분야 3가지 이상 나열
