## 개요

공공데이터 품질인증은 「공공데이터의 제공 및 이용 활성화에 관한 법률」에 근거하여 행정안전부(한국지능정보사회진흥원 위탁)가 공공기관이 보유·관리하는 데이터의 품질 수준을 심사하고 인증하는 제도다. 공공데이터의 신뢰성과 활용성을 제고하는 것이 목적이다.

## 핵심 내용

### 품질인증 체계

**법적 근거**: 공공데이터법 제22조의2 (공공데이터 품질인증)

**인증 기관**: 한국지능정보사회진흥원(NIA)

**인증 유효 기간**: 2년 (갱신 심사 가능)

### 품질 심사 기준 (6개 영역)

| 품질 영역 | 품질 지표 |
|-----------|-----------|
| **정확성** | 데이터 값이 실제와 일치하는 정도 |
| **완전성** | 필수 항목의 누락 없이 기록된 정도 |
| **일관성** | 동일 데이터가 다른 데이터셋과 모순 없는 정도 |
| **유효성** | 정의된 형식·범위에 맞는 정도 |
| **유일성** | 중복 데이터가 없는 정도 |
| **적시성** | 데이터가 최신 상태로 유지되는 정도 |

### 인증 심사 절차

```
신청 → 서류 검토 → 현장 심사 → 심의위원회 → 인증서 발급
```

| 심사 항목 | 내용 |
|-----------|------|
| **관리 체계** | 데이터 품질관리 조직·정책·절차 |
| **데이터 품질** | 샘플 데이터 자동·수동 검증 |
| **개방 수준** | 데이터 공개 범위·형식·갱신 주기 |

### 공공데이터 개방 플랫폼

- **공공데이터포털(data.go.kr)**: 국가 공공데이터 개방 창구
- **표준화 파일 포맷**: CSV, JSON, XML 우선
- **API 제공**: REST API를 통한 실시간 데이터 제공

## 출제 포인트

- 품질인증의 법적 근거 (공공데이터법 제22조의2)
- 6개 품질 영역 (정확성·완전성·일관성·유효성·유일성·적시성)
- 인증 절차와 유효 기간
- 공공데이터 품질인증과 민간 데이터 품질관리의 차이
