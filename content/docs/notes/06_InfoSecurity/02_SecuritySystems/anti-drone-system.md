## 개요

「안티드론 시스템 프레임워크(TTAK.KO-10.1460)」는 불법·위협 드론(UAV)에 대응하기 위한 탐지·추적·식별·대응 절차와 기술 참조구조를 정의한 TTA 정보통신단체표준이다. 주요 국가 시설 보호와 개인 사생활 침해 드론 대응을 목적으로 한다.

## 핵심 내용

### 안티드론 시스템 참조구조

```
┌──────────────────────────────────────────┐
│           지휘·통제(C2) 계층              │ ← 상황 인식·대응 결정
├──────────────────────────────────────────┤
│  탐지·추적  │  식별·분류  │   대응 실행  │
├──────────────────────────────────────────┤
│           센서·작동기(Sensor/Actuator)    │
└──────────────────────────────────────────┘
```

| 기능 계층 | 역할 |
|-----------|------|
| **탐지(Detection)** | 드론 존재 여부 탐지 |
| **추적(Tracking)** | 드론 위치·경로 실시간 추적 |
| **식별(Identification)** | 위협 드론과 합법 드론 구분 |
| **대응(Countermeasure)** | 위협 드론 무력화 또는 강제 착륙 |
| **지휘·통제(C2)** | 전체 시스템 통합 운영·상황 인식 |

### 기술적 조치 참조구조

#### 탐지·추적 기술

| 기술 | 원리 | 특징 |
|------|------|------|
| **레이다(Radar)** | 전파 반사 탐지 | 장거리, 기상 영향 적음 |
| **RF 센서** | 드론 통신 주파수 탐지 | 조종 신호·텔레메트리 탐지 |
| **광학 카메라** | 영상 분석(AI) | 주간 시각 탐지 |
| **열화상 카메라** | 적외선 탐지 | 야간·저시정 탐지 |
| **음향 센서** | 드론 모터 소음 탐지 | 단거리, 소형 드론 효과적 |

#### 대응(무력화) 기술

| 기술 | 방법 | 규제 현황 |
|------|------|-----------|
| **RF 재밍** | 조종·GPS 신호 방해 | 허가 필요 |
| **GPS 스푸핑** | 가짜 GPS 신호로 경로 변조 | 허가 필요 |
| **사이버 탈취** | 드론 통신 해킹·제어권 탈취 | 허가 필요 |
| **레이저** | 고출력 레이저로 물리 무력화 | 제한적 허가 |
| **포획 네트** | 그물로 물리적 포획 | 비교적 자유 |
| **훈련 조류** | 맹금류 드론 포획 훈련 | 실험적 |

### 시스템 운영 절차

```
탐지 → 추적 → 식별·위협 평가 → 대응 결정 → 대응 실행 → 기록·보고
```

## 출제 포인트

- 5단계 기능: 탐지→추적→식별→대응→C2
- 탐지 기술 4가지 이상: 레이다, RF, 광학, 열화상, 음향
- 대응 기술: RF 재밍·GPS 스푸핑(전파 법령 허가 필요)
- 합법 드론 vs 위협 드론 식별의 중요성
